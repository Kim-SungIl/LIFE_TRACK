# 02. 스탯 시스템

## 플레이어에게 보이는 스탯 (단순)

| 스탯 | 설명 |
|------|------|
| 학업 (Academic) | 전반적인 학습 능력 |
| 인기 (Social) | 교우관계, 사회성 |
| 재능 (Talent) | 예체능, 특기 분야 |
| 멘탈 (Mental) | 정신적 안정, 스트레스 내성 |
| 체력 (Health) | 신체 건강, 컨디션 |

## 내부 계산용 (UI에 직접 노출 안 함)

- 학업 성향: 문과계열 / 이과계열
- 예체능 성향: 음악 / 미술 / 체육
- 특기: 코딩 / 경영 / 창작 등

> 원칙: UI는 단순, 내부는 복잡

---

## 멘탈 시스템

| 상태 | 효과 |
|------|------|
| 정상 | 기본 플레이 |
| 피로 | 성장 효율 감소 |
| 번아웃 | 활동 선택 제한, 강제 휴식 |
| 각성 | 리스크 높지만 폭발적 성장 가능 |

> 체력 스탯을 무시하면 시험 당일 컨디션 저하 — 랜덤이 아니라 "내 선택의 결과"
