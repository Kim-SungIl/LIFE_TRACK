# 02. 스탯 시스템 — 수치 설계서

> 설계 원칙
> - 360주 동안 스탯이 의미있게 성장하되, 만렙은 사실상 불가능
> - "모든 스탯 올리기"는 구조적으로 불가능 (핵심 철학: 포기의 고통)
> - 수치는 내부적으로 작동하되, 플레이어에게는 단순하게 보여야 함
> - 한국 학생의 현실적 시간 배분을 반영

---

## 1. 스탯 수치 범위

### 1-1. 보이는 스탯 (5종)

| 스탯 | 범위 | 설명 |
|------|------|------|
| 학업 (Academic) | 0 ~ 100 | 전반적인 학습 능력, 시험 성적에 직결 |
| 인기 (Social) | 0 ~ 100 | 교우관계, 사회성, NPC 호감도 기반 |
| 재능 (Talent) | 0 ~ 100 | 예체능, 특기 분야의 숙련도 |
| 멘탈 (Mental) | 0 ~ 100 | 정신적 안정, 스트레스 내성 |
| 체력 (Health) | 0 ~ 100 | 신체 건강, 컨디션, 지구력 |

**플레이어에게 보여주는 방식:** 수치 직접 노출 대신 5단계 등급으로 표시

| 내부 수치 | 표시 등급 | 아이콘/색상 |
|-----------|-----------|-------------|
| 0 ~ 19 | E (매우 부족) | 회색 |
| 20 ~ 39 | D (부족) | 빨강 |
| 40 ~ 59 | C (보통) | 노랑 |
| 60 ~ 79 | B (우수) | 초록 |
| 80 ~ 100 | A (최상) | 금색 |

### 1-2. 기본 시작값

모든 플레이어 공통 기본값 (부모/가정환경 보정 적용 전):

| 스탯 | 기본값 | 설명 |
|------|--------|------|
| 학업 | 30 | 초6 수준의 평범한 학생 |
| 인기 | 25 | 새 환경에서의 출발점 |
| 재능 | 15 | 아직 특기를 발견하지 못한 상태 |
| 멘탈 | 50 | 건강한 아이의 기본 정신 상태 |
| 체력 | 40 | 성장기 아이의 기본 체력 |

> 멘탈은 의도적으로 중간값에서 시작한다. "깎이는 자원"의 성격이 강하기 때문.
> 체력은 약간 낮게 시작하여 운동의 가치를 느끼게 한다.

**부모 유형별 보정 (별도 문서에서 정의):**
- 예) 교육열 높은 부모: 학업 +10, 멘탈 -5
- 예) 방임형 부모: 재능 +5, 학업 -10, 멘탈 -5
- 예) 균형형 부모: 모든 스탯 +3

### 1-3. 성장률

**기본 성장 공식:**
```
실제 성장량 = 기본 성장량 × 체력 보정 × 멘탈 보정 × 피로 보정 × 구간 감쇠
```

**기본 성장량** (활동 1회 = 1슬롯 기준, 활동 테이블에서 상세 정의):
- 저효율 활동 (예: 독학): +1 ~ +2
- 중효율 활동 (예: 학원): +2 ~ +3
- 고효율 활동 (예: 집중 훈련): +3 ~ +5

**구간 감쇠 (diminishing returns):**
스탯이 높아질수록 같은 활동의 효과가 줄어든다.

| 현재 스탯 구간 | 성장 배율 |
|----------------|-----------|
| 0 ~ 29 | ×1.2 (초반 가속) |
| 30 ~ 49 | ×1.0 (기본) |
| 50 ~ 69 | ×0.8 |
| 70 ~ 84 | ×0.5 |
| 85 ~ 94 | ×0.3 |
| 95 ~ 100 | ×0.1 |

> 이 설계의 의미: 학업 30→60은 비교적 쉽지만, 80→90은 극도로 어렵다.
> 360주 동안 하나의 스탯만 올려도 95 이상은 매우 힘들다.
> "올인해도 만렙은 거의 불가능" → 선택과 집중을 강제한다.

**시뮬레이션 예시 (학업에만 매주 2슬롯 투자, 기본 활동 +2 기준):**
- 50주 (약 1년): 30 → 약 55
- 150주 (약 3년): 30 → 약 75
- 360주 (7년 전체): 30 → 약 88
- → 한 스탯에 전 기간의 40%를 쏟아도 A등급 하위권

### 1-4. 자연 감소 (Decay)

스탯은 해당 활동을 하지 않으면 **느리게 감소**한다.
단, 멘탈은 자연 감소하지 않고 피로 시스템에 의해서만 변동한다.

| 스탯 | 주당 감소량 (미활동 시) | 감소 하한선 |
|------|--------------------------|-------------|
| 학업 | -0.5 / 주 | 10 (기초 학력) |
| 인기 | -1.0 / 주 | 5 (잊혀짐) |
| 재능 | -0.3 / 주 | 5 (잔여 감각) |
| 멘탈 | 감소 없음 (피로 시스템으로 관리) | — |
| 체력 | -0.8 / 주 | 15 (기본 체력) |

> 인기가 가장 빨리 감소한다: 관계는 관리하지 않으면 소원해진다.
> 재능은 느리게 감소한다: 한번 배운 기술은 쉽게 잊히지 않는다.
> 감소 하한선이 있어 완전히 0이 되지는 않는다.

### 1-5. 스탯 상호작용

스탯은 서로 영향을 미친다. 핵심 상호작용은 다음과 같다:

**체력 보정 (모든 활동에 적용):**

| 체력 구간 | 보정 배율 | 설명 |
|-----------|-----------|------|
| 60 이상 | ×1.0 | 정상 |
| 40 ~ 59 | ×0.9 | 약간 부진 |
| 20 ~ 39 | ×0.7 | 컨디션 저하 |
| 0 ~ 19 | ×0.5 | 심각한 체력 저하, 병결 이벤트 발생 가능 |

**멘탈 보정 (모든 활동에 적용):**

| 멘탈 상태 | 보정 배율 | 설명 |
|-----------|-----------|------|
| 정상 (멘탈 50 이상) | ×1.0 | 기본 |
| 피로 (멘탈 30~49) | ×0.7 | 의욕 저하 |
| 번아웃 (멘탈 15~29) | ×0.3 | 거의 성장 불가 |
| 위험 (멘탈 0~14) | ×0.0 | 활동 불가, 강제 이벤트 |

> **핵심:** 체력과 멘탈을 무시하고 학업만 올리면, 보정 배율이 떨어져 오히려 효율이 나빠진다.
> 이것이 "밸런스를 맞추는 플레이"에도 가치가 있는 이유다.

## 2. 주당 행동 슬롯

### 2-1. 기본 구조

**주당 총 슬롯: 5**

| 슬롯 | 대응하는 시간대 |
|------|-----------------|
| 슬롯 1 | 월~화 방과 후 |
| 슬롯 2 | 수~목 방과 후 |
| 슬롯 3 | 금 방과 후 |
| 슬롯 4 | 토요일 |
| 슬롯 5 | 일요일 |

> 5슬롯 근거:
> - 한국 학생의 일주일은 "수업 + 방과후 + 주말"로 나뉜다.
> - 수업은 모두 공통이므로 선택의 대상이 아니다.
> - 방과후 3블록 + 주말 2블록 = 5슬롯이 가장 현실적이다.
> - 매주 5번의 선택 = 플레이어가 부담 없이 빠르게 결정할 수 있는 수.

### 2-2. 시기별 변화

슬롯 수 자체는 변하지 않지만, **강제/반강제 슬롯**이 생긴다.

| 시기 | 자유 슬롯 | 강제/반강제 슬롯 | 설명 |
|------|-----------|------------------|------|
| 초6 (1~48주) | 5 | 0 | 완전한 자유 |
| 중1 (49~96주) | 4~5 | 0~1 | 부모 성향에 따라 학원 강제 가능 |
| 중2 (97~144주) | 4 | 1 | 내신 관리 압박 시작 |
| 중3 (145~192주) | 3~4 | 1~2 | 고입 준비, 학원/자습 반강제 |
| 고1 (193~240주) | 3 | 2 | 야자/학원 고정, 자유시간 감소 |
| 고2 (241~288주) | 2~3 | 2~3 | 입시 체제 본격화 |
| 고3 (289~360주) | 2 | 3 | 대부분의 시간이 수능 준비에 묶임 |

> **핵심 메커니즘:** 고등학교로 갈수록 자유 슬롯이 줄어든다.
> 플레이어는 점점 좁아지는 선택지 안에서 무엇을 지킬지 결정해야 한다.
> "포기의 고통"이 시기별로 자연스럽게 강화된다.

**강제 슬롯 저항:**
- 부모 관계가 좋으면: 강제 슬롯 1개를 자유 슬롯으로 전환 가능 (설득 이벤트)
- 반항 선택지: 강제 슬롯을 무시할 수 있지만, 부모 관계 -10, 용돈 삭감 위험
- 자립도 (은닉 스탯)가 높으면: 고3에서도 자유 슬롯 3개 유지 가능

### 2-3. 1슬롯의 정의

1슬롯 = **방과 후 약 3~4시간에 해당하는 하나의 활동 블록**

- 같은 활동을 여러 슬롯에 배치 가능 (예: 공부 ×3)
- 같은 활동 반복 시 효율 체감: 2번째 ×0.8, 3번째 ×0.6, 4번째 이상 ×0.4
- 일부 활동은 2슬롯을 소모 (예: 대회 참가, MT)

## 3. 피로/에너지 시스템

### 3-1. 기본 구조: 피로 누적 방식

**피로(Fatigue)가 0에서 시작하여 누적되는 방식을 채택한다.**

> 에너지 소모 방식(매주 리셋) 대신 피로 누적 방식을 선택한 이유:
> - "무리하면 다음 주에도 영향이 남는다"는 현실감
> - 장기적 건강 관리의 중요성을 부각
> - 번아웃으로의 자연스러운 전환

| 항목 | 값 |
|------|-----|
| 피로 최솟값 | 0 (완전 회복) |
| 피로 최댓값 | 100 (한계) |
| 주 시작 시 자연 회복 | -5 (매주 월요일 자동) |
| 기본 시작 피로 | 0 |

### 3-2. 활동별 피로 비용

| 피로 등급 | 피로 증가량 | 해당 활동 예시 |
|-----------|-------------|----------------|
| 가벼움 | +2 ~ +4 | 산책, SNS, 친구와 수다 |
| 보통 | +5 ~ +8 | 독학, 학원, 가벼운 운동 |
| 무거움 | +9 ~ +12 | 집중 공부, 강도 높은 훈련, 알바 |
| 극한 | +13 ~ +16 | 시험 벼락치기, 대회 준비 |

> 활동별 구체적 피로 비용은 [5. 활동 테이블] 참조

### 3-3. 피로 회복 방법

| 방법 | 피로 감소량 | 비용/조건 |
|------|-------------|-----------|
| 주간 자연 회복 | -5 / 주 | 자동 (매주 적용) |
| 휴식 (1슬롯) | -12 | 슬롯 1개 사용 |
| 푹 쉬기 (2슬롯) | -25 | 슬롯 2개 사용, 주말 전체 |
| 여행 이벤트 | -30 | 방학 중 특수 선택지 |
| 친구와 놀기 | -6 (부가 효과) | 인기 스탯도 상승 |
| 운동 (가벼운) | -3 (부가 효과) | 체력 스탯도 상승 |

> 휴식은 피로를 크게 줄이지만 **다른 스탯이 성장하지 않는다.**
> 이것이 "쉬어야 하는데 쉴 수 없는" 딜레마를 만든다.

### 3-4. 피로 단계별 효과

| 피로 구간 | 상태명 | 효과 |
|-----------|--------|------|
| 0 ~ 29 | 양호 | 정상 플레이, 보정 없음 |
| 30 ~ 49 | 주의 | 스탯 성장 ×0.9, UI에 경고 표시 |
| 50 ~ 69 | 과로 | 스탯 성장 ×0.7, 멘탈 주당 -2 자동 감소 |
| 70 ~ 89 | 위험 | 스탯 성장 ×0.5, 멘탈 주당 -4 자동 감소, 병결 확률 발생 |
| 90 ~ 100 | 한계 | 스탯 성장 ×0.2, 멘탈 주당 -6, 강제 휴식 이벤트 발생 |

> **피로 → 멘탈 연결이 핵심이다.**
> 피로가 쌓이면 멘탈이 깎이고, 멘탈이 낮으면 모든 효율이 떨어진다.
> "무리 → 피로 → 멘탈 저하 → 효율 하락 → 더 무리 → 번아웃" 악순환을 만든다.

### 3-5. 피로와 멘탈의 연결

```
[피로 누적] → [멘탈 자동 감소] → [멘탈 상태 전환] → [활동 제한/보정 변화]
     ↑                                                        ↓
     ← ← ← ← ← [효율 저하로 더 무리하게 됨] ← ← ← ← ← ← ←
```

## 4. 멘탈 상태 전환 시스템

### 4-1. 4단계 상태

```
정상 ←→ 피로 ←→ 번아웃
                    ↓
                  각성 (특수 조건)
```

### 4-2. 전환 조건 (구체적 수치)

#### 정상 → 피로

| 조건 | 값 |
|------|-----|
| 멘탈 수치 | 50 → 30 미만으로 하락 시 |
| 또는 | 피로 50 이상이 3주 연속 유지 시 |
| 전환 연출 | "요즘 뭘 해도 재미없다..." 독백 컷신 |

#### 피로 → 정상 (회복)

| 조건 | 값 |
|------|-----|
| 멘탈 수치 | 30 → 50 이상으로 회복 시 |
| 그리고 | 피로 30 미만 유지 2주 |
| 회복 방법 | 휴식, 친구 관계, 상담 이벤트 |

#### 피로 → 번아웃

| 조건 | 값 |
|------|-----|
| 멘탈 수치 | 30 → 15 미만으로 하락 시 |
| 또는 | 피로 상태에서 4주 연속 피로 70 이상 유지 시 |
| 또는 | 피로 상태에서 중요 이벤트 실패 (시험 하위 30% 등) |
| 전환 연출 | 화면 채도 저하 + "...더 이상 못하겠다" 컷신 |

#### 번아웃 효과 (구체적 제한)

| 항목 | 제한 내용 |
|------|-----------|
| 슬롯 | 자유 슬롯 중 1개 강제 휴식으로 고정 |
| 학업 활동 | 선택 가능하지만 성장량 ×0.3 |
| 관계 활동 | 선택 불가 ("만날 기운이 없다") |
| 재능 활동 | 선택 가능하지만 성장량 ×0.3 |
| 체력 활동 | 가벼운 운동만 가능 |
| 이벤트 | 부정적 선택지만 나타남 (짜증, 회피, 거절) |
| 지속 기간 | 최소 4주 (회복 조건 충족해도 4주는 유지) |
| NPC 반응 | 친한 NPC가 걱정 이벤트 발생 |

#### 번아웃 → 피로 (회복)

| 조건 | 값 |
|------|-----|
| 최소 지속 | 4주 |
| 피로 | 30 미만으로 감소 |
| 멘탈 | 20 이상으로 회복 |
| 회복 촉진 | 상담 이벤트 선택 시 회복 속도 ×2 |
| 회복 촉진 | 친밀 NPC의 위로 이벤트 수락 시 멘탈 +10 |

#### 각성 조건 (특수 상태)

**각성은 "극한 상황에서의 돌파"를 표현하는 고위험·고보상 상태다.**

| 조건 | 값 |
|------|-----|
| 진입 조건 1 | 피로 상태 + 특정 스탯 70 이상 + 해당 스탯 관련 대회/시험 이벤트 발생 |
| 진입 조건 2 | 번아웃 상태에서 NPC 각성 이벤트 발생 시 "그래도 해보겠다" 선택 |
| 진입 조건 3 | 멘탈 25~35 구간 + 피로 60 이상 + 재능 60 이상 + 재능 활동 선택 |
| 확률 | 조건 충족 시 50% 확률로 각성 진입 (실패 시 번아웃으로 전환) |
| 지속 기간 | 4주 고정 |

#### 각성 효과

| 항목 | 효과 |
|------|------|

| 피로 증가 | ×1.5 (무리의 대가) |
| 멘탈 변동 없음 | 각성 기간 동안 멘탈 고정 |
| 종료 후 | 피로 +20, 멘탈 -15 (반동) |
| 재발동 | 같은 플레이에서 최대 3회까지 |

> **각성의 도박성:** 실패 시 번아웃, 성공해도 반동이 크다.
> "여기서 무리하면 성장할 수 있지만, 대가를 치른다"는 선택을 강제한다.

## 6. 밸런스 검증 시뮬레이션

### 6-1. 공부 특화 플레이 (360주)

```
매주 구성: 학원 ×2 + 독학 ×1 + 휴식 ×1 + 자유 ×1
주당 피로: +8+8+6-12+α = ~+10 → 자연회복 -5 = 순증 +5
→ 약 6주마다 피로 30 도달, 휴식 조절 필요

예상 최종 스탯:
- 학업: ~85 (A 하위)
- 인기: ~10 (E, 관계 미관리로 하락)
- 재능: ~8 (E)
- 멘탈: ~35 (피로 상태 왔다갔다)
- 체력: ~20 (D, 운동 미관리)
```

### 6-2. 밸런스 플레이 (360주)

```
매주 구성: 독학 ×1 + 친구 ×1 + 가벼운운동 ×1 + 창작 ×1 + 휴식 ×1
주당 피로: +6+3+2+4-12 = +3 → 자연회복 -5 = 순증 -2 (피로 안정)

예상 최종 스탯:
- 학업: ~55 (C)
- 인기: ~50 (C)
- 재능: ~45 (C)
- 멘탈: ~65 (정상 유지)
- 체력: ~45 (C)
→ 모든 스탯 C 근처: "평범하지만 안정적인 학생"
```

### 6-3. 번아웃 플레이 (무리하는 경우)

```
매주 구성: 학원 ×2 + 시험벼락치기 ×1 + 독학 ×2 (휴식 없음)
주당 피로: +8+8+14+6+6 = +42 → 자연회복 -5 = 순증 +37
→ 3주 만에 피로 100 도달
→ 4~5주 차에 번아웃 진입
→ 강제 휴식으로 4주 손실
→ 실질적으로 더 비효율적
```

> **결론:** 쉬지 않고 달리는 것은 오히려 손해다.
> 이 시스템은 "적절한 휴식"의 가치를 수치로 증명한다.

## 8. 수식 요약

### 최종 성장 공식

```
실제_성장량 = floor(
    기본_성장량
    × 구간_감쇠(현재_스탯)
    × 체력_보정(현재_체력)
    × 멘탈_보정(현재_멘탈_상태)
    × 피로_보정(현재_피로)
    × 반복_감쇠(같은_활동_반복_횟수)
)

// 최소 보장: 조건 충족 시 최소 +1 (완전히 0이 되지 않음, 번아웃 제외)
실제_성장량 = max(실제_성장량, 1)  // 번아웃이 아닌 경우
실제_성장량 = max(실제_성장량, 0)  // 번아웃인 경우
```

### 주간 스탯 업데이트 순서

```
1. 주 시작: 피로 자연 회복 (-5)
2. 피로 상태에 따른 멘탈 자동 감소 적용
3. 멘탈 상태 전환 체크 (정상↔피로↔번아웃)
4. 플레이어 활동 선택 (5슬롯)
5. 각 활동별 스탯 변동 계산 (보정 적용)
6. 각 활동별 피로 변동 적용
7. 미활동 스탯 자연 감소 적용
8. 용돈 변동 적용
9. 이벤트 트리거 체크
10. 결과 표시
```
