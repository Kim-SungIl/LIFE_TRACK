# 02. 스탯 시스템 — 수치 설계서

> 설계 원칙
> - 360주 동안 스탯이 의미있게 성장하되, 만렙은 사실상 불가능
> - "모든 스탯 올리기"는 구조적으로 불가능 (핵심 철학: 포기의 고통)
> - 수치는 내부적으로 작동하되, 플레이어에게는 단순하게 보여야 함
> - 한국 학생의 현실적 시간 배분을 반영

---

## 1. 스탯 수치 범위

### 1-1. 보이는 스탯 (5종)

| 스탯 | 범위 | 설명 |
|------|------|------|
| 학업 (Academic) | 0 ~ 100 | 전반적인 학습 능력, 시험 성적에 직결 |
| 인기 (Social) | 0 ~ 100 | 교우관계, 사회성, NPC 호감도 기반 |
| 재능 (Talent) | 0 ~ 100 | 예체능, 특기 분야의 숙련도 |
| 멘탈 (Mental) | 0 ~ 100 | 정신적 안정, 스트레스 내성 |
| 체력 (Health) | 0 ~ 100 | 신체 건강, 컨디션, 지구력 |

**플레이어에게 보여주는 방식:** 수치 직접 노출 대신 5단계 등급으로 표시

| 내부 수치 | 표시 등급 | 아이콘/색상 |
|-----------|-----------|-------------|
| 0 ~ 19 | E (매우 부족) | 회색 |
| 20 ~ 39 | D (부족) | 빨강 |
| 40 ~ 59 | C (보통) | 노랑 |
| 60 ~ 79 | B (우수) | 초록 |
| 80 ~ 100 | A (최상) | 금색 |

### 1-2. 기본 시작값

모든 플레이어 공통 기본값 (부모/가정환경 보정 적용 전):

| 스탯 | 기본값 | 설명 |
|------|--------|------|
| 학업 | 30 | 초6 수준의 평범한 학생 |
| 인기 | 25 | 새 환경에서의 출발점 |
| 재능 | 15 | 아직 특기를 발견하지 못한 상태 |
| 멘탈 | 50 | 건강한 아이의 기본 정신 상태 |
| 체력 | 40 | 성장기 아이의 기본 체력 |

> 멘탈은 의도적으로 중간값에서 시작한다. "깎이는 자원"의 성격이 강하기 때문.
> 체력은 약간 낮게 시작하여 운동의 가치를 느끼게 한다.

**부모 유형별 보정 (별도 문서에서 정의):**
- 예) 교육열 높은 부모: 학업 +10, 멘탈 -5
- 예) 방임형 부모: 재능 +5, 학업 -10, 멘탈 -5
- 예) 균형형 부모: 모든 스탯 +3

### 1-3. 성장률

**기본 성장 공식 (하이브리드 연산):**
```
최종 성장량 = (기본값 + 버프가산) × 상태보정
  - 버프가산 = 루틴보너스 + 정보력보너스 + NPC시너지 등 (가산)
  - 상태보정 = 구간감쇠 × 체력보정 × 멘탈보정 × 피로보정 (곱셈, 최저 x0.15 보장)
```

**기본 성장량** (활동 1회 = 1슬롯 기준, 활동 테이블에서 상세 정의):
- 저효율 활동 (예: 독학): +1 ~ +2
- 중효율 활동 (예: 학원): +2 ~ +3
- 고효율 활동 (예: 집중 훈련): +3 ~ +5

**구간 감쇠 (diminishing returns):**
스탯이 높아질수록 같은 활동의 효과가 줄어든다.

| 현재 스탯 구간 | 성장 배율 |
|----------------|-----------|
| 0 ~ 29 | ×1.2 (초반 가속) |
| 30 ~ 49 | ×1.0 (기본) |
| 50 ~ 69 | ×0.8 |
| 70 ~ 84 | ×0.5 |
| 85 ~ 94 | ×0.3 |
| 95 ~ 100 | ×0.1 |

> 이 설계의 의미: 학업 30→60은 비교적 쉽지만, 80→90은 극도로 어렵다.
> 360주 동안 하나의 스탯만 올려도 95 이상은 매우 힘들다.
> "올인해도 만렙은 거의 불가능" → 선택과 집중을 강제한다.

**시뮬레이션 예시 (학업에만 매주 2슬롯 투자, 기본 활동 +2 기준):**
- 50주 (약 1년): 30 → 약 55
- 150주 (약 3년): 30 → 약 75
- 360주 (7년 전체): 30 → 약 88
- → 한 스탯에 전 기간의 40%를 쏟아도 A등급 하위권

### 1-4. 자연 감소 (Decay)

스탯은 해당 활동을 하지 않으면 **느리게 감소**한다.
단, 멘탈은 자연 감소하지 않고 피로 시스템에 의해서만 변동한다.

| 스탯 | 주당 감소량 (미활동 시) | 감소 하한선 |
|------|--------------------------|-------------|
| 학업 | -0.5 / 주 | 10 (기초 학력) |
| 인기 | -1.0 / 주 | 5 (잊혀짐) |
| 재능 | -0.3 / 주 | 5 (잔여 감각) |
| 멘탈 | 감소 없음 (피로 시스템으로 관리) | — |
| 체력 | -0.8 / 주 (학기 중 학교 수업 시 -0.4) | 15 (기본 체력) |

> 인기가 가장 빨리 감소한다: 관계는 관리하지 않으면 소원해진다.
> 재능은 느리게 감소한다: 한번 배운 기술은 쉽게 잊히지 않는다.
> 감소 하한선이 있어 완전히 0이 되지는 않는다.
> **체력 자연감소 상쇄 (v3):** 학기 중 학교 수업 슬롯이 존재하면 체력 자연감소가 50% 상쇄된다 (-0.8 → -0.4/주).
> "학교를 다니는 것 자체가 최소한의 신체활동 (체육 수업, 등하교)."
> 방학 중에는 학교 슬롯이 없으므로 -0.8/주 그대로 적용된다.

### 1-5. 스탯 상호작용

스탯은 서로 영향을 미친다. 핵심 상호작용은 다음과 같다:

**체력 보정 (스탯 성장):**

| 체력 구간 | 보정 배율 |
|-----------|-----------|
| 80~100 | ×1.1 |
| 65~79 | ×1.0 |
| 50~64 | ×0.95 |
| 40~49 | ×0.9 |
| 30~39 | ×0.8 |
| 20~29 | ×0.65 |
| 10~19 | ×0.4 |
| 0~9 | ×0.1 |

**체력 → 피로 내성 보정 (피로 축적/회복에 직접 영향):**

체력은 단순 성장 보정을 넘어, **피로 축적량과 휴식 회복량에 직접 영향**을 미친다.
모든 활동의 피로 증가/감소값은 아래 체력 보정을 거친 후 최종 적용된다.

| 체력 구간 | 피로 축적 보정 | 휴식 회복 보정 |
|-----------|--------------|--------------|
| 80~100 | -20% | +25% |
| 65~79 | -10% | +15% |
| 50~64 | 기본 | +5% |
| 40~49 | +5% | 기본 |
| 30~39 | +10% | -10% |
| 20~29 | +20% | -15% |
| 0~19 | +30% | -25% |

> **체력은 "숨겨진 사기 스탯"이다.**
> 초반에 체력에 투자하면 당장은 눈에 보이는 성과가 없지만,
> 장기적으로 피로 관리가 쉬워져 모든 활동의 실질 효율이 올라간다.
> 숙련된 플레이어는 초6~중1에 체력을 B 이상으로 올린 뒤 본격적으로 다른 스탯을 키운다.
>
> **계산 예시:**
> - 학원 수업(피로 +8) → 체력 85인 경우: 8 × 0.80 = +6.4 → +6
> - 학원 수업(피로 +8) → 체력 15인 경우: 8 × 1.30 = +10.4 → +10
> - 휴식(피로 -12) → 체력 85인 경우: 12 × 1.25 = -15
> - 휴식(피로 -12) → 체력 15인 경우: 12 × 0.75 = -9
>
> → 체력 80+ vs 0~19의 차이: 같은 학원+휴식 사이클에서 순 피로가 **약 -9 vs +1** 차이

**멘탈 보정 (스탯 성장):**

| 멘탈 구간 | 보정 배율 |
|-----------|-----------|
| 80~100 | ×1.1 |
| 65~79 | ×1.0 |
| 50~64 | ×0.95 |
| 40~49 | ×0.85 |
| 30~39 | ×0.75 |
| 20~29 | ×0.6 |
| 10~19 | ×0.4 (번아웃) |
| 0~9 | ×0.1 (위험) |

> **핵심:** 체력과 멘탈을 무시하고 학업만 올리면, 보정 배율이 떨어져 오히려 효율이 나빠진다.
> 이것이 "밸런스를 맞추는 플레이"에도 가치가 있는 이유다.

## 2. 주당 행동 슬롯

### 2-1. 기본 구조

**주당 총 슬롯: 5**

| 슬롯 | 대응하는 시간대 |
|------|-----------------|
| 슬롯 1 | 월~화 방과 후 |
| 슬롯 2 | 수~목 방과 후 |
| 슬롯 3 | 금 방과 후 |
| 슬롯 4 | 토요일 |
| 슬롯 5 | 일요일 |

> 5슬롯 근거:
> - 한국 학생의 일주일은 "수업 + 방과후 + 주말"로 나뉜다.
> - 수업은 모두 공통이므로 선택의 대상이 아니다.
> - 방과후 3블록 + 주말 2블록 = 5슬롯이 가장 현실적이다.
> - 매주 5번의 선택 = 플레이어가 부담 없이 빠르게 결정할 수 있는 수.

### 2-2. 시기별 변화

슬롯 수 자체는 변하지 않지만, **강제/반강제 슬롯**이 생긴다.

| 시기 | 자유 슬롯 | 강제/반강제 슬롯 | 설명 |
|------|-----------|------------------|------|
| 초6 (1~48주) | 5 | 0 | 완전한 자유 |
| 중1 (49~96주) | 4~5 | 0~1 | 부모 성향에 따라 학원 강제 가능 |
| 중2 (97~144주) | 4 | 1 | 내신 관리 압박 시작 |
| 중3 (145~192주) | 3~4 | 1~2 | 고입 준비, 학원/자습 반강제 |
| 고1 (193~240주) | 3 | 2 | 야자/학원 고정, 자유시간 감소 |
| 고2 (241~288주) | 2~3 | 2~3 | 입시 체제 본격화 |
| 고3 (289~360주) | 2 | 3 | 대부분의 시간이 수능 준비에 묶임 |

> **핵심 메커니즘:** 고등학교로 갈수록 자유 슬롯이 줄어든다.
> 플레이어는 점점 좁아지는 선택지 안에서 무엇을 지킬지 결정해야 한다.
> "포기의 고통"이 시기별로 자연스럽게 강화된다.

**강제 슬롯 저항:**
- 부모 관계가 좋으면: 강제 슬롯 1개를 자유 슬롯으로 전환 가능 (설득 이벤트)
- 반항 선택지: 강제 슬롯을 무시할 수 있지만, 부모 관계 -10, 용돈 삭감 위험
- 자립도 (은닉 스탯)가 높으면: 고3에서도 자유 슬롯 3개 유지 가능

### 2-3. 1슬롯의 정의

1슬롯 = **방과 후 약 3~4시간에 해당하는 하나의 활동 블록**

- 같은 활동을 여러 슬롯에 배치 가능 (예: 공부 ×3)
- 같은 활동 반복 시 효율 체감: 2번째 ×0.8, 3번째 ×0.6, 4번째 이상 ×0.4
- 일부 활동은 2슬롯을 소모 (예: 대회 참가, MT)

## 3. 피로/에너지 시스템

### 3-1. 기본 구조: 피로 누적 방식

**피로(Fatigue)가 0에서 시작하여 누적되는 방식을 채택한다.**

> 에너지 소모 방식(매주 리셋) 대신 피로 누적 방식을 선택한 이유:
> - "무리하면 다음 주에도 영향이 남는다"는 현실감
> - 장기적 건강 관리의 중요성을 부각
> - 번아웃으로의 자연스러운 전환

| 항목 | 값 |
|------|-----|
| 피로 최솟값 | 0 (완전 회복) |
| 피로 최댓값 | 100 (한계) |
| 주 시작 시 자연 회복 | 피로 잔량의 15% (최소 -3, 최대 -12) — 매주 월요일 자동 |
| 기본 시작 피로 | 0 |

### 3-2. 활동별 피로 비용

> ⚠ 아래 수치는 **체력 보정 적용 전 기본값**이다. 실제 피로 변동은 §1-5 체력→피로 내성 보정을 거쳐 최종 결정된다.

| 피로 등급 | 피로 증가량 | 해당 활동 예시 |
|-----------|-------------|----------------|
| 가벼움 | +2 ~ +4 | 산책, SNS, 친구와 수다 |
| 보통 | +5 ~ +8 | 독학, 학원, 가벼운 운동 |
| 무거움 | +9 ~ +12 | 집중 공부, 강도 높은 훈련, 알바 |
| 극한 | +13 ~ +16 | 시험 벼락치기, 대회 준비 |

> 활동별 구체적 피로 비용은 [5. 활동 테이블] 참조

### 3-3. 피로 회복 방법

| 방법 | 피로 감소량 | 비용/조건 |
|------|-------------|-----------|
| 주간 자연 회복 | 피로 잔량의 15% (최소 -3, 최대 -12) | 자동 (매주 적용). 정서적 지지 가정: 최종 회복량에 고정 **+2** 추가 (최대 -12 유지) |
| 휴식 (1슬롯) | -12 | 슬롯 1개 사용 |
| 푹 쉬기 (2슬롯) | -25 | 슬롯 2개 사용, 주말 전체 |
| 여행 이벤트 | -30 | 방학 중 특수 선택지 |
| 친구와 놀기 | -6 (부가 효과) | 인기 스탯도 상승 |
| 운동 (가벼운) | -3 (부가 효과) | 체력 스탯도 상승 |

> 휴식은 피로를 크게 줄이지만 **다른 스탯이 성장하지 않는다.**
> 이것이 "쉬어야 하는데 쉴 수 없는" 딜레마를 만든다.

**피로 비례 자연 회복 상세 (v3):**

기존의 고정 자연 회복(-8)을 **피로 잔량 비례 회복**으로 변경한다.

```
주간 자연 회복량 = 피로 잔량 × 0.15 (최소 -3, 최대 -12)
```

| 현재 피로 | 자연 회복량 | 설명 |
|-----------|-------------|------|
| 0 ~ 19 | **-3** (최소 보장) | 피로가 낮아도 최소한의 회복 |
| 20 | -3 | 20 × 0.15 = 3 |
| 40 | **-6** | 40 × 0.15 = 6 |
| 60 | **-9** | 60 × 0.15 = 9 |
| 80 이상 | **-12** (최대 제한) | 80 × 0.15 = 12 |

**정서적 지지 가정 보정:**
- 기존: 고정 추가 -3 (총 -11)
- v3: 최종 회복량에 고정 **+2** 추가 (최대 -12 유지)
- 예시: 피로 40 → 기본 -6 + 정서적지지 -2 = **-8**
- 예시: 피로 80 → 기본 -12 + 정서적지지 → **-12** (최대 제한)

> **설계 의도:** 피로가 높을수록 몸이 "알아서" 회복하려 하지만 한계가 있다.
> 피로가 낮을 때는 회복이 느려 "완전 회복"이 쉽지 않고,
> 피로가 높을 때는 빠르게 회복되어 "극한 피로 → 즉시 붕괴"를 방지한다.
> 전략적으로 "약간 피로한 상태를 유지하며 돌리는" 플레이가 최적해가 된다.

### 3-4. 피로 단계별 효과

| 피로 구간 | 상태명 | 보정 | 효과 |
|-----------|--------|------|------|
| 0~19 | 양호 | ×1.0 | 정상 플레이, 보정 없음 |
| 20~34 | 경미 | ×0.95 | UI에 경미 경고 표시 |
| 35~49 | 주의 | ×0.85 | UI에 경고 표시 |
| 50~64 | 과로 | ×0.75 | 멘탈 주당 -2 자동 감소 |
| 65~79 | 위험 | ×0.6 | 멘탈 주당 -4 자동 감소, 병결 확률 발생 |
| 80~89 | 한계 | ×0.4 | 멘탈 주당 -5 자동 감소, 병결 확률 높음 |
| 90~100 | 붕괴 | ×0.2 | 멘탈 주당 -6, 강제 휴식 이벤트 발생 |

> **피로 → 멘탈 연결이 핵심이다.**
> 피로가 쌓이면 멘탈이 깎이고, 멘탈이 낮으면 모든 효율이 떨어진다.
> "무리 → 피로 → 멘탈 저하 → 효율 하락 → 더 무리 → 번아웃" 악순환을 만든다.

### 3-5. 피로와 멘탈의 연결

```
[피로 누적] → [멘탈 자동 감소] → [멘탈 상태 전환] → [활동 제한/보정 변화]
     ↑                                                        ↓
     ← ← ← ← ← [효율 저하로 더 무리하게 됨] ← ← ← ← ← ← ←
```

### 3-6. 소수점 누적 시스템

기존에는 매주 성장량 계산 후 소수점을 버렸으나(floor), 이는 저성장 구간에서 실질 성장이 0이 되는 문제를 야기했다.
이를 해결하기 위해 **소수점 누적 시스템**을 도입한다.

| 항목 | 규칙 |
|------|------|
| 소수점 처리 | 버리지 않고 **누적** |
| 반영 조건 | 누적값이 1.0 이상이 되면 정수부(1)를 스탯에 반영 |
| 이월 | 반영 후 남은 소수점은 다음 주로 이월 |

**계산 예시:**

| 주차 | 해당 주 성장량 | 누적값 | 스탯 반영 | 이월 |
|------|--------------|--------|-----------|------|
| 1주차 | +0.7 | 0.7 | 0 | 0.7 |
| 2주차 | +0.8 | 1.5 | +1 | 0.5 |
| 3주차 | +0.6 | 1.1 | +1 | 0.1 |
| 4주차 | +0.7 | 0.8 | 0 | 0.8 |

> **설계 의도:** 고스탯 구간(85+)이나 상태가 나쁜 경우에도 미세한 성장이 누적되어
> 결국 반영된다. "노력이 사라지지 않는다"는 체감을 제공한다.
> 각 스탯 및 하위 스킬별로 별도의 누적값을 관리한다.

## 4. 멘탈 상태 전환 시스템

### 4-1. 4단계 상태

```
정상 ←→ 피로 ←→ 번아웃
                    ↓
                  각성 (특수 조건)
```

### 4-2. 전환 조건 (구체적 수치)

#### 정상 → 피로

| 조건 | 값 |
|------|-----|
| 멘탈 수치 | 50 → 30 미만으로 하락 시 |
| 또는 | 피로 50 이상이 3주 연속 유지 시 |
| 전환 연출 | "요즘 뭘 해도 재미없다..." 독백 컷신 |

#### 피로 → 정상 (회복)

| 조건 | 값 |
|------|-----|
| 멘탈 수치 | 30 → 50 이상으로 회복 시 |
| 그리고 | 피로 30 미만 유지 2주 |
| 회복 방법 | 휴식, 친구 관계, 상담 이벤트 |

#### 피로 → 번아웃

| 조건 | 값 |
|------|-----|
| 멘탈 수치 | 30 → 15 미만으로 하락 시 |
| 또는 | 피로 상태에서 4주 연속 피로 70 이상 유지 시 |
| 또는 | 피로 상태에서 중요 이벤트 실패 (시험 하위 30% 등) |
| 전환 연출 | 화면 채도 저하 + "...더 이상 못하겠다" 컷신 |

#### 번아웃 효과 (구체적 제한)

| 항목 | 제한 내용 |
|------|-----------|
| 슬롯 | 자유 슬롯 중 1개 강제 휴식으로 고정 |
| 학업 활동 | 선택 가능하지만 성장량 ×0.3 |
| 관계 활동 | 선택 불가 ("만날 기운이 없다") |
| 재능 활동 | 선택 가능하지만 성장량 ×0.3 |
| 체력 활동 | 가벼운 운동만 가능 |
| 이벤트 | 부정적 선택지만 나타남 (짜증, 회피, 거절) |
| 지속 기간 | 최소 4주 (회복 조건 충족해도 4주는 유지) |
| NPC 반응 | 친한 NPC가 걱정 이벤트 발생 |

#### 번아웃 → 피로 (회복)

| 조건 | 값 |
|------|-----|
| 최소 지속 | 4주 |
| 피로 | 30 미만으로 감소 |
| 멘탈 | 20 이상으로 회복 |
| 회복 촉진 | 상담 이벤트 선택 시 회복 속도 ×2 |
| 회복 촉진 | 친밀 NPC의 위로 이벤트 수락 시 멘탈 +10 |

#### 각성 조건 (특수 상태)

**각성은 "극한 상황에서의 돌파"를 표현하는 고위험·고보상 상태다.**

| 조건 | 값 |
|------|-----|
| 진입 조건 1 | 피로 상태 + 특정 스탯 70 이상 + 해당 스탯 관련 대회/시험 이벤트 발생 |
| 진입 조건 2 | 번아웃 상태에서 NPC 각성 이벤트 발생 시 "그래도 해보겠다" 선택 |
| 진입 조건 3 | 멘탈 25~35 구간 + 피로 60 이상 + 재능 60 이상 + 재능 활동 선택 |
| 확률 | 조건 충족 시 50% 확률로 각성 진입 (실패 시 번아웃으로 전환) |
| 지속 기간 | 4주 고정 |

#### 각성 효과

| 항목 | 효과 |
|------|------|
| 트리거 스탯 성장 | ×2.0 (폭발적 성장) |
| 기타 스탯 성장 | ×0.5 (집중의 대가) |
| 피로 증가 | ×1.5 (무리의 대가) |
| 멘탈 변동 없음 | 각성 기간 동안 멘탈 고정 |
| 종료 후 | 피로 +20, 멘탈 -15 (반동) |
| 재발동 | 같은 플레이에서 최대 3회까지 |

> **각성의 도박성:** 실패 시 번아웃, 성공해도 반동이 크다.
> "여기서 무리하면 성장할 수 있지만, 대가를 치른다"는 선택을 강제한다.

### 4-3. 멘탈 회복 수단 정리

| 수단 | 멘탈 회복량 | 조건 |
|------|-------------|------|
| 휴식 (1슬롯) | +3 | 무조건 |
| 푹 쉬기 (2슬롯) | +8 | 주말 전체 사용 |
| 친구와 놀기 | +4 | 친밀 NPC 필요 |
| 상담 이벤트 | +10 | 학교 상담사, 4주에 1회 |
| 운동 (가벼운) | +2 | 무조건 |
| 취미 활동 | +3 | 재능 활동 중 일부 |
| 부모와 대화 | +5 또는 -5 | 부모 관계에 따라 양방향 |
| 연애 성공 이벤트 | +8 | NPC 관계 조건 |
| 연애 실패 이벤트 | -10 | NPC 관계 조건 |

### 4-4. 멘탈 회복 감쇠 (v3)

> "행복의 정점을 유지하는 것이 스탯 올리기보다 어렵다."

멘탈이 높을수록 회복 효율이 떨어진다. 위 표의 멘탈 회복량에 아래 감쇠 배율을 적용한다.

| 현재 멘탈 구간 | 회복 배율 |
|----------------|-----------|
| 0 ~ 69 | ×1.0 (기본) |
| 70 ~ 79 | ×0.8 |
| 80 ~ 89 | ×0.55 |
| 90 ~ 100 | ×0.3 |

**계산 예시:**
- 멘탈 50에서 "친구와 놀기" → +4 × 1.0 = +4
- 멘탈 75에서 "친구와 놀기" → +4 × 0.8 = +3.2 → 소수점 누적(§3-6)
- 멘탈 92에서 "상담 이벤트" → +10 × 0.3 = +3

> **설계 의도:** 멘탈 80 이상을 유지하는 것은 매우 어렵다.
> 고멘탈 유지를 위해서는 지속적인 투자가 필요하며,
> "멘탈을 A등급으로 유지하는 것 자체가 하나의 전략"이 된다.
> 이는 멘탈이 단순한 버퍼가 아니라 적극적 관리 대상임을 강조한다.

> 동네 체육관과 도서관은 무료로 접근 가능한 현실적 선택지다.

### 정서적 지지 가정 전용 활동

| 활동 | 슬롯 | 피로 | 학업 | 인기 | 재능 | 멘탈 | 체력 | 용돈 | 개방 조건 | 비고 |
|------|------|------|------|------|------|------|------|------|----------|------|
| 부모와 같이 공부 | 1 | +3 | +2 | 0 | 0 | +1 | 0 | 0 | 정서적 지지 강점 선택 | 학원의 저비용/저피로 대안. 학업 효율은 낮지만 멘탈까지 회복. 부모 친밀도 +1 |
| 가족 저녁시간 | 1 | -2 | 0 | 0 | 0 | +3 | 0 | 0 | 정서적 지지 강점 선택 | 피로 회복 + 멘탈 회복. 주 1회 제한. 부모 친밀도 +2 |
| 공원 산책 | 1 | -1 | 0 | 0 | 0 | +2 | +1 | 0 | 없음 (범용) | 가장 약한 휴식이지만 체력 미세 상승. 누구나 사용 가능 |

> **설계 의도:**
> - **"부모와 같이 공부"**는 학원(학업+3, 피로+8, 용돈-3)의 저효율/저비용/저피로 대안이다. 학업 성장은 +2로 학원(+3)보다 낮지만, 피로가 +3(vs +8)으로 절반 이하이고, 용돈 소모가 0이며, 멘탈 +1까지 얻는다. 저소득 가정에서 "느리지만 꾸준하게" 학업을 올릴 수 있는 핵심 활동.
> - **정서적 지지 가정의 플레이 정체성**: "느리지만 꾸준한 빌드". 매주 학업 +2씩 쌓으면서 멘탈과 피로를 안정적으로 관리하는 장기전 스타일. 재력 가정이 학원으로 빠르게 치고 올라가는 동안, 정서적 지지 가정은 무너지지 않는 기반을 다진다.
> - **재력 가정은 이 활동을 쓸 이유가 없다.** 학원(학업+3)이 더 효율적이고, 용돈 8만원이면 학원비(-3)를 충분히 감당할 수 있다. 따라서 이 활동은 재력 가정과 저소득 가정의 플레이 경험을 자연스럽게 차별화한다.
> - **핵심 설계 철학 전환**: "저소득 = 하드모드"가 아니라 **"저소득 = 안정형 빌드"**. 부유한 집은 "속도를 위해 멘탈을 포기"하는 공격형 빌드, 가난한 집은 "안정을 위해 속도를 포기"하는 방어형 빌드. 둘 다 유효한 전략이며, 각각 다른 형태의 "포기의 고통"을 경험한다.
> - **"공원 산책"**은 범용 활동으로, 모든 가정에서 사용할 수 있는 가장 약한 휴식이다. 휴식(-12 피로)보다 훨씬 약하지만 체력 +1이 붙어 있어 체력 관리와 가벼운 피로 회복을 동시에 원하는 상황에서 선택지가 된다.

### 5-6. 휴식 계열

| 활동 | 슬롯 | 피로 | 학업 | 인기 | 재능 | 멘탈 | 체력 | 용돈 | 비고 |
|------|------|------|------|------|------|------|------|------|------|
| 휴식 | 1 | -12 | 0 | 0 | 0 | +3 | +1 | 0 | 피로 대폭 회복 |

## 6. 밸런스 검증 시뮬레이션

> ⚠ **주의 (v3):** 아래 시뮬레이션은 v2 기준(고정 자연회복 -8)으로 작성되었다.
> v3에서는 피로 비례 회복(잔량 15%, 최소 -3, 최대 -12)이 적용되므로
> 실제 피로 흐름이 달라진다. 대략적인 경향은 유사하나, 정밀 수치는 재시뮬 필요.

### 6-1. 공부 특화 플레이 (360주)

```
매주 구성: 학원 ×2 + 독학 ×1 + 휴식 ×1 + 자유 ×1
주당 피로: +8+8+6-12+α = ~+10 → 자연회복 -8 = 순증 +2
→ 약 10주마다 피로 20 도달, 이전보다 여유로운 피로 관리

예상 최종 스탯:
- 학업: ~85 (A 하위)
- 인기: ~10 (E, 관계 미관리로 하락)
- 재능: ~8 (E)
- 멘탈: ~35 (피로 상태 왔다갔다)
- 체력: ~20 (D, 운동 미관리)
```

### 6-2. 밸런스 플레이 (360주)

```
매주 구성: 독학 ×1 + 친구 ×1 + 가벼운운동 ×1 + 창작 ×1 + 휴식 ×1
주당 피로: +6+3+2+4-12 = +3 → 자연회복 -8 = 순증 -5 (피로 매우 안정)

예상 최종 스탯:
- 학업: ~55 (C)
- 인기: ~50 (C)
- 재능: ~45 (C)
- 멘탈: ~65 (정상 유지)
- 체력: ~45 (C)
→ 모든 스탯 C 근처: "평범하지만 안정적인 학생"
```

### 6-3. 번아웃 플레이 (무리하는 경우)

```
매주 구성: 학원 ×2 + 시험벼락치기 ×1 + 독학 ×2 (휴식 없음)
주당 피로: +8+8+14+6+6 = +42 → 자연회복 -8 = 순증 +34
→ 3주 만에 피로 100 도달
→ 4~5주 차에 번아웃 진입
→ 강제 휴식으로 4주 손실
→ 실질적으로 더 비효율적
```

> **결론:** 쉬지 않고 달리는 것은 오히려 손해다.
> 이 시스템은 "적절한 휴식"의 가치를 수치로 증명한다.

## 7. 하위 스킬 (세부 성장 체감)

### 7-1. 하위 스킬 구조

메인 5대 스탯 아래에 세부 하위 스킬이 존재하며, 활동의 결과가 더 구체적으로 체감된다.

```
학업 ──┬── 문과 (국어/사회/영어)
       └── 이과 (수학/과학)

재능 ──┬── 음악
       ├── 미술
       ├── 체육
       └── 기술 (코딩/공학)

인기 ──┬── 친화력 (1:1 관계 형성 속도)
       └── 존재감 (집단 내 위치/영향력)

멘탈 ──── (단일 — 하위 없음)
체력 ──── (단일 — 하위 없음)
```

### 7-2. 하위 스킬 기본 규칙

| 항목 | 규칙 |
|------|------|
| 범위 | 0 ~ 100 (메인 스탯과 동일) |
| 상한 제약 | 하위 스킬은 **메인 스탯 수치를 초과할 수 없다** (메인 스탯 = 상한 캡) |
| 메인 스탯과의 관계 | 메인 스탯 = 소속 하위 스킬들의 가중 평균 (반올림) |
| 시작값 | 메인 스탯 기본값과 동일하게 시작 (학업 30이면 문과 30, 이과 30) |
| UI 표시 | 메인 5대 스탯을 크게 표시, 탭/터치하면 하위 스킬 확장 표시 |

### 7-3. 하위 스킬 성장 방식

활동이 메인 스탯을 올릴 때, **해당 활동과 연관된 하위 스킬은 더 큰 폭으로 상승**한다.

| 활동 예시 | 메인 스탯 변동 | 하위 스킬 변동 |
|-----------|---------------|---------------|
| 학원(수학) | 학업 +3 | 이과 +5 |
| 학원(영어) | 학업 +3 | 문과 +5 |
| 스터디 그룹 | 학업 +2 | 문과 +2, 이과 +2 (범용) |
| 예체능 레슨(음악) | 재능 +3 | 음악 +5 |
| 예체능 레슨(미술) | 재능 +3 | 미술 +5 |
| 학교 체육부 | 재능 +2, 체력 +4 | 체육 +4 |
| 코딩 독학 | 학업 +1, 재능 +2 | 이과 +2, 기술 +4 |
| 친구와 놀기 | 인기 +3 | 친화력 +4 |
| 동아리 활동 | 인기 +2 | 존재감 +3 |
| SNS 활동 | 인기 +2 | 존재감 +3 |
| 연락하기 | 인기 +1 | 친화력 +2 |

> **설계 의도:** 같은 "학업 활동"이라도 무엇을 하느냐에 따라 문과/이과 성향이 갈린다.
> 플레이어는 자연스럽게 자신의 진로 방향을 선택하게 된다.

### 7-4. 하위 스킬의 용도

| 용도 | 설명 | 예시 |
|------|------|------|
| 특정 학교 진학 조건 | 특목고/자사고 입시, 대학 학과별 요구치 | 과학고: 이과 75 이상 필요 |
| 엔딩 분기 조건 | 특정 엔딩 달성에 하위 스킬 수치 요구 | "천재 프로그래머" 엔딩: 기술 85 이상 |
| NPC 이벤트 트리거 | 하위 스킬 수치에 따라 특정 NPC 이벤트 해금 | 음악 60 이상일 때 밴드부 NPC 이벤트 |
| 대회/오디션 판정 | 세부 분야별 대회 결과에 하위 스킬 직접 반영 | 수학 올림피아드: 이과 수치로 판정 |
| 선택지 해금 | 하위 스킬이 일정 수준이면 새로운 선택지 등장 | 기술 40 이상: "해커톤 참가" 선택지 등장 |

### 7-5. 인기 vs NPC 친밀도 — 구분 정의

인기의 하위 스킬인 **친화력/존재감**과 개별 **NPC 친밀도**는 별개의 시스템이다.

| 구분 | 인기(존재감) | NPC 친밀도 |
|------|-------------|-----------|
| 성격 | 학교 전체에서의 영향력/위상 | 특정 NPC 한 명과의 관계 깊이 |
| 영향 범위 | 전체 학생, 교사, 학교 행사 | 해당 NPC 1명 |
| 올리는 방법 | 동아리, SNS, 대회 입상, 학교 행사 | 1:1 대화, 함께 활동, 선물, 이벤트 선택 |
| 주요 효과 | 신규 NPC 첫인상 보정, 학교 행사 주도권, 랭킹 | NPC 개인 이벤트 해금, 사이드 엔딩, 개인 보너스 |
| 감소 속도 | 느림 (주당 -1.0) | 빠름 (미접촉 시 주당 -1.5) |

**조합에 따른 유형:**

| 인기(존재감) | NPC 친밀도 | 유형 | 학교생활 묘사 |
|-------------|-----------|------|-------------|
| 높음 | 높음 | 인싸 중의 인싸 | 누구나 아는 인기인이면서 진짜 친구도 많음 |
| 높음 | 낮음 | 유명하지만 외로운 아이 | 반에서 존재감은 크지만 진심으로 털어놓을 친구가 없음 |
| 낮음 | 높음 | 소수정예 인간관계 | 학교에선 조용하지만 절친 2~3명과 깊은 유대 |
| 낮음 | 낮음 | 은둔형 | 학교에서 투명인간, 관계 이벤트 거의 발생 안 함 |

> **설계 의도:** "인기가 높다 ≠ 친구가 많다"를 표현한다.
> 인기(존재감)는 학교 사회에서의 위치, 친밀도는 개인 간 유대.
> 둘 다 가치 있지만 올리는 방법이 다르므로, 플레이어는 어느 쪽에 시간을 쓸지 선택해야 한다.

## 8. 마일스톤 (성장 체감 이벤트)

메인 스탯이 핵심 구간에 도달하면 **마일스톤 이벤트**가 발생한다.
마일스톤은 성장의 체감을 높이고, 다른 시스템과의 연결고리를 만든다.

> 원칙:
> - 각 마일스톤은 **최초 1회만** 발생한다 (스탯이 내려갔다 다시 올라도 재발동 안 함)
> - 마일스톤 도달 시 알림 + 짧은 컷신/독백이 표시된다
> - 보너스는 작지만 체감 가능한 수준으로 설계한다

### 8-1. 학업 마일스톤

| 스탯 | 구간 | 마일스톤 | 효과 |
|------|------|---------|------|
| 학업 | 30 돌파 | "수업 내용이 들리기 시작했다" — 수업 시간에 졸지 않게 됨 | 독학 효율 +10% (기본 성장량 보정) |
| 학업 | 50 돌파 | "중간권 진입" — 선생님이 이름을 기억하기 시작함 | 학업 관련 선택지 +1 (인강 추천, 스터디 초대 등) |
| 학업 | 70 돌파 | "상위권" — 반에서 공부 잘하는 애로 인식됨 | 인기(존재감) +3, 과외(하는) 활동 조기 해금 |
| 학업 | 85 돌파 | "전교 상위 5%" — 부모님이 감동하는 이벤트 발생 | 부모 관계 +10, 용돈 +2만원/주, 특목고/자사고 지원 자격 |

### 8-2. 인기 마일스톤

| 스탯 | 구간 | 마일스톤 | 효과 |

| 인기 | 30 돌파 | "아는 사람이 생겼다" — 급식 시간에 혼자 먹지 않게 됨 | NPC 친밀도 자연 감소 속도 -0.5/주 |

| 멘탈 | 85 돌파 | "강철 멘탈" — 주변에서 멘탈이 강하다고 인정함 | 각성 실패 시 번아웃 대신 피로 상태로 전환, NPC 상담 이벤트 해금 (도움 주는 쪽) |

### 8-5. 체력 마일스톤

| 스탯 | 구간 | 마일스톤 | 효과 |
|------|------|---------|------|

| 체력 | 50 돌파 | "기본기 완성" — 체력장에서 중위권 | 병결 이벤트 확률 0%, 가벼운 운동의 멘탈 회복 +1 |
| 체력 | 70 돌파 | "체력왕" — 체육대회에서 활약, 반에서 운동 잘하는 애로 인식 | 인기(존재감) +3, 대회 훈련 활동 효율 +15% |
| 체력 | 85 돌파 | "철인 체력" — 밤샘해도 다음날 멀쩡함 | 피로 자연 회복 +2 (주당 -8 → -10), 극한 활동 피로 비용 -20% |

> **마일스톤은 플레이어에게 "지금 잘하고 있다"는 피드백을 준다.**
> 숫자만 올라가는 것이 아니라, 학교생활 속 구체적인 변화를 보여줌으로써
> 스탯 투자의 의미를 체감하게 만든다.
> 또한 마일스톤 보너스가 다른 시스템(부모, NPC, 진학)과 연결되어
> "하나를 올리면 다른 곳에서도 변화가 생긴다"는 유기적 성장을 경험하게 한다.

## 9. 내부 계산용 세부 스탯 (UI에 직접 노출 안 함)

| 분류 | 세부 스탯 | 영향받는 보이는 스탯 | 용도 |

| 학업 성향 | 문과 지수 / 이과 지수 | 학업 | 문·이과 분기, 대학 진로 판정 |
| 예체능 성향 | 음악 / 미술 / 체육 | 재능 | 재능 세부 분기, 대회 판정 |
| 특기 | 코딩 / 경영 / 창작 / 요리 등 | 재능 | 엔딩 분기, 특수 이벤트 |
| NPC 호감도 | 각 NPC별 0~100 | 인기 (평균값) | 관계 이벤트 트리거 |
| 부모 관계 | 0~100 | 강제 슬롯 수, 용돈 | 가정 이벤트, 자유도에 영향 |
| 자립도 | 0~100 (은닉) | 강제 슬롯 저항 | 알바, 자기 결정 이벤트로 상승 |
| 평판 | 모범생/일진/은둔형 등 태그 | 이벤트 분기 | NPC 반응, 교사 태도에 영향 |

> 원칙: UI는 단순 (5개 스탯 + 등급), 내부는 복잡 (20개 이상 변수)

## 10. 수식 요약

### 최종 성장 공식 (하이브리드 연산)

```
// ─── 핵심 변경: 순수 곱셈 → 하이브리드(가산 + 곱셈) 방식 ───
//
// [구 공식] 최종 성장량 = 기본값 × 구간감쇠 × 체력보정 × 멘탈보정 × 피로보정
// [신 공식] 최종 성장량 = (기본값 + 버프가산) × 상태보정
//
// 버프가산 = 루틴보너스 + 정보력보너스 + NPC시너지 등
//   → 기존에 곱셈으로 적용되던 긍정적 버프들은 이제 기본값에 **더한다**.
//   → 이렇게 하면 상태가 나빠도 버프 효과가 완전히 사라지지 않는다.
//
// 상태보정 = 구간감쇠 × 체력보정 × 멘탈보정 × 피로보정
//   → 상태 관련 보정은 여전히 곱셈이지만, 8단계 부드러운 곡선을 사용한다.
//   → 최저 보장: 상태보정은 아무리 낮아도 x0.15 이하로 내려가지 않는다.
//     (절대 0에 수렴하지 않아 완전한 무성장 구간이 사라짐)

// 메인 스탯 성장
버프가산 = 루틴보너스 + 정보력보너스 + NPC시너지  // 가산 합계
상태보정_원본 = 구간_감쇠(현재_스탯)
             × 체력_보정(현재_체력)
             × 멘탈_보정(현재_멘탈)
             × 피로_보정(현재_피로)
             × 반복_감쇠(같은_활동_반복_횟수)
상태보정 = max(상태보정_원본, 0.15)  // 최저 보장 x0.15

실제_성장량_raw = (기본_성장량 + 버프가산) × 상태보정

// ─── 소수점 누적 시스템 (floor 대신 누적) ───
// 매주 성장량의 소수점은 버리지 않고 누적한다.
// 누적값이 1.0 이상이 되면 1을 반영하고 나머지를 이월한다.
누적값 += 실제_성장량_raw
실제_성장량 = floor(누적값)
누적값 -= 실제_성장량   // 나머지 소수점 이월
// 예시: 주1 +0.7 (누적 0.7) → 주2 +0.8 (누적 1.5 → +1 반영, 0.5 이월)

// 하위 스킬 성장 (메인 스탯과 별도 계산, 동일한 하이브리드 방식)
하위_상태보정 = max(
    구간_감쇠(현재_하위_스킬) × 체력_보정(현재_체력)
    × 멘탈_보정(현재_멘탈) × 피로_보정(현재_피로),
    0.15
)
하위_성장량_raw = (하위_기본_성장량 + 버프가산) × 하위_상태보정
// 하위 스킬도 동일하게 소수점 누적 적용
하위_누적값 += 하위_성장량_raw
하위_스킬_성장량 = floor(하위_누적값)
하위_누적값 -= 하위_스킬_성장량
하위_스킬 = min(하위_스킬 + 하위_스킬_성장량, 메인_스탯)  // 메인 스탯이 상한

// 피로 변동 (체력 피로 내성 보정 적용)
실제_피로_증가 = round(기본_피로_증가 × 체력_피로축적_보정(현재_체력))
실제_피로_회복 = round(기본_피로_회복 × 체력_휴식회복_보정(현재_체력))
```

### 주간 스탯 업데이트 순서

```
1. 주 시작: 피로 자연 회복 (-8)
2. 피로 상태에 따른 멘탈 자동 감소 적용
3. 멘탈 상태 전환 체크 (정상↔피로↔번아웃)
4. 플레이어 활동 선택 (5슬롯)
5. 각 활동별 스탯 변동 계산 (보정 적용)
6. 각 활동별 피로 변동 계산 → 체력 피로 내성 보정 적용 → 최종 피로 변동 적용
7. 미활동 스탯 자연 감소 적용
8. 용돈 변동 적용
9. 마일스톤 도달 체크 및 이벤트 발생
10. 부모 개입 안전망 체크 (피로 85+ 또는 멘탈 15 이하 시)
11. 이벤트 트리거 체크
12. 결과 표시
```

## 11. 부모 개입 안전망 (Pity System)

플레이어의 상태가 극단적으로 나빠졌을 때, 부모 유형에 따라 **강제 개입 이벤트**가 발동된다.
이는 "바닥을 치더라도 완전히 무너지지 않는" 최소한의 안전장치 역할을 한다.

### 11-1. 발동 조건

| 조건 | 임계값 |
|------|--------|
| 피로 | 85 이상 |
| 멘탈 | 15 이하 |
| 발동 시점 | 주간 업데이트 10단계에서 체크 |

> 피로 85 이상 **또는** 멘탈 15 이하 중 하나만 충족해도 발동한다.

### 11-2. 부모 유형별 개입 효과

| 부모 특성 | 개입 이벤트 | 효과 |
|-----------|------------|------|
| 재력 | 보약/영양제 지원 | 피로 -20 |
| 정보력 | 스케줄 재조정 | 다음 주 강제 슬롯 전부 해제 (자유 슬롯으로 전환) |
| 정서적 지지 | 가족 외식/심층 대화 | 멘탈 +15 |
| 엄격함 | 부모도 위기 인식 | 1주간 압박 해제 (강제 슬롯 없음) |
| 시간/자유 | 이미 자유로우므로 발동 변형 | 친구 또는 이웃 어른 조언 이벤트 (멘탈 +10) |
| 유전자 | 타고난 회복력 발현 | 1주간 피로 자연회복 2배 (-8 → -16) |

> **복합 특성 가정:** 부모가 여러 특성을 보유한 경우, 현재 상태에 가장 적합한 개입이 우선 발동된다.
> 예) 피로 90 + 멘탈 20인 경우 → 재력 특성의 "피로 -20"이 우선.
> 예) 피로 50 + 멘탈 10인 경우 → 정서적 지지의 "멘탈 +15"가 우선.

### 11-3. 발동 제한

| 항목 | 제한 |
|------|------|
| 학기당 최대 발동 횟수 | **2회** |
| 학기 단위 | 초6 전반기/후반기, 중1~고3 각 학기 (1학기/2학기) |
| 재사용 대기 | 같은 학기 내에서 최소 4주 간격 |

> **설계 의도:** 안전망은 완전한 붕괴를 방지하는 최후의 보루이지,
> 남발할 수 있는 자원이 아니다. 학기당 2회 제한으로 인해
> 플레이어는 안전망에 의존하지 않고 스스로 상태를 관리해야 한다.
> "부모가 도와줄 수 있지만, 매번은 아니다"라는 현실감을 유지한다.
