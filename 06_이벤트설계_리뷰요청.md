# 06_이벤트설계 리뷰 요청서

> **LIFE TRACK: 선택의 결과** — 이벤트 설계 초안 완성
> 디렉터 관점에서 검토 부탁드립니다.

---

## 배경

기존 시스템 설계(스탯/부모/관계/NPC/게임루프)가 완료된 상태에서,
두 디렉터(GPT/Gemini)가 공통으로 지적한 **"감정 밀도 부족"**과 **"첫 10분 미설계"**를 해결하기 위해
06_이벤트설계를 작성했습니다.

**이전 리뷰에서 채택된 방향:**
- 첫 10분 별도 설계 (GPT + Gemini 공통 최우선)
- 스탯→지위 연출: NPC 톤 변화 (Gemini 제안)
- 에필로그 카드 / 후회 지수 (GPT + Gemini 공통)
- 정보 점진적 공개 (GPT 제안)
- 회고 카드는 매주가 아닌 구간 전환점에서만 (Claude 필터링)
- 피로 학년 리셋: 기각 (v3 비례회복 + 방학으로 충분)
- 히든학원 전용 NPC: 보류 (텍스트 연출로 대체)

## 문서 구조 (10개 섹션)

### 1. 첫 10분 설계

**0~2분: 부모 선택 = 생활 장면**

6개 강점을 숫자 나열이 아닌 짧은 생활 장면으로 보여줍니다.
플레이어는 장면 2개를 골라 가정을 결정합니다.

| 장면 | 핵심 이미지 |
|------|-----------|
| 재력 | "학원 차가 교문 앞에 와 있다. 간식비가 넣어져 있다." |
| 정서적 지지 | "아빠가 국을 떠주며 묻는다: 오늘 학교 어땠어?" |
| 시간/자유 | "집에 왔는데 아무도 없다. 냉장고에 메모: 밥 시켜 먹어." |
| 엄격함 | "현관문이 열리자마자: 손 씻고 바로 숙제." |
| 유전자 | "체육 시간. 별로 연습한 적 없는데 달리기가 빠르다." |
| 정보력 | "엄마가 전화 중이다. 나는 뭘 정했는지도 모르는데 학원이 바뀌어 있다." |

**2~5분: NPC 3명만 등장**

| NPC | 첫 인상 | 역할 |
|-----|--------|------|
| 한지훈 | "야, 너 몇 동이야?" — 같은 아파트 동, 같은 라인 | 편안함 |
| 윤서아 | 급식 줄에서 눈이 마주침. 먼저 시선을 돌림 | 호기심 |
| 강민재 | "집중 좀 해. 지는 거 싫거든." | 긴장감 |

나머지 7명은 2~4주차에 점진적 등장.

**5~10분: 첫 주말 → 첫 후회**

첫 주말에 2슬롯을 선택한 뒤, **뭘 골라도 약간의 아쉬움**을 주는 연출:
- 지훈이랑 놀았으면 → "서아와 더 긴 대화가 가능했던 기회를 놓침"
- 공부했으면 → "단톡방에 지훈이가 올린 사진. 나만 없다."

> 핵심: "포기의 고통"이 이 게임의 본질이라는 걸 첫 10분에 각인시킵니다.

**정보 점진적 공개:**

| 시점 | 공개 |
|------|------|
| 시작 | 학업, 멘탈, 피로 (3개만), 용돈, NPC 이름 |
| 2주차 | 체력, 인기 ("체육 시간에 힘들었다" → 체력 등장) |
| 4주차 | 루틴 시스템 ("매주 같은 걸 하니까 익숙해졌다") |
| 첫 시험 | 등급(E~A), 교내 순위 |
| 재능 3회 반복 | 하위 스킬 ("너 음악 감각 있는 것 같아") |
| 첫 방학 | 방학 목표, 묶음 활동 |
| 중학교 | SNS, 평판 태그 |

### 3. 딜레마 이벤트 20개

모든 딜레마는 **"정답이 없는 선택"** — 뭘 골라도 무언가를 잃고, 무언가를 얻습니다.

#### 초등 (3개)

| # | 이름 | 핵심 갈등 |
|---|------|----------|
| 1 | "같이 놀자" vs "학원 가야 해" | 관계 vs 학업 루틴 |
| 2 | "떨어뜨린 지우개" | 서아와의 첫 접점. 행동 vs 방관 |
| 3 | "체육대회 조 편성" | 편한 친구(지훈) vs 이기는 팀(민재) |

#### 중학교 (8개)

| # | 이름 | 핵심 갈등 |
|---|------|----------|
| 4 | "새 교복, 새 시작" | 적극적 vs 소극적 vs 옛 친구 의존 |
| 5 | "시험 성적표" | 부모에게 보여주기 — 솔직/숨김/선제 방어 |
| 6 | "학원 vs 동아리" | 루틴 보너스 유지 vs 새 성장 경로 (+ 부모 갈등) |
| 7 | "친구의 비밀" (수빈 이혼) | 비밀 유지 vs 공유 vs 배신 |
| 8 | "수학여행 룸메이트" | 4명 자리에 친한 친구가 5명. 누굴 뺄 것인가 |
| 9 | "지훈이네 아버지" | 병문안(시간 투자) vs 카톡(최소) vs 무관심 |
| 10 | "질풍노도기" | 부모 충돌 — 참기/대들기/집 나가기 |
| 11 | **"고등학교 선택"** | 일반고(친구) vs 특목고(꿈) — 게임 최대 분기점 |

#### 고등학교 (9개)

| # | 이름 | 핵심 갈등 |
|---|------|----------|
| 12 | "첫 야자" | 참여(학업+피로) vs 빠지기(평판 리스크) |
| 13 | "민재의 슬럼프" | 라이벌이 무너졌을 때 — 손 내밀기 vs 모른 척 |
| 14 | "지훈이의 알바 제안" | 같이 알바(용돈+관계, 학업-) vs 거절 vs "하지 마" |
| 15 | "SNS 사건" (수빈 비밀 유출) | 공개 옹호 vs 조용히 알림 vs 방관 vs 영향력 행사 |
| 16 | "예린이의 정보" (추천서) | 진학 기회(주말 소비) vs 포기(자립) |
| 17 | "서아의 갈림길" (미대 vs 부모) | 응원 vs 현실 조언 vs 함께 방법 찾기 |
| 18 | "루나의 정체" (오프라인 만남) | 만남(현실 직면) vs 안 만남(온라인 유지) |
| 19 | **"수능 전날"** | 공부/잠/지훈 전화 — 가장 비효율적 선택이 가장 아름다운 회상이 됨 |
| 20 | **"수능이 끝나고"** | 가장 먼저 연락할 사람 1명 선택. 7년의 관계가 한 번에 압축됨 |

### 4. 마일스톤 반응 이벤트

02_스탯시스템의 마일스톤(20개)이 달성될 때, **NPC 반응 대사**가 추가됩니다.

예시:
- 학업 50 돌파 → 민재가 흘끗 쳐다봄. 서아: "시험 어땠어?"
- 학업 85 돌파 → 민재: "...대단하다." 단톡방: "쟤 공부 미쳤대"
- 인기 30 돌파 → "야 여기 앉아!" (이름 모르는 반 친구)
- 인기 85 돌파 → 다른 반 애: "너 혹시 OO이야?" SNS 팔로워 급증

> 원칙: 숫자 상승이 아니라 "세상이 나를 다르게 보기 시작한다"는 체감

### 5. 안전망 이벤트 (서사 포장)

부모 개입 안전망(피로 85+ / 멘탈 15 이하)을 **가정 온도**로 연출:

| 부모 | 멘탈 위기 | 피로 위기 |
|------|----------|----------|
| 정서적 지지 | "오늘 학교 안 가도 돼. 우리 어디 갈까?" (멘탈+15) | 가족 외식 (피로-15, 멘탈+5) |
| 엄격함 | "...밥은 먹었어?" (평소 안 하던 한마디. 1주 압박 해제) | "오늘은 쉬어." (루틴 강제 해제) |
| 재력 | 병원 상담 예약 (피로-20) | 보약/영양제 (피로-20) |
| 정보력 | "이번 주는 학원 쉬어" (강제 슬롯 해제) | 스케줄 재조정 |
| 시간/자유 | 아무도 눈치 못 챔. **대신 NPC 안전망 작동** (지훈 전화) | 혼자 하루 종일 잠 (피로-10) |

> 시간/자유 가정은 부모 안전망이 약한 대신 NPC 관계가 안전망 역할

### 6. 루틴 이벤트

**권태 (8주+ 동일 루틴):**
- 8주: "또 같은 길, 같은 학원, 같은 자리" → 권태 피로 +2
- 12주+: 권태 피로 +3으로 증가
- 선택지: 그래도 간다 / 오늘만 빼먹기 (1회 허용) / 루틴 변경

**숙련 (엄격함 전용, 8주 연속):**
- "몸이 기억한다" → 피로 -1/주 (조건부, 루틴 변경 시 해제)

### 7. 진학 적응 이벤트 (첫 4주)

학교 전환 시 4주간 적응기 이벤트 집중 배치.

**인기 환경 리셋 연출:**
- 인기 높았으면: "새 학교에선 아무도 나를 모른다"
- 인기 낮았으면: "아무도 나에 대해 모른다. 그게 오히려 좋았다"

### 8. 회고 카드 시스템

**매주가 아닌 구간 전환점에서만** 생성 (희소성 = 임팩트):

| 시점 | 예시 |
|------|------|
| 학기말 | "이번 학기, 나는 성적을 올렸지만 지훈과 점심을 같이 먹은 횟수가 줄었다." |
| 방학 끝 | "이번 여름, 나는 누구와 가까워졌고 누구와 멀어졌다." |
| 진학 | "초등학교 6년이 끝났다. 같이 가는 사람, 헤어지는 사람." |
| 수능 직후 | "7년이 끝났다. 내가 가장 자주 선택한 건 ____였다." |

- 실제 플레이 데이터 기반 자동 생성
- 카드형 이미지 + 한 줄 문장 → SNS 공유 최적화
- 인생 앨범에 자동 수집

### 9. 부모 기대치 이벤트

재력 가정 성적 하락 시 3단계 에스컬레이션:
1. 경고: "이번엔 좀 떨어졌네." (효과 없음)
2. 반강제: 주말 1슬롯 "학원 특강" 고정 (수용 vs 거부 선택)
3. 확대: 반강제 지속 + 용돈 삭감 + 멘탈 추가 감소

### 10. 특수 이벤트

**계절 감정 이벤트:** 선택지 없이 감정만 전달
- "벚꽃이 교문 앞에 폈다. 새 교과서 냄새가 났다." (3월)
- "하교길에 눈이 왔다. 누군가와 같이 걸으면 좋겠다고 생각했다." (12월)

**"그때 그 선택" 회상:** 초반 딜레마가 후반에 회상으로 돌아옴
- 예: 지훈 아버지 병문안을 갔었으면 → 고3 겨울 "그때 네가 와줘서 고마웠어" 회상 + 멘탈 +3

## 리뷰 요청 사항

### 디렉터 관점에서 판단 부탁드리는 것들:

**1. 첫 10분 구조**
- 부모 선택을 "생활 장면"으로 보여주는 방식이 효과적인가?
- NPC 3명 (편안/호기심/긴장) 배치가 적절한가?
- "첫 후회"의 강도가 적당한가? (너무 약하면 각인 안 됨, 너무 강하면 이탈)

**2. 딜레마 20개의 배분과 품질**
- 초등 3개 / 중학 8개 / 고등 9개 — 배분이 적절한가?
- 각 딜레마가 "정답 없는 선택"으로 느껴지는가?
- 빠진 딜레마가 있다면? (예: 연애 전용, 알바 전용, 온라인 관련 등)
- 딜레마 #19(수능 전날)~#20(수능 후 첫 연락)이 감정적 클라이맥스로 충분한가?

**3. 감정 밀도**
- 이전 리뷰에서 지적된 "시스템은 좋지만 드라마가 부족하다"가 해결되었는가?
- 계절 감정 이벤트, 회고 카드, "그때 그 선택" 회상 — 이 3가지가 감정 보충 장치로 충분한가?
- 더 강화해야 할 감정 포인트가 있다면?

**4. 정보 점진적 공개**
- GPT가 제안한 "초반 1시간: 학업/멘탈피로/용돈/사람만" 방향을 반영했는데, 공개 순서가 자연스러운가?
- 하위 스킬을 "발견형"으로 열어주는 타이밍이 적절한가?

**5. 회고 카드**
- 매주가 아닌 구간 전환점(학기말/방학/진학)에서만 생성하도록 제한했는데, 이게 맞는가?
- 아니면 특정 딜레마 직후에도 생성하는 게 좋은가?

**6. 전체적으로 부족한 부분**
- 이 이벤트 설계에서 가장 약한 섹션은?
- 추가해야 할 이벤트 카테고리가 있다면?
- 흥행 관점에서 가장 보완이 필요한 것은?
