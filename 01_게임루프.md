# 01. 핵심 게임 루프

## 주 단위 진행

```
1. 활동 선택 (카드 UI)
   - 공부 / 운동 / 취미 / 친구 / 연애 / 휴식 / 알바 등
   - 가정환경에 따라 보이는 카드가 다름 (핵심!)

2. 스탯 변화 + 자원 소모
   - 용돈 잔액이 활동 카드에 비용으로 표시됨
   - 별도 용돈 배분 화면 없음 → 선택하면서 자연스럽게 소비

3. 이벤트 발생
   - 딜레마 이벤트 (플레이어 상태가 트리거)
   - 체크포인트 이벤트 (시험, 축제, 체육대회 등)

4. 결과 반영
   - 랭킹 변동 / 관계 변화 / 부모 반응

→ 반복
```

## 리듬 설계 (긴장-이완)

```
[평소]       선택 → 스탯 변화 → 소소한 이벤트 (낮은 긴장)
                ↓
[체크포인트]  중간고사 / 축제 / 체육대회 / 기말고사
                ↓
[결산]       그동안의 선택이 한꺼번에 결과로 터짐 (높은 긴장)
                ↓
             랭킹 변동 + 관계 변화 + 부모 반응 + 회고 컷신
```
