# 01. 핵심 게임 루프

## 설계 철학

> "평일은 정해져 있고, 주말이 진짜 내 시간이다."

한국 학생의 실제 생활 리듬을 게임 구조로 가져온다.
프린세스 메이커의 "10일 스케줄 → 자동 진행 → 중간 이벤트" 방식을 참고하되,
평일/주말 문화에 맞게 재설계.

---

## 1주의 구조: 평일 블록 + 주말 블록

```
[1주 = 1턴]

┌─────────────────────────────┐
│  평일 블록 (월~금)           │  ← 루틴 자동 진행
│  학교 + 방과후 활동           │  ← 사전 설정한 스케줄대로
│  중간에 이벤트 확률 발생       │
├─────────────────────────────┤
│  주말 블록 (토~일)           │  ← 플레이어 실시간 선택
│  자유 활동 2슬롯             │  ← 매주 직접 고르기
│  이벤트 확률 발생             │
└─────────────────────────────┘
```

### 평일 블록 (자동 진행)

플레이어가 **사전에 세팅한 루틴**이 자동으로 돌아간다.
한국 학생의 평일: 학교 → (학원/과외/독학/알바) → 잠깐의 자유시간

```
[평일 구성: 3슬롯]

슬롯1: 학교 수업 (고정 — 바꿀 수 없음)
  → 학업 +1 (기본 수업 효과)
  → 같은 학교 NPC 자동 접촉
  → 학년/학교에 따라 효과 변동

슬롯2: 방과후 활동 (루틴 설정)
  → 학원 / 과외 / 독학 / 부모와 공부 / 예체능 레슨 / 알바 등
  → 한 번 설정하면 바꾸기 전까지 매주 자동 반복
  → "루틴 보너스": 3주 연속 같은 활동 시 가산 +1.0

슬롯3: 방과후 활동 2 또는 자유시간 (루틴 설정)
  → 슬롯2와 같은 선택지 OR "자유시간"
  → 자유시간 선택 시: 피로 -3, 멘탈 +1 (쉬는 시간)
  → 학원 2개 돌리기 vs 학원1+자유시간 = 효율 vs 멘탈 트레이드오프
```

**평일은 "세팅"이 게임이다.** 매주 직접 고르는 게 아니라, 한 번 짜놓은 시간표가 자동으로 돌아간다. 시간표를 바꾸려면 "루틴 변경"을 선택해야 하고, 그러면 루틴 보너스 카운트가 리셋된다.

### 주말 블록 (실시간 선택)

**매주 플레이어가 직접 선택하는 핵심 의사결정 구간.**

```
[주말 구성: 2슬롯]

슬롯A (토요일): 자유 선택
슬롯B (일요일): 자유 선택

선택 가능:
- 친구와 놀기 (NPC 선택)
- 연애 데이트
- 운동 / 체력 관리
- 취미 / 재능 활동
- 추가 공부 (주말 독학)
- 알바
- 휴식 (푹 쉬기)
- 가족 시간 (정서적 지지 전용)
- 공원 산책
- SNS / 온라인 활동
```

**주말이 이 게임의 핵심이다.** 평일은 루틴이 돌아가고, 주말에 "이번 주에 뭘 할 것인가"를 결정한다. 친구를 만날지, 쉴지, 추가 공부를 할지 — 이 선택이 곧 "어떤 인생을 살 것인가"가 된다.

### 이벤트 (확률적 발생)

```
[이벤트 타이밍]

평일 중: 학교/학원에서 발생 (확률적)
  → NPC 갈등, 선생님 면담, 시험 결과, 급식실 사건 등
  → "학교 자동 진행 중 알림" 형태로 끼어들기

주말 중: 선택한 활동 도중 발생 (확률적)
  → NPC 만남 이벤트, 우연한 재회, 고백, 다툼 등

체크포인트: 학기 일정에 따라 확정 발생
  → 중간고사, 기말고사, 축제, 체육대회, 수학여행, 진학 등
```

**이벤트는 항상 발생하지 않는다.** 아무 일 없이 루틴대로 흘러가는 주도 있다. 그래서 이벤트가 터졌을 때 더 강하게 느껴진다.

## 기존 5슬롯 → 새 구조 매핑

```
[기존]                    [새 구조]
슬롯1 (활동)      →      평일 슬롯1: 학교 수업 (고정)
슬롯2 (활동)      →      평일 슬롯2: 방과후 루틴 1 (자동)
슬롯3 (활동)      →      평일 슬롯3: 방과후 루틴 2 (자동)
슬롯4 (활동)      →      주말 슬롯A: 토요일 (직접 선택)
슬롯5 (활동)      →      주말 슬롯B: 일요일 (직접 선택)

총 5슬롯은 동일 — 기존 수치 시스템과 호환
```

### 부모 시스템 연동

```
[재력 가정]
→ 방과후 슬롯에 고가 학원/과외 선택지 개방

[정보력 가정]
→ 방과후 슬롯에 히든 학원 선택지 개방

[엄격함 가정]
→ 방과후 슬롯 1개가 부모 지정 (루틴 강제)
→ 주말 슬롯 중 1개가 "추가 공부" 반강제 (거부 시 부모 갈등)

[시간/자유 가정]
→ 방과후 "자유시간" 슬롯의 효과 강화 (+2 멘탈)
→ 주말 선택지 추가 개방 (이중생활, 야간 외출 등)

[정서적 지지 가정]
→ "부모와 같이 공부" 방과후 슬롯 개방
→ "가족 저녁시간" 주말 슬롯 개방

[유전자 가정]
→ 학교 수업 효과 +0.5 (기본 재능/학업 보너스)
```

### 학년별 슬롯 변화

```
[초등 6학년]
평일: 학교(고정) + 방과후 2슬롯 (자유)
주말: 2슬롯 (자유)
→ 가장 여유로운 시기. 루틴 세팅의 자유도 최대.
→ 학원/과외/놀기/자유시간 등 자유롭게 배치

[중학교]
평일: 학교(고정) + 방과후 2슬롯 (자유)
주말: 2슬롯 (자유)
→ 초등과 슬롯 구조는 동일하나, 학업 난이도 상승 + NPC 관계 복잡화
→ 학교 수업 효과가 초등보다 낮음 (수업 난이도↑, 같은 시간 대비 흡수율↓)
→ 야자는 없음 (중학교는 야간자율학습 없음)
→ 대신 "시험 기간(시험 2주 전)"에 방과후 슬롯 1개가 "시험 공부" 반강제
  (거부 가능하지만 시험 성적에 직접 영향)

[고등학교 1~2학년]
평일: 학교(고정) + 방과후 1슬롯 (자유) + 야자(반강제)
주말: 2슬롯 (자유)
→ 야자를 선택하면: 학업+2, 피로+5 (안정적 학업 확보)
→ 야자를 거부하면: 자유 슬롯 1개 추가 확보, 대신 학교 평판↓ + 부모 반응
→ 엄격함 가정: 야자 거부 불가 (강제)
→ 시간/자유 가정: 야자 거부 패널티 감소

[고등학교 3학년]
평일: 학교(고정) + 야자(강제) + 야자2(강제)
주말: 1슬롯 (자유) + 자습(반강제)
→ 자유 슬롯이 주당 1~2개로 극감
→ "이 1슬롯에 뭘 할 것인가"가 고3의 핵심 딜레마
→ 시간/자유 가정: 야자2를 자유로 전환 가능 (유일한 탈출구)
```

## 학기 구조 (실제 한국 교과과정 기반)

> 한국 초중고 실제 학사일정을 기반으로 설계.
> 여름방학 약 4~5주, 겨울방학 약 5~6주 (학교마다 차이 있으나 평균치 적용).

```
[1학기: 3월 ~ 7월 중순] (약 19주)
├─ 3월 (4주): 개학 + 반배정 + 새 NPC 만남
├─ 4월 (4주): 적응기 + 소소한 이벤트 + 4월 말 중간고사
├─ 5월 (5주): 체육대회/축제 + 어버이날 이벤트
├─ 6월 (4주): 일상 + 진로 상담 + 6월 말~7월 초 기말고사
├─ 7월 1~2주: 기말고사 + 학기 결산 + 회고 컷신
│
[여름방학: 약 5주] (7월 중순 ~ 8월 중순)
├─ 평일/주말 구분 해제 (매일 자유 슬롯)
├─ 방학 과제, 캠프, 여행, 알바
├─ 8월 중순 개학 준비
│
[2학기: 8월 말 ~ 12월 말] (약 18주)
├─ 9월 (4주): 개학 + 관계 변화 확인
├─ 10월 (4주): 10월 중순 중간고사 + 수학여행
├─ 11월 (4주): 수능(고3) + 일상 + 진로 결정 시기
├─ 12월 (4주): 12월 초중순 기말고사 + 학년 결산 + 크리스마스
├─ 12월 말: 종업식/졸업식
│
[겨울방학: 약 6주] (12월 말 ~ 2월 중순)
├─ 가장 긴 방학 = 가장 많은 자유 시간
├─ 새 학년 준비 기간
├─ 진학 분기점 (초→중, 중→고 결정 및 준비)
├─ 연말/새해/설날 이벤트
├─ 2월: 졸업식(해당 학년), 새 학교 준비
│
[총 구성]
1학기 수업: 19주
여름방학:    5주
2학기 수업: 18주
겨울방학:    6주
──────────────
합계:       48주 (게임 내 1년)

※ 실제 52주와 4주 차이는 개학/종업 전후 행사주간으로 흡수.
※ 7년(초6~고3) = 48주 x 7 = 336주 (약 340턴)
```

### 체크포인트 이벤트 배치 (연간)

| 시기 | 이벤트 | 효과 |
|------|--------|------|
| 3월 1주 | **개학/반배정** | 새 NPC 만남, 관계 리셋 확인 |
| 5월 3주 | **중간고사** | 학업 스탯 기반 등수 산정, 부모 반응 |
| 5월 4~5주 | **체육대회/축제** | 인기/재능/체력 종합 이벤트, NPC 관계 변동 |
| 7월 2주 | **기말고사** | 등수 + 학기 성적표, 부모 반응 |
| 7월 4주 | **학기 결산 + 회고 컷신** | "이번 학기, 나는 ___했다" |
| 여름방학 (5주) | **방학 이벤트** | 자유 루틴, 캠프/여행/알바 |
| 10월 3주 | **중간고사** | 등수 산정 |
| 10월 4~5주 | **수학여행** | NPC 관계 집중 이벤트 (강제 합숙) |
| 12월 2주 | **기말고사** | 등수 + 학년 성적표 |
| 12월 3~4주 | **학년 결산 + 크리스마스** | 연간 회고, NPC 선물/고백 이벤트 |
| 겨울방학 (6주) | **진학 분기점 + 연말 이벤트** | 초→중, 중→고 진학 선택, 크리스마스, 새해 |

## 방학 구조 (특별 모드) — 옵션 C 하이브리드 + 목표 설정

> 방학은 학기와 완전히 다른 게임 리듬.
> 학기 = "루틴 속 소소한 선택", 방학 = "자유 속 큰 선택".
> 같은 C 구조라도 여름방학은 "해방/탐색", 겨울방학은 "정리/전환"으로 무드가 다르다.

### 방학 기본 규칙

```
[여름방학: 5주] (7월 중순 ~ 8월 중순)
[겨울방학: 6주] (12월 말 ~ 2월 중순) — 가장 긴 방학

공통:
- 평일/주말 구분 해제 — 학교가 없으니까
- 주당 5슬롯 (학교 고정 슬롯 사라짐 → 전부 자유)
- 시간/자유 가정: +1 → 주당 6슬롯
- 루틴 자동 반복 없음 — 매주 직접 선택
- 피로 자연회복 +2 추가 (학교 안 가니까 여유)
- 학교 자동 접촉 정지 — 방학 동안 "같은 학교" NPC 자동 +1 멈춤
```

### 방학 진행 흐름

```
[STEP 1: 방학 시작 — "이번 방학 목표" 선택]

"여름방학이 시작됩니다! 이번 방학, 뭘 하고 싶나요?"

목표 선택 (1개):
├─ 🎓 성적 올리기 — 학업 활동 추천 강화, 결산에서 학업 중심 평가
├─ 🎉 실컷 놀기 — 관계/놀기 활동 추천, 결산에서 관계 중심 평가
├─ 💰 돈 모으기 — 알바 활동 추천, 결산에서 저축 중심 평가
├─ 💪 체력 만들기 — 운동 활동 추천, 결산에서 체력 중심 평가
└─ 🔍 진로 찾기 — 재능/탐색 활동 추천, 결산에서 재능 중심 평가

※ 강제 아님! 목표와 다른 활동을 해도 패널티 없음.
※ 영향 범위: 활동 추천 순서, 방학 결산 문구, 소소한 이벤트 확률
```

```
[STEP 2: 매주 — 5슬롯 직접 선택 (묶음 활동 가능)]

방학 중에는 "묶음 활동"으로 한 번의 선택이 여러 슬롯을 소모.
→ 5슬롯이지만 실질 선택 횟수는 2~4회 (선택 피로 감소)

예시 (여름방학 2주차):
┌────────────────────────────────────────┐
│  이번 주 슬롯: 5개                       │
│                                        │
│  [방학 특강] 2슬롯  [알바] 2슬롯         │
│  [친구놀기] 1슬롯   [자유연습] 1슬롯      │
│  [가족여행] 3슬롯   [푹쉬기] 1슬롯       │
│  [종일놀기] 2슬롯   [방학숙제] 1슬롯      │
│                                        │
│  선택: [방학특강(2)] + [알바(2)] + [쉬기(1)] │
│  → 실질 선택 3번, 슬롯 5개 소모            │
└────────────────────────────────────────┘
```

```
[STEP 3: 매주 — 이벤트 확률 체크]

방학 이벤트는 학기보다 빈도 낮음 (학교 이벤트가 없으니까).
대신 NPC 개인 이벤트 중심:
- "지훈이가 놀러오자고 전화했다" (관계 유지 기회)
- "수빈이 SNS에 여행 사진 올림" (간접 접촉)
- 방학 목표 관련 소이벤트 (목표: 돈모으기 → 알바처 소개 이벤트)

확률: 주당 30~40% (2~3주에 1번은 뭔가 터짐)
```

### 여름방학 vs 겨울방학 무드 차이

```
[여름방학] — 해방과 탐색
├─ 분위기: 밝고 자유로움
├─ 핵심 활동: 놀기, 여행, 알바, 재능 탐색
├─ 이벤트 톤: "여름이니까 뭐든 해보자!"
├─ NPC: 바다/수영장/축제 등 여름 이벤트
└─ 결산: "이번 여름, 나는 ___했다"

[겨울방학] — 정리와 전환
├─ 분위기: 차분하고 무거움 (연말/새해)
├─ 핵심 활동: 진학 준비, 정리, 관계 결산
├─ 이벤트 톤: "다음 해를 위해 뭘 준비할까"
├─ NPC: 크리스마스, 새해 카운트다운, 졸업 이벤트
├─ 특수: 진학 분기점 (초→중, 중→고)
└─ 결산: "올해를 돌아보며..." + "내년에는..."
```

### 방학 중 관계 하락과 "개학 어색함"

```
[방학 중 NPC 관계]

학교 자동 접촉 (+1/주) → 정지!
→ 접촉 없는 NPC: 매주 -1 자연 하락

방학 중 만나려면:
├─ "친구와 종일 놀기" (2슬롯) — 적극적 투자
├─ "연락하기" (1슬롯) — 최소 유지
├─ 같은 동네/학원 채널이 살아있으면 — 약한 자동 유지
└─ 온라인 친구(루나) — 온라인 활동 시 유지

[개학 첫 주]
방학 동안 안 만난 NPC → "어색함 이벤트"
├─ 친밀도 60 이상이었으면: "오랜만이다! 방학 잘 보냈어?" (자연 회복)
├─ 친밀도 40~59였으면: "...안녕. 방학 때 뭐 했어?" (어색한 대화)
└─ 친밀도 39 이하였으면: (인사만 하고 지나감) → 관계 추가 -2

→ 방학에 관계를 챙겼는지 안 챙겼는지가 개학 첫 주에 바로 체감됨
```

### 겨울방학 특수: 진학 이벤트

```
[겨울방학 마지막 2주]

초6 → 중1: 중학교 배정 발표, NPC 채널 변동 확인, 새 교복 이벤트
중3 → 고1: 고등학교 선택 (일반고/특목고), 원서 제출, 결과 발표
            → 이 선택이 NPC 관계를 대규모로 변동시킴

진학 결정 후:
→ "앞으로의 3년" 프리뷰 컷신
→ "같이 가는 친구 / 헤어지는 친구" 목록 표시
→ 감정적 클라이맥스 중 하나
```

## 진학 시스템 (학교 전환)

```
[초등 6학년 → 중학교] (겨울방학)
├─ 진학 선택 없음 (배정)
├─ 같은 중학교 가는 NPC / 다른 중학교 가는 NPC 자동 결정
├─ 관계 채널 변동 (같은 학교 유지/상실)
└─ 새 NPC 등장 (중학 친구 A, B, 선배)

[중학교 → 고등학교] (겨울방학)
├─ 진학 선택: 일반고 / 특목고
│  ├─ 일반고: 중학교 친구 일부 유지, 난이도 보통
│  └─ 특목고: 아는 사람 0명, 학업 보너스, 높은 압박
├─ 선택 조건: 학업 스탯 + 하위스킬에 따라 선택지 제한
├─ 관계 대규모 변동
└─ 새 NPC 등장 (고등 친구 A, B, 전학생)

진학 선택 = 이 게임의 가장 큰 딜레마 중 하나
→ "친구 따라 일반고" vs "꿈 위해 특목고"
→ 스탯뿐 아니라 NPC 관계에 직접 영향
```

### STEP 1: 주 시작 (월요일 아침)

```
"3월 2주차가 시작됩니다."
```

### STEP 2: 평일 자동 진행 + 결산 (핵심!)

```
"월요일~금요일이 지나갑니다..."

[평일 진행 연출]
→ 캐릭터가 학교 가는 짧은 애니메이션 (1~2초)

  예: "수요일 점심, 민재가 말을 걸어왔다."
  → 선택지 → 결과 → 다시 진행
→ 이벤트 없으면 바로 결산으로

[평일 결산 화면] ← 반드시 표시 (스킵 불가)
┌─────────────────────────────────────┐
│  📊 이번 주 평일 결과               │
│                                     │
│  학업  ██████████░░  42 → 45 (+3)   │  ← 변동 있는 스탯만 표시

│  피로  ███░░░░░░░░░  12 → 22 (+10)  │
│                                     │
│  💰 용돈: 8만원 → 5만원 (-3 학원비)  │
│                                     │
│  👥 NPC                             │
│  지훈: 보통 (56) — "오늘도 같이 급식 먹음" │
│  민재: 보통 (47) — [이벤트 발생!]     │
│                                     │
│  📌 다음 주 참고                     │
│  "중간고사까지 3주 남았습니다"        │
│                                     │
│         [주말로 넘어가기 →]           │
```

**이 화면이 중요한 이유:**
- 플레이어가 "이번 루틴이 효과가 있는지" 매주 확인 가능
- 학업이 안 오르면 → "루틴을 바꿔야 하나?" 고민 유도
- 멘탈/피로가 떨어지면 → "주말에 쉬어야겠다" 판단 근거
- 시험까지 남은 주 표시 → 단기 목표 의식
- **이 결산을 보고 주말 선택을 하는 구조** = 정보 기반 의사결정

### 주간 가이드 시스템 (평일 결산 → 주말 추천)

평일 결산 하단에 **상황 기반 추천 메시지**가 표시된다.
강제가 아니라 "이런 상태니까 이런 걸 하면 좋겠다"는 가이드.

```
[추천 로직]

피로 50+ 또는 멘탈 30 이하:
  ⚠ "이번 주말은 좀 쉬는 게 좋겠어요"
  → 휴식/산책 카드에 ✨ 추천 표시

시험 2주 전:
  📝 "중간고사가 2주 남았습니다. 주말에 공부를 보충할까요?"
  → 공부 카드에 ✨ 추천 표시

NPC 친밀도 하락 감지 (전주 대비 -3 이상):
  💬 "지훈이랑 요즘 좀 소원해진 것 같아요"
  → 지훈과 놀기 카드에 ✨ 추천 표시

특정 스탯 장기 정체 (3주 이상 변동 없음):
  📊 "재능이 3주째 제자리입니다. 취미 활동은 어때요?"
  → 재능 활동 카드에 ✨ 추천 표시

용돈 부족 (잔액 1만원 이하):
  💰 "용돈이 거의 없어요. 알바를 해볼까요?"
  → 알바 카드에 ✨ 추천 표시

마일스톤 근접 (스탯이 목표치 -3 이내):
  🎯 "학업이 47이에요. 조금만 더 하면 50 돌파!"
  → 해당 스탯 활동에 🔥 도전 표시

아무 위기 없을 때:
  😊 "이번 주는 여유 있네요. 하고 싶은 거 하세요!"
  → 추천 없음 (자유 선택)
```

**설계 원칙:**
- 추천은 **절대 강제가 아님** — 무시해도 패널티 없음
- "왜 이걸 추천하는지" 이유를 짧게 보여줌 (상태 기반)
- 초보 플레이어에게는 가이드, 숙련자에게는 무시 가능
- 추천을 따랐을 때 별도 보상 없음 (추천 의존 방지)
- 핵심: **"지금 내 상태에서 뭘 하면 좋을지"를 10초 안에 판단**할 수 있게 해줌

## 수치 시스템 연동 (02_스탯시스템과의 호환)

### 슬롯 재매핑

```
[기존 5슬롯 시스템]
슬롯1~5: 모두 동일한 자유 선택

[새 구조]
평일 슬롯1: 학교 수업 — 고정, 학업+1, 피로+2
평일 슬롯2: 방과후 루틴1 — 기존 활동 테이블에서 선택 (자동 반복)
평일 슬롯3: 방과후 루틴2 — 기존 활동 테이블에서 선택 OR 자유시간
주말 슬롯A: 토요일 — 기존 활동 테이블에서 매주 직접 선택
주말 슬롯B: 일요일 — 기존 활동 테이블에서 매주 직접 선택

→ 02_스탯시스템의 활동 테이블, 피로 계산, 성장 공식 모두 그대로 적용
→ 반복 감쇠: 같은 주 내에서 같은 활동 반복 시 적용 (기존과 동일)
→ 루틴 보너스: 방과후 루틴이 3주 연속 같은 활동 시 가산+1.0
```

### 피로 계산 (주간)

```
평일 피로:
  학교 수업 (고정): +2
  방과후 루틴1: 활동별 피로 (테이블 참조)
  방과후 루틴2: 활동별 피로 OR 자유시간(-3)

주말 피로:
  토요일 활동: 활동별 피로
  일요일 활동: 활동별 피로

자연회복: -8/주 (기본), 정서적 지지 -11/주

→ 기존 시뮬레이션 결과와 호환
```

### 체크포인트 주의 스탯 산정

```
[중간고사/기말고사]
→ 시험 등수 = 학업 스탯 + 하위스킬(문과/이과) 기반
→ 시험 직전 2주의 학업 활동에 보너스 (벼락치기 효과)
→ 체력/멘탈이 낮으면 시험 당일 디버프

[체육대회]
→ 체력 + 재능(체육) 기반
→ NPC 팀 구성에 따라 시너지/갈등

[축제]
→ 인기 + 재능 종합
→ 무대/부스 참여 시 인기 대량 변동 가능
```
