# 중학교 도달형 풀 설계 발주 v2 — 학년당 3개 (2026-06-11)

당신은 **최고의 게임 시나리오 작가**입니다. 배정된 NPC **한 명**의 중학교 도달형(reach) 이벤트를 **학년당 3개씩** 완성하세요. 5명의 작가가 1차로 학년별 핵심 beat(앵커)와 후보 아이디어를 이미 냈고, 그 풀이 아래 있습니다. 그걸 **재료**로 삼아 다듬고 채워 완성본을 만드세요.

## 게임/도달형 기본 (요약)
한국 학교생활 시뮬레이션. 친밀도 임계(tier)를 floor로 게이트하는 잔잔한 일상 컷 = 도달형. 페이싱 엔진이 발동 시점을 조절(빨리 도달하면 즉시 fresh, 이미 도달해 있으면 학년 내 분산 drip)하므로, 작가는 "그 관계의 그 순간"만 설계하면 됨. 선택지 2~3개, 스탯 효과 작게(±1~3)·친밀도 +1~5. 중요 컷은 memorySlotDraft(엔딩 회상).

## 학년당 3개 구성 원칙
각 학년(중1=Y2 / 중2=Y3 / 중3=Y4)에 **3개**를, 그 학년 tier 사다리(낮은→높은 floor) 순서로:
1. **진입(낮은 tier)**: 가벼운 일상, 관계의 결을 슬쩍.
2. **중간(중간 tier)**: 한 겹 들어간 단서/사건.
3. **핵심(높은 tier, 앵커)**: 그 학년의 결정적 순간. memorySlotDraft 부여.

학년이 올라갈수록 더 솔직하고 깊어짐(관계 누적). 같은 NPC 안에서 무대·소재·memory category가 반복되지 않게.

## 절대 규칙
- **기존 Y1(초등) 인벤토리와 겹침 금지** (아래 각 NPC "Y1 소진 모티프" 참조).
- **중학교 고유 무대만**: 매점·급식실·사물함·복도·체육대회·수련회·학원/셔틀·동아리방·방송실·음악실·옥상·버스/등하굣길·편의점·교무실앞 등. (Y1의 교실창가·칠판·도서코너·운동장스탠드·밤단톡·졸업식·종이비행기·계획표 재사용 금지)
- **전부 전용 CG 전제** → 각 이벤트에 "CG 장면(1줄): 한 컷 구도·배경·표정".
- memory category 팔레트: courage/betrayal/reconciliation/failure/discovery/growth/bypass/unspoken_debt.

## 산출물 형식 (각 이벤트)
```
[NPC] Y? · tier ?? · (진입/중간/핵심)
제목:
무대:
premise(1~2줄):
드러나는 내면:
선택지 2~3개(각 한 줄): 태도 + 효과(스탯/친밀도)
memory: category/importance/toneTag/recallText (핵심 컷만)
CG 장면(1줄):
```
마지막에 "이 NPC 9개(또는 6개) 사이 + Y1과의 차별화" 자기 점검 2~3줄.

# 배정 NPC별 자료

## ▣ 지훈(jihun) — 9개 (중1·중2·중3 ×3)
**프로필**: 활발한 소꿉친구, 운동 좋아함, 늘 먼저 말 걸어줌. 가장 오래 본 사이(초등부터). 밝고 단순해 보이지만 그게 방패. **Y1 reach 없음(중학 첫 도입)**. 시작 친밀도 40.
**tier 사다리**: 중1 `45·55·65` / 중2 `70·76·82` / 중3 `85·88·90`
**Y1 소진 모티프**: 없음(첫 도입) — 단 "운동·먼저 말 걸기·소꿉친구" 코어 살릴 것.
**후보 풀(5작가)**:
- 중1: 사물함 비번(작년 네 생일로 맞춰둠—다정한 기억력)/ 교복 키차이(분식집 거울)/ 새 운동화 흙(가진 것 없다는 의식)/ 매점 삼각김밥 반쪽
- 중2(앵커=계주): ⭐**체육대회 계주에서 흔들림**(바통 놓침/물통 양보 → 처음 "안 괜찮아"·"잘하는 척했어", 활기가 방패) / 사물함 옛날 사진(6년 우정 애틋)
- 중3(앵커=갈림): ⭐**고등 진학 갈림**("처음으로 안 같이 갈 수도", 관계를 처음 의식·언어화) / 매점 라면내기(너 따라가려 야자) / 버스 우산("10년 친구")
- ⚠️ "아버지 사고·돈 걱정"은 캐릭터 바이블에 없으니 쓰지 말 것.

## ▣ 수빈(subin) — 9개
**프로필**: 어디서든 금방 친해지는 외향형, 어느 그룹서든 "곁다리"(중심 아님). 엄마와 둘이 삶(한부모). Y1에 90까지 도달 가능.
**tier 사다리(높은 floor)**: 중1 `70·74·78` / 중2 `78·82·85` / 중3 `85·88·90`
**Y1 소진 모티프(금지)**: "부담 없는 척" 호의 / 마트·사은품 / 밤·집의 불안 / 떠남 동경.
**후보 풀**:
- 중1(앵커=가장자리): ⭐**무리의 가장자리**(단체사진 "또 끝자리네"/급식실 빈자리 — 인맥 많아도 1순위 아님)
- 중2: 장기자랑 후 "박수는 무대한테 치는 거지 나한테 아니잖아" / 다른학교 이름표("내 이름보다 학교가 먼저") / 삭제된 태그("없어도 사진은 완성돼")
- 중3: 마트 야간파트(한부모를 *직접* 말하게 된 성장) / 진짜친구 "몇 명일까"+머무르고 싶음 / 엄마 명함(생존방식 수용) / 반배정표 "너랑은 아쉽다" / 단톡 못 끄겠음

## ▣ 민재(minjae) — 9개
**프로필**: 전교 1등이나 천재 아닌 노력형. 새벽까지 공부+아침 태연한 척. 학원 원장 엄마+교사 아빠(교육가정 압박). Y1엔 tier 70 하나만.
**tier 사다리**: 중1 `50·58·66` / 중2 `70·76·82` / 중3 `84·88·92`
**Y1 소진 모티프(금지)**: "구겨진 만점/실수 공포".
**후보 풀**:
- 중1: 매점 빵 3분("단거 먹으면 효율 떨어져"—강박)/ 형광펜 닳음(새벽의 증거)/ 사물함 선행자료 쏟음
- 중2(앵커=등수): ⭐**2등/0.1점 "1등 아닌 내가 누군지 모르겠어"**(등수=정체성)/ 과학실 오답 색깔정리/ 새벽 메시지 오타(눈밑 검정)
- 중3: 셔틀 5분 해방(엄마 셔틀 빼먹고 빗속 숨구멍)⭐ / 빈칸에 자기 이름(정답 아닌 자기를 출발점, capstone)⭐ / 상담실 밖 "교재 샘플"(전시되는 모범생)/ 교무실앞 "우리 앤 알아서"/ 빨간펜 아빠(채점관)

## ▣ 유나(yuna) — 9개 (중3까지 재적, 고1 퇴장)
**프로필**: 밝고 에너지 넘치는 상위권, 피아노 잘 침. 밝음 뒤 "1등 놓치면 안 된다" 압박. Y1에 90 도달 가능.
**tier 사다리**: 중1 `72·78·82` / 중2 `82·85·88` / 중3 `86·89·92`
**Y1 소진 모티프(금지)**: 완벽주의·1등 압박 정면 / 별·창문 이미지 / "웃음=역할".
**후보 풀**:
- 중1(앵커=피아노): ⭐**피아노의 변질**(콩쿠르/연습 "잘 쳐야 한다 생각하면 손이 굳어, 나 이거 좋아했었나")/ 합주 박자놓침 회복/ 성적표 손톱(2등이라며 웃는데 손톱 물어뜯김)
- 중2: 못 놓는 부장("손 든 게 아니라 손 안 들면 불안")/ 등수 버스 "분한 게 너라서 다행"(경쟁+우정)/ 소견문 "열심히 보이면 부담"(타인 시선)/ 무대 뒤 조명담당(중심 대신 뒤)
- 중3: 전학 소문 "피아노로 승부할 수 있대"+웃지않음(웃음=역할 반전, 퇴장 전조)⭐ / 젖은악보 "서두르지 않아도"(성숙) / *중3은 풀이 얇으니 새 beat 보강 가능*(예: 마지막 콩쿠르, 떠나기 전 정리)

## ▣ 하은(haeun) — 6개 (중1·중2 ×3, 이후 고등 진학)
**프로필**: 1년 위 선배(플레이어 중1 때 하은 중2). 겉은 여유·속은 불안. 오빠 수능 실패 트라우마 + 자기 고등 진학 불안. 후배 챙기며 자길 다독임. Y2 등장(친밀도 0).
**tier 사다리**: 중1 `30·42·55` / 중2 `62·72·80`
**Y1 소진 모티프**: 없음(신규) — 단 선배·오빠트라우마 코어.
**후보 풀**:
- 중1(진입): 분실물함 체육복("중학교는 물건도 사람도 자주 길 잃어")/ 매점 줄 끼워주기/ 동아리 인수인계 노트(후배 위해 미리 적은 배려)/ 방송실 마이크("떨려도 끝까지 읽으면 방송이 돼")/ 사물함 번호
- 중2(앵커=오빠트라우마): 오빠 참고서(다른 이름, "이걸로 했는데도 잘 안 됐거든"—균열)/ 자습실 입시자료 떨리는 손 / ⭐**원서 앞 "오빠가 원하는 데 썼다 망했거든, 난 안전한 데 써야 하나"**(선배 역할 내려놓고 처음 무섭다고)/ 옥상 계단 / 합격발표 "안 기뻐"
