## AI에게 줄 프롬프트 (이 아래부터 복사)

당신은 한국 학교생활을 다루는 성장 시뮬레이션 게임의 시나리오 작가입니다. 게임의 NPC/부모 "잡담 라인"을 추가로 작성해야 합니다.

### 게임 배경

- 한국의 12세(초6) → 18세(고3)까지 7년간 학교생활을 시뮬레이션
- 플레이어는 NPC 친구들과 친밀도를 쌓고, 부모와도 가끔 대화함
- 학년 진행: Y1=초6, Y2=중1, Y3=중2, Y4=중3, Y5=고1, Y6=고2, Y7=고3

### "잡담 라인"의 역할

플레이어가 NPC/부모 모달에서 "말 걸기" 버튼을 클릭했을 때, **이번 주에 특별한 미니 이벤트가 없는 경우** 표시되는 짧은 한 줄. 매주 한 번씩 클릭할 때마다 풀에서 RNG로 픽됨. 같은 라인이 반복되면 지루하므로 다양성이 중요.

### 톤 규칙 (반드시 지킬 것)

**DO ✅**
- NPC/부모가 **지금 너에게 말 거는 직접 인용형**: `"...?"`, `"..."`
- 진행 중인 대화의 한 조각 — 다음에 어떤 답이 올지 자연스럽게 예상되는
- 예: `"이번 주 숙제 진짜 많지 않아? 나 아직 절반도 못 했어."`

**DON'T ❌**
- 종결형 묘사 (`"~하고 헤어졌다"`, `"~하고 가버렸다"`, `"~지나갔다"`, `"그게 다였다"`) — 대화가 끝나는 느낌
- 학년 종속 표현 (`"야자 째자"` ← 야자=고등 / `"수능 끝나면"` 등) — Y1 초등에선 부적절
- 너무 길거나 복잡한 문장 (한 화면에 들어가야 함, 1~2문장 이내)
- 게임 내에 없는 시스템·기능을 가정하는 라인

### 캐릭터 프로필 (NPC 6명)

각 NPC의 캐릭터 결을 살린 라인이어야 함. 절대 "누구나 할 법한 말"이 아닌, 그 NPC만 할 법한 말로.

| ID | 이름 | 등장 | 캐릭터 |
|---|---|---|---|
| `jihun` | 지훈 | Y1+ | 활발하고 운동을 좋아하는 소꿉친구. 항상 먼저 말을 걸어준다. 농구·게임·분식 좋아함. |
| `subin` | 수빈 | Y1+ | 어디서든 금방 친해지는 외향형. 친구가 많아 보이지만 어느 그룹에서든 "같이 어울리는 애" 포지션. 엄마와 둘이 산다. 차분하고 책 좋아함. |
| `minjae` | 민재 | Y1+ | 전교 1등이지만 천재가 아니라 노력형. 새벽까지 공부하고 아침에 태연한 척하는 아이. 학원 원장 엄마와 교사 아빠 밑에서 자란 교육 가정. |
| `yuna` | 유나 | Y1+ | 성적은 항상 상위권인데 공부벌레 느낌은 아님. 밝고 에너지 넘침. 피아노도 잘 치고 친구도 잘 사귄다. 그 밝음 뒤에 1등 놓치면 안 된다는 압박. |
| `haeun` | 하은 | Y2+ (1살 위 선배) | 겉으로는 여유 있어 보이지만 고등학교 진학 + 오빠의 수능 실패 트라우마로 속은 불안. 후배를 챙기면서 자기 자신을 다독이는 사람. 거리감 있는 따뜻함. |
| `junha` | 준하 | Y6+ (고2 전학생) | 부산에서 전학 온 직설적인 남자아이. 원래 밝고 사람을 좋아하지만 전학이 눌러놓고 있다. 요리 좋아하고 주먹밥을 싸온다. 엄마가 식당에서 일함. 사투리 가끔 나옴. |

### 부모 강점 (6종)

플레이어의 부모는 두 가지 "강점" 조합으로 정의됨. 각 강점의 톤이 부모 잡담 라인에 반영되어야 함.

| 강점 | 가정 분위기 | 잡담 톤 |
|---|---|---|
| `emotional` | 정서적 지지 | 따뜻한 격려, 공감, 안부 묻기 |
| `wealth` | 여유 있는 집 | 물질적 지원, 외식 제안, 용돈 |
| `info` | 정보가 있는 집 | 분석적 조언, 책·영상 추천, 진로 정보 |
| `strict` | 엄격한 집 | 잔소리, 기대 압박, 시험·정리 강조 |
| `resilience` | 타고난 체질 | 무심한 격려 ("그냥 해", "잘 자") |
| `freedom` | 자유로운 집 | 거리감 있는 자유 ("알아서 해", "네 결정") |

### 현재 풀 (참고용 — 톤 일관성 위해)

**NPC 잡담 (현재 각 3개씩, 총 18개):**

```
jihun:
  "야 너 어제 그 농구 경기 봤어? 막판에 진짜 대박이었어."
  "오늘 급식 뭐 나오지? 나 배고파 죽겠어."
  "우리 동네에 새 분식집 생겼대. 같이 가볼래?"

subin:
  "이번 주 숙제 진짜 많지 않아? 나 아직 절반도 못 했어."
  "이 책 봤어? 빌려줄까? 너도 좋아할 것 같아."
  "우리 학원 같은 시간대에 하잖아. 같이 갈래?"

minjae:
  "수학 7번 풀이 봤어? 답지 봐도 잘 모르겠더라."
  "너 시험 범위 어디까지 봤어? 나는 아직 멀었어."
  "이번 주말에 뭐 해? 같이 도서관 갈래?"

yuna:
  "나 어제 진짜 좋은 노래 발견했어. 한번 들어볼래?"
  "너 요즘 뭐 보고 있어? 나 추천 좀 해줘."
  "오늘 하늘 너무 예쁘지 않아? 사진 찍어놨어."

haeun:
  "...너 요즘 잘 지내? 잘 안 보이더라."
  "너무 무리하지 마. 그러다 진짜 쓰러져."
  "이거 자판기에서 하나 뽑았는데, 너도 마실래?"

junha:
  "오늘 점심 뭐 나오지? 학식 별로면 우리 매점 갈까?"
  "엄마가 어제 만두 만들었거든. 내일 좀 싸올까?"
  "...너 부산 와본 적 있어? 한번 와봐, 진짜야."
```

**부모 잡담 (현재 각 강점 3개씩, 총 18개):**

```
emotional:
  "오늘은 어땠어? 무슨 일 있었어?"
  "엄마가 핫초코 만들었어. 좀 마실래?"
  "피곤해 보이네. 좀 쉬어야 하지 않아?"

wealth:
  "용돈 부족하지 않아? 책이라도 좀 사 봐."
  "이번 주말에 가족 외식할까? 먹고 싶은 거 골라."
  "필요한 거 있으면 언제든 말해."

info:
  "이 책 한번 읽어봐. 너한테 도움될 거야."
  "엄마가 알아본 학원 있는데, 한번 가볼래?"
  "오늘 뉴스에서 그 분야 기사 봤어. 전망 좋다더라."

strict:
  "숙제 다 했어? 안 했으면 지금 해."
  "이번 시험 잘 봐야 한다. 알지?"
  "방 좀 정리해라. 책상이 그게 뭐냐."

resilience:
  "피곤해 보이네. 너무 무리하지 마."
  "넌 잘 할 거야. 평소대로만 해."
  "오늘은 일찍 자. 내일 또 있어."

freedom:
  "네가 알아서 잘 하잖아."
  "이번 주말에 뭐 할 거야? 알아서 정해."
  "네 결정이라면 엄마는 응원해."
```

### 작업 의뢰

위 톤 규칙과 캐릭터 결을 따라, 다음을 작성해주세요:

**1. NPC 잡담 추가 — 각 NPC당 7개씩 (총 42개)**
- 기존 3개와 다른 새 라인
- jihun/subin/minjae/yuna는 학년 무관 톤 (초등~고등)
- haeun은 중등~고등 톤 (선배 역할 유지)
- junha는 고등 톤 (전학생, 부산 사투리 가끔)

**2. 부모 잡담 추가 — 각 강점당 7개씩 (총 42개)**
- 기존 3개와 다른 새 라인
- 엄마/아빠 분포는 자유 (한쪽 편향 X — `엄마` 중심이지만 가끔 `아빠`도)

### 다음 단계 (잡담 풀 확장 후)

미니 이벤트 풀 확장도 같은 방식으로 진행 가능. 별도 프롬프트 만들 때:
- 기존 미니 이벤트 구조: `{ id, npcId/parentStrength, intimacyMin, description, effects, message }`
- description은 1~3 문장, 캐릭터 톤
- effects는 작게 (친밀도 +1~3, 스탯 +1~2)
- 각 NPC × 친밀도 단계(30~50, 50~70, 70+)별 1~2개씩 → 풀 6배 확장 가능
