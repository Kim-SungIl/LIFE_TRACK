# 잡담 풀 발주 프롬프트 — LIFE_TRACK NPC Smalltalk (친밀도 × 학년 톤 분기)

> **이 문서의 용도**: 외부 LLM(GPT·Gemini는 이 파일을 그대로 업로드, Codex·Cursor는 CLI로 동일 내용 전달)에게
> 게임 NPC들의 "잡담(smalltalk)" 대사 풀을 **친밀도 구간 × 학년(학교급) × 성별**로 발주하기 위한 자기완결 명세서다.
> 레포 접근 없이 이 문서만 읽고 작업할 수 있도록 필요한 컨텍스트를 모두 담았다.

---

## ⚠️ 작업 전 확인 — 티어 경계 가정

이 문서는 친밀도 티어 경계를 게임 내 **미니 이벤트와 동일하게 30 / 50 / 70** 으로 가정한다 → 4구간(base/warm/close/deep).
만약 발주자가 **3구간**(예: <30 / 30~70 / 70+)을 원했다면, 작업 시작 전에 알려달라. 구간 정의만 바꾸면 나머지 지침은 동일하다.

| 티어 | 친밀도 | 한 줄 정의 |
|---|---|---|
| `base` | 0~29 | 갓 알게 됐거나 아직 데면데면. **이미 게임에 존재 → 발주 대상 아님(참고용)** |
| `warm` | 30~49 | 친해지는 중. 관심·호의 표현, 가벼운 농담 |
| `close` | 50~69 | 편한 사이. 사소한 취향·비밀 공유, 장난, 챙겨주기 |
| `deep` | 70+ | 속마음. 약한 모습·진심·서로 의지하는 톤 |

**발주 대상 = `warm` / `close` / `deep` 신규 라인.** base(기존 라인)는 학년 중립이라 그대로 둔다.

---

## 1. 게임 한 줄 소개 & 잡담이 뭔지

- **LIFE_TRACK**: 초등학교 6학년(Y1)부터 고등학교 3학년(Y7)까지 7년을 한 주씩 살아가는 한국 학창시절 시뮬레이션. 주인공(플레이어)은 남/여 선택 가능.
- **잡담(smalltalk)**: 플레이어가 NPC에게 "말 걸기"를 했는데 그 주에 특별한 미니 이벤트가 발동되지 않았을 때 나오는 **짧은 대사 한 줄**. "이번 주는 별일 없는 평범한 만남"을 표현한다.
- 화면에는 NPC 초상·이름이 따로 표시되고, **잡담 라인은 그 NPC가 지금 너에게 말 거는 대사** 한 줄로 출력된다.

### 학년(연차) ↔ 학교급 매핑

| 연차 | 학년 | 학교급(level) |
|---|---|---|
| Y1 | 초6 | `elementary` (초등) |
| Y2~Y4 | 중1~중3 | `middle` (중등) |
| Y5~Y7 | 고1~고3 | `high` (고등) |

→ **같은 NPC라도 플레이어가 어느 학교급일 때 말 거느냐에 따라 소재가 달라야 한다.** (초등생에게 진로·대학·입시 얘기는 어색하다.)

---

## 2. 무엇을 써야 하나 — 출력 차원

각 NPC에 대해 아래 3축의 조합별로 대사 라인을 쓴다:

```
NPC  ×  티어(warm/close/deep)  ×  학교급(elementary/middle/high, NPC가 존재하는 급만)  ×  성별 트랙(common / male / female)
```

- **성별 트랙**:
  - `common` = 플레이어 성별 무관(기본). **대부분의 라인은 common.**
  - `male` / `female` = 플레이어 성별에 따라 활동·소재가 달라질 때만 추가 (예: 함께 하는 운동 종목이 다름). 불필요하면 비워도 된다.
- **존재 학교급**: NPC마다 다르다(§8 각 프로필 참조). 존재하지 않는 급은 쓰지 않는다.

## 3. 공통 작성 규칙 (반드시 지킬 것)

### 형식
1. 각 라인은 **그 NPC가 입으로 말하는 한 줄 대사**. 게임 저장 형식대로 **큰따옴표로 감싼다**. 예: `"오늘 급식 뭐 나오지? 나 배고파 죽겠어."`
2. **1~2문장, 구어체**. 너무 길지 않게(모바일 화면 한 줄~두 줄).
3. 한국어. 캐릭터 말투(사투리 등) 유지.

### 톤
4. **진행 중인 대화**처럼. NPC가 지금 너 앞에서 말 거는 느낌.
5. **종결형 묘사 금지**: "헤어졌다 / 가버렸다 / 지나갔다 / ~했다" 같이 장면이 끝난 듯한 서술 금지. (이건 대사지 나레이션이 아니다.)
6. **나레이션·지문 금지**: 따옴표 밖 행동 묘사(`지훈이 웃는다` 등) 넣지 말 것. 순수 대사만.
7. **플레이어 대사 금지**: NPC가 말하는 것만. 플레이어 답변을 가정해 쓰지 말 것.

### 연령·맥락 적합성 (이번 작업의 핵심)
8. **학교급에 맞는 소재만**:
   - `elementary`(초6): 급식, 학원 숙제, 쉬는 시간 놀이, 분식, 게임, 가족, 사소한 동네 일. **진로/대학/입시/수능/연애 진지 금지.**
   - `middle`(중1~3): 시험·내신, 동아리, 친구관계, 약간의 미래 고민(어렴풋이), 취미. 입시는 아직 멀게.
   - `high`(고1~3): 진로·성적·입시 압박, 야자/자습, 미래 불안, 더 깊은 속마음. 단 무겁기만 하지 않게 일상도 섞기.
9. **우정 톤 유지**: 연애·고백 직설 금지. 친밀해져도 "친한 친구/선후배"의 따뜻함이지 로맨스가 아니다. (호감의 은근한 결은 가능하나 고백·연애 단정 금지.)
10. **무거움 과잉 금지**: deep 티어라도 매 라인이 신파면 안 된다. 가벼운 일상 + 가끔 진심 한 조각이 섞이는 비율.
11. 특정 시점·사건에 못 박지 말 것(잡담은 아무 주에나 반복 노출됨). "어제 그 시험" 같은 1회성 사건 단정보다 반복 가능한 소재로.

## 4. 티어별 톤 가이드 (warm → close → deep)

- **warm (30~49) — 친해지는 중**
  먼저 다가오고 관심을 표현. "같이 ~할래?", 가벼운 농담, 사소한 칭찬. 아직 약간의 거리감/예의.
- **close (50~69) — 편한 사이**
  둘만 아는 농담·취향·습관 언급. 챙겨주기("이거 너 좋아하잖아"), 편한 장난, 사소한 비밀 공유. 거리감 거의 없음.
- **deep (70+) — 속마음**
  약한 모습·고민·진심을 슬쩍 보여줌. 서로 의지하는 톤. "너한테니까 하는 말인데", "혼자 다 참지 마". 단 §10대로 무겁기만 하지 않게.

각 티어는 **아래 티어를 대체하는 게 아니라 풀에 누적**된다(친밀도 오를수록 따뜻한 라인이 등장 확률에 더해짐). 그러니 티어가 올라갈수록 **그 티어다운 새로운 결**을 더해주면 된다.

## 5. 출력 형식 / 코드 매핑

아래 **마크다운 헤더 + 불릿** 형식으로 출력하라. (JSON은 한국어 큰따옴표 이스케이프 문제로 쓰지 않는다.)

```
## NPC: jihun (지훈)

### warm / elementary / common
- "라인1"
- "라인2"
...

### warm / elementary / male
- "라인1"
...

### close / middle / common
- "라인1"
...

### deep / high / common
- "라인1"
...
```

규칙:
- 헤더는 정확히 `### <tier> / <level> / <track>` 형식. tier∈{warm,close,deep}, level∈{elementary,middle,high}, track∈{common,male,female}.
- 해당 NPC에 **존재하지 않는 학교급은 헤더 자체를 만들지 말 것**.
- male/female 트랙은 그 조합에서 성별로 소재가 갈릴 때만. 없으면 생략.
- 불릿 한 줄 = 큰따옴표 포함 완성 대사 하나.

> 참고(개발자용): 이 출력은 코드에서 아래 구조로 매핑된다 — 작성자는 신경 쓸 필요 없음.
> ```ts
> type GenderedPool = { common?: string[]; male?: string[]; female?: string[] };
> type SmalltalkBucket = { elementary?: GenderedPool; middle?: GenderedPool; high?: GenderedPool };
> type NpcSmalltalkPool = { base: ...; warm?: SmalltalkBucket; close?: SmalltalkBucket; deep?: SmalltalkBucket };
> ```

---

## 6. 분량 목표

- `common` 셀(티어×학교급): **각 6~8줄**
- `male`/`female` 셀: 필요한 조합만, **각 3~4줄**
- 중복·동어반복 금지. 같은 소재라도 표현·상황을 달리.

예: 지훈(3개 학교급 전부 존재) → warm/close/deep × elementary/middle/high = 9개 common 셀 × 6~8줄 ≈ 60~70줄 + 성별 트랙 약간.

## 7. 캐릭터별 프로필 (성격·말투·존재 학교급·기존 잡담 예시)

> 각 NPC의 **성격/말투를 최우선**으로 반영하라. 존재 학교급 외엔 쓰지 않는다.
> "기존 base 예시"는 톤 감을 잡기 위한 참고용(=친밀도 무관 라인). 이걸 베끼지 말고 더 친한 결로 발전시켜라.

### jihun (지훈) — 소꿉친구 / 존재: **초·중·고 전부**
- **성격**: 활발하고 운동 좋아하는 소꿉친구. 항상 먼저 말 걸어줌. 초등 때부터 알던 사이라 시작 친밀도가 높음(말 텄음).
- **말투**: 밝고 직설, "야~" 로 시작하는 친근체.
- **성별 분기**: 남주와는 농구/축구, 여주와는 배드민턴을 같이 함(활동 종목이 갈림).
- **기존 base 예시**:
  - common: `"오늘 급식 뭐 나오지? 나 배고파 죽겠어."` / `"매점 줄 지금 짧대. 지금 가면 우리 이긴다, 가자."` / `"너 요즘 좀 조용한데? 무슨 일 있으면 나한테 먼저 말해도 돼."`
  - male: `"야 너 어제 그 농구 경기 봤어? 막판에 진짜 대박이었어."`
  - female: `"라켓 챙겼어? 체육관 잠깐 비었대, 한 판만 치자."` / `"우리 어릴 때 배드민턴 친 거 기억나? 그땐 네트도 없었잖아."`

### subin (수빈) — 같은 동네 학원 친구 / 존재: **초·중·고 전부**
- **성격**: 어디서든 금방 친해지는 외향형. 친구 많아 보이지만 어느 그룹에서든 "같이 어울리는 애" 포지션(은근한 외로움). 엄마와 둘이 산다.
- **말투**: 다정하고 말 많은 편, 책·문장 좋아함. 속마음을 잘 털어놓음.
- **성별 분기**: 거의 common (특별한 성별 활동 분기 약함).
- **기존 base 예시**: `"이 책 봤어? 빌려줄까? 너도 좋아할 것 같아."` / `"나 사람들 이름은 잘 외우는데, 가끔 내 마음은 잘 모르겠더라."` / `"너는 친구 많아 보이는데도 가끔 혼자 있고 싶을 때 있어?"`

### minjae (민재) — 전교 1등(노력형) / 존재: **초·중·고 전부**
- **성격**: 천재가 아니라 노력형. 새벽까지 공부하고 아침에 태연한 척. 학원 원장 엄마 + 교사 아빠의 교육 가정. 완벽주의·자기검열.
- **말투**: 차분하고 약간 자조적인 유머. 문제풀이·효율 얘기를 일상처럼.
- **성별 분기**: 여주와는 체육 종목 칭찬 라인이 일부 있음(기존 female 예시 참고). 대부분 common.
- **기존 base 예시**: `"이 문제, 답은 맞았는데 풀이가 마음에 안 들어. 너는 어떻게 생각해?"` / `"나 안 피곤해 보여? 그럼 성공이네. 오늘은 티 안 나게 버티는 중이야."` / `"잠깐 쉬는 것도 효율에 포함해야겠지? 이론상으로는 그래."`

### yuna (유나) — 밝은 우등생 / 존재: **초·중·고 전부**
- **성격**: 상위권인데 공부벌레 느낌은 아님. 밝고 에너지 넘침, 피아노 잘 침, 친구 잘 사귐. 그 밝음 뒤에 "1등 놓치면 안 된다"는 압박.
- **말투**: 텐션 높고 빠름, 음악·하늘·색깔 같은 감각적 소재. deep로 갈수록 밝음 뒤 불안을 슬쩍.
- **성별 분기**: 대부분 common.
- **기존 base 예시**: `"나 어제 진짜 좋은 노래 발견했어. 한번 들어볼래?"` / `"나 웃고 있으면 다 괜찮아 보이나 봐. 근데 진짜 괜찮은지는 나도 모르겠어."` / `"너는 긴장될 때 어떻게 해? 나는 웃고 있는데 손이 먼저 떨리더라."`

### doyun (도윤) — 초등 체육부장 / 존재: **초등(Y1)만** (중학교 진학 때 학군 이사로 갈라짐)
- **성격**: 반에서 모두가 따르는 자연스러운 리더. 축구 잘하고 예의 바름. "괜찮아"를 입에 달고 사는 착한 아이 페르소나 — 부모 기대에 부응하느라 속마음 잘 안 보여줌.
- **말투**: 씩씩하고 배려 깊음. 힘든 것도 "괜찮아"로 넘기려 함(deep에서 그 틈이 살짝 보임).
- **주의**: **elementary만** 쓴다. 한 학년 안이라 친밀도가 deep(70+)까지 오르긴 어려움 → **warm 위주, close 약간**이면 충분(deep는 최소 또는 생략 가능).
- **성별 분기**: 남주와 축구/계주, 여주와는 물건 옮기기·응원 같이 하기.
- **기존 base 예시**: common `"선생님이 또 나한테 심부름 시키신다. 뭐, 괜찮아."` / `"무릎 좀 까졌는데 별로 안 아파. 진짜야, 멀쩡해."` / male `"야! 점심에 축구 나갈래? 한 명 모자라."` / female `"이거 책상 같이 옮기자, 혼자 들기엔 좀 무거운 것 같아."`

### haeun (하은) — 1년 위 선배 / 존재: **중등(Y2~Y3) 중심** (+ 고등 초반 Y5에 짧은 재회)
- **성격**: 중2~중3 선배. 겉으론 여유롭지만 고교 진학 + 오빠의 수능 실패 트라우마로 속은 불안. 후배 챙기며 자기 자신을 다독이는 사람.
- **말투**: "야, 존댓말 하지 마. 어색해." — 편하게 대하길 원함. 다정하고 어른스럽지만 가끔 자기 불안이 비침.
- **존재 급**: 주로 `middle`. `high`는 Y5 재회용으로 **소량만**(있어도 1~2 셀).
- **성별 분기**: 대부분 common.
