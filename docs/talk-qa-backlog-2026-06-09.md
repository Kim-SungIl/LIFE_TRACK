# 대화 시스템 QA 백로그 (2026-06-09)

QA 5인(학년별 Y1~Y7) + codex/cursor 리뷰에서 나온 발견 중, **이번 작업(부모 잡담 학교급/시점 톤) 범위 밖**의 기존 시스템 이슈를 모은다. 이번 PR(`feat/parent-smalltalk-school-tone`)은 부모 잡담만 다루고, 아래는 별도 작업으로 분리.

> 참고: 도달형 이벤트의 학년 게이트는 의도된 설계(한 판에 다 보는 게임 아님)라는 전제는 유지. 아래 "게이트 부재"는 버그 단정이 아니라 SSOT 일관성/방어 차원.

---

## 1. NPC 친밀도 티어 콘텐츠 공백 (QA2·3·4·5 교차)

- **junha(준하)**: 잡담 `warm/close/deep`이 `high` 셀만 존재. 고등 전용 캐릭터라 정상이나, 만약 중등에서 met 처리되면 미니이벤트는 뜨는데 잡담은 base만 나와 톤 붕괴.
- **doyun(도윤)**: 잡담 `warm/close/deep`이 `elementary` 셀만 존재. → **Y4(중3)·Y7(고3)에서 친밀도 30/50/70+를 올려도 base 잡담만 반복**. 다른 NPC는 학교급 티어 셀이 있어 친밀도별로 깊어지는데 도윤만 평탄.
- **조치안**: doyun에 `middle`/`high` 티어 셀 추가(타 NPC와 동일 구조). junha는 `middle` 공백이 설계 의도면 유지하되 주석 명시.

## 2. NPC 미니이벤트 yearMin 게이트 부재 (QA1·2)

- `talk_junha_*`(riceball/50_seabreeze/70_speech/90_umbrella): 고2 전학생인데 yearMin 없음. 대조적으로 `talk_haeun_90_empty_line`만 `yearMin:6` 명시 → 게이트 일관성 깨짐.
- **80-tier 회상 6종**(`talk_*_90_*`): intimacyMin:80에 year 게이트 없음. 지훈은 Y1 시작 친밀도 40이라 초6에 80 도달이 이론상 가능 → 고등 졸업/인생 회상 정서가 초6에 노출될 위험.
- **조치안**: junha 미니이벤트 4종에 `yearMin:5`(또는 6), 80-tier 코어에 `yearMin:5` 부여 검토. 현재는 met 게이트로 가려져 실害은 낮음.

## 3. haeun(하은) 선배 학사 정합성 (QA4·5)

- 하은은 "선배(1년 위)" 설정인데, 잡담 `high` 셀에 학년 게이트가 없어 **플레이어가 고2·고3이 되어 하은이 졸업했을 시점에도 교내(자판기·자습실)에서 상시 등장**.
- 미니이벤트 `talk_haeun_90_empty_line`은 `yearMin:6`(졸업 강당)이라 시점 정합하나, "졸업을 앞둔 강당"이 고2 연중 아무 주차에나 발동 가능(시점 폭).
- **조치안**: 하은 high 잡담을 "졸업 후 가끔 찾아온/연락하는" 톤으로 조정하거나 상급학년 노출 축소. 도달형 게이트 설계 전제와 함께 기획 판단 필요.

## 4. minjae(민재) base female 운동 칭찬 (QA2·4)

- `NPC_SMALLTALK.minjae.female`(base)의 "서브 잘 들어가더라" 등 운동 칭찬이 **친밀도 0·전 학년 노출**. 공부형·효율 중심 캐릭터의 base 톤과 충돌.
- **조치안**: 해당 2줄을 `warm.middle`/`warm.high` 등 친밀도 게이트 셀로 이동.

## 5. 부모 미니이벤트(잡담 아님) 초6 톤 (QA1)

- `talk_parent_info` 기본: "그 분야 요즘 전망 좋대 / 학원 이름"이 yearMin 없이 Y1 발동 → 초6에 진로 못박는 어른 화법. (잡담 `info.elementary`는 이미 "넌 뭐 할 때 재밌어?"로 잘 처리됨)
- `talk_parent_strict` 기본: "이번엔 잘 봐야 한다. 11시까지는 자고" → 초6엔 성적압박·취침시각 과함.
- **조치안**: Phase 2 부모 미니이벤트에 elementary 변주 분리 또는 본문 완화. (이번 잡담 작업과 별개 모듈)

## 6. 부모 info 강점 — 중3 고입 콘텐츠 공백 (QA3)

- 중3(Y4)은 첫 진학 결정(고교 선택)의 해인데, Y4 발동 부모 info 이벤트는 "전망/학원"(`talk_parent_info`)뿐. 고입 특화 `info_2`는 `yearMin:5`로 고1부터. 잡담엔 고입 언급 있으나 이벤트 레벨엔 부재.
- **조치안**: info 강점에 `yearMin:4` 고입 이벤트 1개 추가 검토.

## 7. NPC 잡담 high 셀 입시 정서 누수 (QA4)

- `subin.warm.high` "입시 얘기만 하다 보면…"이 친밀도 30+로 **고1 초반부터 노출**(high는 고1~고3 공유). "입시" 직접어가 고1엔 이르다.
- **조치안**: "시험/성적 얘기"로 완화하거나, 입시 직접어는 고3 전용 셀로. (NPC 잡담 구조에 senior 시점 분기를 도입할지와 연계 — 부모 잡담처럼)

- `talk_parent_info_2`(yearMin:5), `haeun_90`(yearMin:6) 학년 게이트는 의도된 정상 동작.
- 부모 미니이벤트 선택지 label↔effects/message/resultText 정합성: 전수 확인 일치.
