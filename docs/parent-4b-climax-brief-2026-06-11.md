# Phase 4B 설계 검토 브리프 — 부모 "강점별 절정 순간"

당신은 최고의 게임 시나리오 작가입니다. 아래 설계안을 읽고 **핵심 질문**에 대해 당신의 판단을 한국어로 답해 주세요. 코드 수정은 하지 말고, 판단과 근거·구체안만 주세요.

---

## 게임 맥락
- 한국 7년 학창시절 시뮬레이션(초6→고3, 학년당 48주). React/TS, 모바일.
- `parentIntimacy`: 0~100 **숨은 스탯**, 50에서 시작. 플레이어에게 수치로 안 보이고 **대사·이벤트·엔딩 서사**로만 드러남.
- 부모는 게임 시작 시 **2개 강점**을 가짐(6종 중 택2): `emotional`(정서지지)·`wealth`(여유)·`info`(정보)·`strict`(엄격)·`resilience`(회복탄력)·`freedom`(자율).
- **화자 SSOT(고정)**: 엄마 = emotional·info·resilience·freedom / 아빠 = wealth·strict. (강점별 화자 절대 혼재 금지)

## 이미 있는 것
1. **PARENT_MINI_EVENTS (6종, 반복 발동)** — 강점당 1개씩, 일상 잡담 레벨. 선택지 2개(긍정/부정), 친밀도 델타 + 작은 스탯 + 가끔 회상 슬롯(`memorySlotDraft`). 예: emotional "핫초코 내려놓으며 '힘들면 힘들다고 해'". **매주 무작위로 계속 뜨는 저강도 반복 콘텐츠.**
2. **strict 연 1회 칭찬 어드밴티지** — strict 부모 + 뚜렷한 성적 향상(상위권) 시, **연 1회**(`parentPraiseYears` 가드) "아빠가 드물게 '잘했다'(멘탈 +2)". 강점이 결정적으로 보답하는 순간. **단, 6축 중 strict만 이런 절정이 있음.**
3. **Phase 4A (방금 머지됨)** — *엔딩* 회고에 친밀도 tier(distant/normal/warm)×강점별 부모 챕터. 게임이 끝난 뒤 회고.

## Phase 4B 제안: "강점별 절정 순간"
strict만 가진 "이 부모라서 결정적으로 좋은 순간"을 **6축 전부로 일반화**. 게임 *도중*(엔딩 아님) 강점별로 "이 부모라서 좋았던 단 한 장면"을 터뜨림. wealth·freedom이 특히 평면적이라 캐릭터성 완성이 목적.

설계 문서 §10의 6축 컨셉 씨앗:
- wealth: "말없이 통장을 만들어뒀더라"
- freedom: "끝까지 안 말렸다"
- resilience: "일으키지 않고 일어설 때까지 기다려준"
- info: "내 진로를 같이 찾아줬다"
- strict: (기존 칭찬 어드밴티지를 절정으로 승격·정리)
- emotional: "늘 곁에 있던" 결정적 한 장면

## 절대 원칙 (어기면 안 됨)
- **스탯 퍼주기 금지**: 절정 순간은 *서사·기억(회상 슬롯)* 위주. 스탯 효과를 준다면 strict 칭찬처럼 **절제된 1회성/연 1회 가드** 수준만. 친밀도가 진로·성취 결과를 직접 올리지 않음.
- 화자 SSOT 유지.
- 연령 게이트는 의도된 설계(한 판에 다 못 봄). 절정이 특정 학년/조건에서만 떠도 정상.

## 핵심 질문 (반드시 각각 답할 것)

**Q1. 절정 순간은 강점당 "딱 1회(평생 1번)"가 좋은가, 아니면 조금 더 늘리는 게 좋은가?**
- 1회의 장점: 희소성→감동 극대화, "절정"의 의미 유지, 남발로 인한 평면화 방지.
- 늘리기의 장점: 7년 중 그 강점을 못 보고 지나칠 위험↓, 관계 곡선의 굴곡 표현.
- 당신이 작가라면 몇 개가 적정한가? (예: 강점당 1회 / 학교급당 1회=최대 3회 / 친밀도 임계 도달 시 1회 + 엔딩 직전 1회 / 기타) **구체적 숫자와 발동 조건**으로 답할 것.

**Q2. 발동 트리거는 무엇이 좋은가?** (친밀도 임계값 도달? 특정 학년·주차? 누적 태그 횟수? 특정 선택 직후 보상?) 스탯 퍼주기 없이 "쌓을 이유"가 되려면.

**Q3. 6축 절정 순간 각각에 대한 한 줄 씬 제안** — 위 씨앗을 다듬거나 더 나은 안. 화자 SSOT 지키고, 진부함(클리셰)·강점 성격 불일치를 피해서.

**Q4. 위험·반대 의견** — 이 기능이 오히려 게임을 해칠 수 있는 지점, 4A(엔딩 회고)와 중복되는 위험, 반복 미니이벤트와의 톤 충돌 등.

답변은 Q1~Q4 구조로, 한국어로.
