# Phase 4D "사춘기 골짜기" 검수 브리프

당신은 한국 가족 정서에 정통한 최고의 게임 시나리오 작가입니다. 아래 구현된 콘텐츠를 읽고 **이상한 부분 / 한국정서에 과한 부분(신파·오글거림·현실 안 맞음)**이 없는지 검수해 주세요. 코드 수정은 하지 말고, 발견과 구체적 수정안만 한국어로 주세요.

## 읽을 파일
- `game/src/engine/events/parent-adolescence.ts` — 사춘기 골짜기 이벤트 2종(방문 쾅 / 그땐 미안했어)
- `game/src/engine/parentIntimacy.ts` — 사춘기 한시 배율 하락(×0.75) 블록 ((1.5) 주석 부근)

## 맥락
- 게임: 한국 7년 학창시절 시뮬(초6→고3). parentIntimacy는 0~100 숨은 스탯, 대사·이벤트·엔딩 서사로만 드러남.
- 4D는 중2~3(Y3~4)의 "관계 골짜기→회복" 곡선:
  1. Y3~4엔 유대 행동(가족시간/고민공유)의 친밀도 반응이 ×0.75로 한시 둔화("이 시기 하락은 정상").
  2. 방문 쾅 이벤트(Y3): 방문쾅(친밀도 −, 멘탈+2 후련함) / 삼킴 / 솔직(친밀도 +).
  3. 화해 비트(Y5): 골짜기를 겪고 친밀도를 60+로 회복했을 때만 해금.
- 화자: 강점별 SSOT(엄마: 정서/정보/회복/자유, 아빠: 부유/엄격)가 있으나, 4D는 보편 비트라 clash는 '엄마'(일상 잔소리 디폴트), reconcile은 화자 비명시(어느 부모로도 투영).
- 원칙: 스탯 퍼주기 금지(작은 멘탈 + 친밀도 굴곡 + 회상만). 진로/성적 결과는 불변.

## 검수 포인트 (각각 답해주세요)
1. **한국정서 과잉**: 신파("미안했어" 류 과장된 화해), 오글거림, 드라마 클리셰가 있는가? 특히 reconcile "그땐 미안했어" 비트가 과한가, 적정한가?
2. **현실 정합**: 한국 중2~3 사춘기 마찰("너 요즘 왜 그래?" / 방문 쾅 / 부모의 뒤늦은 사과)이 실제 가정의 결과 맞는가? 어색하거나 과장된 대사가 있는가?
3. **화자 처리**: clash가 '엄마' 고정, reconcile이 화자 비명시인 게 자연스러운가? 부모가 모두 아빠 성향(부유+엄격)인 가정에서도 어색하지 않은가?
4. **선택지 톤**: 3선택(방문쾅/삼킴/솔직)과 화해 2선택의 대사·결과 문구가 사춘기 당사자 1인칭으로 자연스러운가?
5. **회상 문구(recallText)**: 회상에 남는 한 줄들이 과하거나 어색하지 않은가?

심각도(높음/중간/낮음)로 분류하고, 동의/반대 판단이 갈릴 수 있게 구체적으로.
