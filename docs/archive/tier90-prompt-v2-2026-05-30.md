## AI에게 줄 프롬프트 (이 아래부터 복사)

당신은 한국 학교생활 성장 시뮬레이션 게임의 시나리오 작가입니다. 미니 이벤트(말걸기)의 **최상위 단계(친밀도 90, 코어)** 시드를 **탐색적으로** 작성해 주세요.

### 가장 중요한 요청
NPC 6명 각각에 대해 **서로 확연히 다른 90단계 장면 컨셉을 3개씩** 주세요(총 18개). 3개는 **축·장소·감정·소재가 서로 달라야** 합니다. 같은 아이디어의 변주(문장만 바꾼 것)는 금지. "가장 뻔한 정답" 1개 + **의외이지만 그 캐릭터답게 설득되는 대안 2개**를 섞어 주세요. 채택자는 이 중에서 고를 겁니다 — 다양할수록 좋습니다.

### 게임 배경
- 한국 12세(초6)→18세(고3) 7년. 학년: Y1=초6 … Y7=고3. NPC와 친밀도(0~100)를 쌓음.
- 미니 이벤트 = "말 걸기" 시 발동되는 짧은 장면. **선택지 없음**, description + 효과 한 줄. NPC당 평생 1회.

### 90 단계 톤 — "캐릭터 결정적 한 장면 (코어)"
- 30=가벼운 호의 / 50=둘만의 코드 / 70=속마음 한 조각(균열, 다시 일상) / **90=그 NPC를 "그 NPC답게" 만드는 결정적 전환점. 7년 후 회상으로 떠오를 한 장면.**
- **70 vs 90**: 70은 스스로 멈추는 균열, 90은 정체성·관계·미래가 걸린 전환점.
- **절제가 핵심**: 무게는 **감정 폭발이 아니라 행동·관계 인식의 변화**로. 멜로드라마·장황한 독백·눈물 클라이맥스 금지. 90이라도 시끄럽지 않게.
- "너이기 때문에" — 무리/세상엔 안 보이고 플레이어에게만 보여주는 결.
- (※ "고백"은 연애 아니라 자기 내면 털어놓기)

### 기존 30/50/70 시드 (중복 회피 — 이 결을 반복하지 말 것)
- jihun: 운동(30)/떡볶이 식성(50)/사물함 "잘하고 싶다" 자기인식(70)
- subin: 문제풀이(30)/접힌 문장(50)/거실 불빛 가정사 암시(70)
- minjae: 노트(30)/오답 세모(50)/뒤집은 휴대폰 기대압박(70)
- yuna: 추천곡(30)/즉흥 멜로디(50)/분필가루 칭찬압박(70)
- haeun: 자판기 콜라(30)/복도 창문(50)/계단참 "아는 척의 피로"(70)
- junha: 주먹밥(30)/부산 바람(50)/식판 사투리=정체성(70)

### ⚠️ 회피 가드 (반드시 — 위반 시 탈락)
- **minjae**: 30·50(학습도구)·70(기대/성적/가족 압박)이 누적. **90은 기대·성적·가족·공부 축 전면 금지.** (다른 결은 자유)
- **junha**: 30·50·70이 부산·향수·정체성·사투리·음식 축. **90은 이 축 전부 금지(4연속 방지).** 사투리 어조는 캐릭터 voice로만, 주제로는 X. (다른 결은 자유)
- **haeun**: 30·50·70 모두 정지공간(자판기·창문·계단참). **90은 다른 공간/구도.**
- **yuna**: 70이 음악 축에서 벗어남. **90은 음악/소리/연주 감각 축으로 돌아올 것** (단 "칭찬 압박" 반복은 금지 — 압박이 아닌 음악 자체).
- **jihun / subin**: 위 30/50/70 결만 피하면 방향 자유. (특정 정답 강요 안 함 — 3안으로 탐색)

> 회피 가드는 "하지 말 것"이고, 그 안에서 **방향은 당신이 탐색**하세요. 뻔한 1개 + 의외의 2개.

### 형식·제약 (코드 적용용)
각 시드는 아래 형태. **각 NPC 3안 모두 작성.**
```typescript
{
  id: 'talk_<npcId>_90_<keyword>',
  npcId: '<npcId>', intimacyMin: 90,
  description: '<NPC 짧은 직접 대사 + 행동/상황 1~3줄, 절제>',
  effects: { intimacy: 5, stats: { <스탯키>: 1~2 }, fatigue: -2~2 },  // intimacy +5 고정, 스탯 ±1~2
  // 🔒 stats 키는 이 5개만: academic(학업)·social(사회)·talent(특기)·mental(멘탈)·health(체력). 그 외(courage/empathy 등) 금지 — 닫힌 타입.
  message: '<NPC다운 짧은 플레이버 한 줄>',  // "정신력 상승했다" 식 설명 X
  memorySlotDraft: {
    category: <'courage'|'betrayal'|'reconciliation'|'failure'|'discovery'|'growth'>,  // 이 6개만
    importance: 5,                       // 고정
    toneTag: <'warm'|'regret'|'resolve'|'breakthrough'|'melancholy'|'burden'>,  // 이 6개만
    recallText: '<20~35자 회상문, 스탯 단어 금지>',
    npcIds: ['<npcId>'],
  },
}
```
- 🔒 **category/toneTag는 위 목록 값만** (임의값 금지 — 닫힌 타입이라 컴파일 깨짐).
- 종결형 묘사 금지("~하고 가버렸다"), 학년 종속 용어 금지(야자/수능/대학; 단 haeun 졸업·junha 사투리는 캐릭터 사실로 허용), 게임에 없는 시스템(선물/사진) 가정 금지, 선택지 금지.

### 출력
NPC별로 묶어서, **3안마다 한 줄 "이 안의 축/의외성" 메모**를 붙여 주세요. 18개 객체 + 메모. (장면을 베끼지 말고 당신이 새로 발상)
