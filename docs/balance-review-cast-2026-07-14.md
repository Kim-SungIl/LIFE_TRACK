## ② scarcity — "전원 절친" 가능 여부

| 플레이 | peak 70+ 인원 (10명 중) | 비고 |
|---|---|---|
| balanced (일반) | **5명** | 신규 3인 peak 8~10 (intro 후 방치) |
| social-max (관계몰빵, 동행 없음) | **5명** | |
| all-friends-max (동행 2슬롯 분산 + 말걸기) | **10명 전원 peak·final ≈100** (12/12시드) | **scarcity 붕괴** |

- 일반 플레이에선 scarcity가 잘 작동. 그러나 동행을 아는 min-max 플레이어는 전원 100 달성.
- 원인: **동행 +3, 미니톡 +2가 구간 감쇠(scaleIntimacyChange) 미적용 raw 가산** —
  이벤트 친밀도는 70+에서 60%, 85+에서 40%로 감쇠되는데 동행만 100까지 선형.
- 전원 100이어도 후회카드는 2장 노출되지만 전부 부모·자기 서사 — **관계 scarcity가 후회로
  표현되지 않음** (후회카드는 슬롯 기반이라 접점 없는/전부 친한 NPC는 재료가 없음, 기존 인지 이슈).
- **권고**: `applyNpcActivitySelection`(+3)과 미니톡 intimacy(+2)에 `scaleIntimacyChange` 적용.
  70+ 구간부터 동행 +1.8, 85+ +1.2, 95+ +0.75 → 전원 100 불가, 집중한 1~2명만 90+ 도달.
  일반 플레이(70 미만 구간)는 변화 없음 — 저구간 100% 유지라 체감 저하 없음.

## ⑤ 데뷔 타이밍 / 시작 친밀도

  **mini 이벤트 0개** —

  3. 신호-2(nextIntimacyThreshold)가 mini.intimacyMin을 읽는데 신규는 reach 임계만 노출 — 신호 갱신이 거칢
