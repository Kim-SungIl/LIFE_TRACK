# 07. 엔딩 시스템

## 설계 철학

> "엔딩은 성적표가 아니라 인생의 결산이다."
> "같은 대학에 가도, 옆에 누가 있느냐에 따라 완전히 다른 인생이다."

- 메인 엔딩(내가 뭐가 됐는가) + 사이드 엔딩(사람들은 어떻게 됐는가) 2중 구조
- "명문대 = 성공"이 아님. 행복 지수와 성취 지수를 분리
- **후회 지수**: 유저가 "다음엔 다르게 살아야지"라고 느끼게 만드는 핵심 장치
- 엔딩은 끝이 아니라 **리플레이의 시작**

## 1. 메인 엔딩 — "나는 어떤 사람이 되었는가"

### 1-1. 엔딩 판정 변수

수능 이후, 다음 변수들의 조합으로 메인 엔딩이 결정된다:

| 변수 | 산출 방식 |
|------|----------|
| 학업 최종값 | 0~100 |
| 재능 최종값 + 하위스킬 분포 | 음악/미술/체육/기술 중 최고값 |
| 인기 최종값 | 0~100 |
| 문과/이과 지수 | 학업 하위스킬 누적 비율 |
| 자립도 | 0~100 (은닉 스탯) |
| 부모 관계 최종값 | 0~100 |
| NPC 관계 상태 | 각 NPC 친밀도 + 관계 단계 |
| 생기부 활동 | 대회 입상, 동아리, 봉사 등 누적 |
| 진학 선택 | 일반고/특목고/자사고 |

### 1-2. 메인 엔딩 12종

> 원칙: 모든 엔딩에 고유한 가치가 있다. "좋은 엔딩/나쁜 엔딩"이 아니라 "다른 인생".

#### 학업 계열 (4종)

|------|------|------|

| **지방 거점국립대** | 학업 65~84, 생기부 보통 | "서울은 아니지만, 내가 고른 곳이다. 기숙사 신청서를 쓰면서 살짝 설렜다." |

## 2. 이중 지수 — 성취 vs 행복

> "명문대에 갔지만 외로운 사람"과 "평범하지만 주변이 따뜻한 사람"은 다른 인생이다.

### 2-1. 성취 지수 (Achievement Index)

사회적으로 "성공했는가"를 나타내는 지수. **외부 시선의 평가.**

```
성취 지수 =
  학업 최종값 × 0.3
  + 재능 최종값 × 0.2
  + 인기 최종값 × 0.1
  + 대회/수상 보너스 (최대 +15)
  + 진학 결과 보너스 (SKY: +20, 거점국립: +10, 특기자: +10)
```

| 등급 | 범위 | 라벨 |
|------|------|------|
| S | 80+ | "모두가 부러워하는 결과" |
| A | 60~79 | "충분히 잘한 삶" |
| B | 40~59 | "평범한, 하지만 의미 있는" |
| C | 20~39 | "쉽지 않았던 7년" |
| D | 0~19 | "아직 시작하지 못한 이야기" |

### 2-2. 행복 지수 (Happiness Index)

개인적으로 "행복한가"를 나타내는 지수. **내면의 평가.**

```
행복 지수 =
  멘탈 최종값 × 0.3
  + 친한 친구 이상 NPC 수 × 6
  + 절친 NPC 수 × 10
  + 결정적 순간 가중치 (아래 참조)
  + 부모 관계 최종값 × 0.15
  + (연애 유지 중이면 +10)
  + 자립도 × 0.05
  - 번아웃 경험 횟수 × 3
```

**결정적 순간 가중치** — 관계의 양보다 기억의 질:

| 순간 | 가중치 |
|------|--------|
| 수능 전날 통화 (딜레마 #19 ③ 선택) | +8 |
| 번아웃 시 NPC가 도와줌 (안전망 이벤트) | +5 (회당) |
| 질풍노도기 후 부모와 화해 | +6 |
| 딜레마 #9 병문안 선택 | +4 |
| 졸업식 동행 NPC 존재 (절친 1명+) | +5 |
| 딜레마 #20 수능 후 첫 연락에 상대가 바로 받음 | +4 |
| 연애 루트에서 이별 후 친구로 복귀 성공 | +3 |

> 친구가 많은 것보다 **중요한 순간에 곁에 있었던 사람**이 행복을 더 크게 좌우한다.

| 등급 | 범위 | 라벨 |
|------|------|------|
| S | 80+ | "나는 행복했다" |
| A | 60~79 | "좋은 시간이었다" |
| B | 40~59 | "아쉬움도 있지만" |
| C | 20~39 | "외로운 7년이었다" |
| D | 0~19 | "살아남은 것만으로도" |

### 2-3. 이중 지수 조합 연출

엔딩 화면에서 두 지수가 동시에 표시된다. 조합이 감정의 결을 만든다.

| 성취 | 행복 | 연출 톤 |
|------|------|---------|
| S | S | "모든 걸 가졌다. 드문 인생이다." |
| S | C | "다 이뤘지만, 전화할 사람이 없다." — **가장 강한 후회 유발** |
| C | S | "성적은 별로였지만, 주변이 따뜻했다." — **밸런스 엔딩의 가치** |
| C | C | "힘든 7년이었다. 하지만 끝나지 않았다." |
| A | A | "잘했다. 충분히." |
| B | S | "특별한 건 없었지만, 나는 행복했다." — **평범함의 가치** |

> 성취 S + 행복 C 조합이 이 게임의 **핵심 메시지**: "성공했지만 외로운 인생"은 진짜 성공인가?

## 3. 후회 지수 (Regret Index) — "놓친 것들의 무게"

> Gemini 제안 + GPT 제안을 종합. 엔딩에서 유저가 캡처해서 공유하는 핵심 콘텐츠.

### 3-1. 후회 지수 구성 요소

7년간의 플레이 데이터를 기반으로 **자동 집계**된다.

```
[관계 후회]
- 놓친 NPC 약속 횟수: ___회
- 읽고 답장하지 않은 메시지: ___개
- 연락 안 해서 멀어진 친구: ___명
- 절친이었다가 어색한 사이가 된 NPC: ___명

[선택 후회]
- 딜레마에서 "안 한" 선택지 중 후회 가중치가 높은 것: ___개
  (예: 병문안 안 감, 고백 안 함, 집 나간 뒤 화해 안 함)
- 루틴 변경 횟수: ___회 (적으면 "한 길만 걸었다", 많으면 "끊임없이 방황했다")

[시간 후회]
- 혼자 보낸 주말 수: ___주
- 방학 동안 아무도 안 만난 주: ___주
- 쉬지 않고 공부만 한 주: ___주

[가족 후회]
- 부모와 함께한 활동 횟수: ___회
- 부모와 충돌한 횟수: ___회
- 어버이날 선물 안 한 횟수: ___회
```

### 3-2. 후회 카드 표시

엔딩 화면에서 **가장 큰 후회 3개**가 카드 형태로 표시된다.
각 카드는 **통계 + 대표 사건 1개 + 한 줄 문장**으로 구성. 숫자가 근거, 사건이 얼굴, 문장이 감정.

```
[후회 카드 예시]

┌──────────────────────────────────────────┐
│  7년간 놓친 약속: 142회                    │
│  연락 안 해서 멀어진 친구: 4명              │
│                                          │
│  가장 많이 미룬 사람: 지훈 (17회)           │
│  끝내 답장하지 않은 마지막 메시지:           │
│  "야 괜찮아? 요즘 힘들어 보여서"            │
│                                          │
│  "독하게 살았다."                          │
└──────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│  가족과 함께한 시간: 89회                   │
│  절친으로 남은 친구: 3명                    │
│                                          │
│  가장 오래 함께한 사람: 지훈 (7년)           │
│  가장 기억에 남는 순간:                     │
│  수능 전날 밤, 30분 통화                    │
│                                          │
│  "따뜻하게 살았다."                        │
└──────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│  혼자 보낸 주말: 87주                      │
│  쉬지 않고 공부한 주: 124주                 │
│                                          │
│  가장 늦게 답장한 메시지: 13일 후            │
│  보낸 사람: 수빈                           │
│  "우리 오랜만에 만날래?"                    │
│                                          │
│  "버텼다. 그것만으로도."                    │
└──────────────────────────────────────────┘
```

### 3-3. [고독한 승리자] 특수 칭호

후회 지수가 특별히 높을 때(관계 후회 상위 20%) **전용 칭호와 에필로그**가 발동한다.

```
[조건: 성취 A 이상 + 절친 0명 + 멀어진 NPC 5명+]

칭호: [고독한 승리자]

전용 에필로그:
  "원하는 걸 얻었다.
   합격 발표 날, 교실은 축제 같았다.
   축하 문자가 왔다. 이름을 하나씩 읽었다.
   답장을 쓰다가 멈췄다.
   '고마워'라는 두 글자가 이상하게 무거웠다.

   돌아보니 아무도 없었다.
   아니, 있었는데 내가 돌아보지 않았던 거였다."

→ 히든 카드 "외로운 왕좌" 자동 획득
→ 이 에필로그가 이 게임에서 가장 아픈 엔딩이 되어야 함
```

### 3-3. 후회 한 줄 자동 생성

후회 지수 데이터 기반으로 **한 줄 문장**이 자동 생성된다. 이것이 SNS 공유의 핵심.

```
[생성 로직]

관계 후회가 가장 크면:
  → "성적은 올랐지만, 같이 급식 먹는 횟수는 줄었다."
  → "명문대에 갔다. 그런데 합격 문자를 보여줄 사람이 없었다."

시간 후회가 가장 크면:
  → "7년 중 혼자 보낸 주말이 87주였다."
  → "쉬지 않았다. 그래서 이뤘다. 그래서 지쳤다."

가족 후회가 가장 크면:
  → "부모님이 해준 도시락을 감사하다고 말한 적이 없다."
  → "어버이날은 매년 지나갔다. 한 번도 꽃을 사지 않았다."

후회가 거의 없으면:
  → "후회는 없다. 다만, 다른 선택을 했으면 어땠을까 하는 궁금증은 남는다."
```

## 4. 엔딩 화면 흐름

### 4-1. 순서

> 원칙: **먼저 울리고, 그다음 분석하게 한다.** 감정 몰입을 끝까지 끌고 간 뒤, 마지막에 회고와 수집 욕구를 준다.

```
[1] 수능 당일 연출 (딜레마 #20)
    → 교문을 나선다. 핸드폰을 켠다.
    → 가장 먼저 연락할 사람을 고른다.
    → 상대의 반응 분기: 바로 받음 / 안 받음(나중에 콜백) / 부모가 먼저 전화
    ↓
[2] 수능 이후 2주 (자유 구간)
    → 연락하고 싶은 사람에게 연락
    → 짧은 에필로그 이벤트들 (지훈과 PC방, 서아와 카페 등)
    ↓
[3] 합격 발표 / 진로 확정
    → 메인 엔딩 컷신
    → "나의 결과" 표시
    ↓
[4] "수능 끝나고 내가 가장 먼저 전화한 사람" 카드
    → 딜레마 #20에서 선택한 NPC
    → 그 NPC와의 7년 관계 요약 한 줄
    ↓
[5] 사이드 엔딩 순차 재생 ← 감정 피크 구간
    → "그리고, 그 사람들은..."
    → NPC별 에필로그 (05_NPC설계에 상세 기술)
    → 관계 궤적에 따라 분기
    ↓
[6] 졸업식 장면
    → "마지막으로 눈이 마주친 사람"
    ↓
[7] 최종 나레이션 (1차) ← 감정 정점
    → 메인 엔딩 + 이중 지수 조합으로 결정
    ↓
── 여기서부터 회고 구간 ──
    ↓
[8] 이중 지수 표시
    → 성취 지수 + 행복 지수 동시 공개
    → 조합에 따른 한 줄 코멘트
    ↓
[9] 후회 카드 표시
    → 후회 지수 3개 카드 (통계 + 대표 사건 + 한 줄)
    → [고독한 승리자] 칭호 (해당 시)
    → 공유 버튼
    ↓
[10] 인생 앨범 공개
    → 수집한 카드 전체 타임라인 열람
    → 새로 획득한 카드 하이라이트
    → 빈 슬롯 표시: "아직 만나지 못한 이야기가 ___개 있습니다"
    ↓
[11] "다시 살아볼까?" — 미련 + New Game
```

## 5. 사이드 엔딩 — "그리고, 그 사람들은..."

### 5-1. 재생 순서

NPC 에필로그는 **친밀도가 높은 순서**로 재생된다. 가장 소중한 사람이 가장 먼저.

```
[재생 조건]
- 친밀도 30 이상인 NPC만 에필로그 재생
- 친밀도 30 미만: "이름은 기억나지만, 얼굴이 흐릿하다."
- 한 번도 만나지 못한 NPC: 표시 안 됨
```

### 5-2. NPC별 에필로그 구조

각 NPC의 에필로그는 05_NPC설계에 상세 기술되어 있으며, 다음 변수에 따라 분기:

| 변수 | 영향 |
|------|------|
| 최종 친밀도 | 에필로그 톤 (따뜻함/아쉬움/무관심) |
| 관계 단계 | 절친/친한친구/보통/어색/연락없음 |
| 핵심 이벤트 선택 | 딜레마에서 뭘 골랐는지 (예: 병문안 여부) |
| 연애 루트 진입 여부 | 연애→유지/이별 분기 |
| 같은 학교 여부 | 진학 선택에 따른 관계 유지/약화 |

### 5-3. "졸업식 날 마지막으로 눈이 마주친 사람"

사이드 엔딩 마지막에, **졸업식 장면**이 삽입된다.

```
졸업식이 끝났다.
교문을 나서는데, 뒤를 돌아봤다.

[친밀도 1위 NPC]가 손을 흔들고 있었다.
[친밀도 2위 NPC]가 웃고 있었다.
[멀어진 NPC]는 이미 가고 없었다.

마지막으로 눈이 마주친 건 ________였다.
```

→ 마지막으로 눈이 마주친 NPC = 친밀도 1위 NPC (자동 결정)
→ 이 장면이 "졸업식 카드"로 인생 앨범에 저장됨

## 7. 리플레이 유도

### 7-1. "다른 인생" — 미련의 장치

> 원칙: 정보를 보여주는 게 아니라, **감정적 미련**을 심는다. "그날 답장했다면" 한 줄이 수치 열 개보다 강하다.

엔딩의 마지막, New Game 버튼 직전에 **미완의 장면 3개**가 스쳐 지나간다.

```
[미련 1 — 놓친 관계]
  가장 많이 친밀도가 하락한 NPC의 "유지했을 경우" 에필로그 첫 줄만 표시.
  그 위에 한 줄:
  "그날 답장했다면, 너희는 졸업식에서 같이 웃고 있었다."

[미련 2 — 다른 선택]
  핵심 딜레마 중 "선택하지 않은 길"의 결과가 한 줄로 표시.
  예: "특목고를 갔다면 — 민재와 3년을 더 보냈을 것이다."
  예: "동아리를 선택했다면 — 전국대회 무대에 서 있었을 것이다."

[미련 3 — 다른 가정]
  선택하지 않은 부모 조합의 장면이 흐릿하게 스쳐지나감.
  "만약 그 집에서 태어났다면, 당신의 7년은 완전히 달랐을 것이다."
```

> 핵심: 유저가 이걸 보고 "아, 저 장면은 절대 다시 보고 싶지 않다" 또는 "이번엔 그 애를 놓치지 않겠다"고 느끼면 성공.

### 7-2. New Game 선택지

```
[다시 살아볼까?]

① 처음부터 — 부모 선택부터 다시
② 같은 가정, 다른 선택 — 부모 유지, 이벤트 선택만 다시
③ 앨범 감상 — 다시 보기
```
