# 07. 엔딩 시스템

## 엔딩 구조: 메인 엔딩 + 사이드 엔딩 (2중 구조)

엔딩은 **"내가 뭐가 됐는가"**(메인) + **"나와 함께한 사람들은 어떻게 됐는가"**(사이드)로 구성.
같은 메인 엔딩이라도 관계 상태에 따라 감정적 결이 완전히 달라짐.

---

## 메인 엔딩 — 내 인생 결과

모든 엔딩에 고유한 가치. "명문대 = 성공, 알바 = 실패"가 아님.
**행복 지수 / 자아실현 지수**를 별도로 두어 사회적 성공과 개인적 만족을 분리.

| 엔딩 유형 | 예시 |
|-----------|------|
| 학업 | 명문대 진학, 연구자의 길 |
| 관계 | 인플루언서, 넓은 인맥의 사업가 |
| 재능 | 국가대표, 아티스트 |
| 밸런스 | 평범하지만 안정되고 행복한 삶 |
| 자수성가 | 비주류 루트에서의 성공 |

---

## 사이드 엔딩 — NPC별 관계 에필로그

메인 엔딩 이후, **관계를 맺었던 NPC별로 에필로그**가 펼쳐짐.
7년간의 관계 궤적(친밀도 변화, 주요 이벤트 선택)에 따라 분기.

```
[엔딩 화면 흐름]

1. 메인 엔딩 컷신 — "나의 결과"
2. 사이드 엔딩 순차 재생 — "그리고 그 사람들은..."
   ├─ 소꿉친구 에필로그
   ├─ 연애 상대 에필로그
   ├─ 라이벌 에필로그
   └─ ...
3. 인생 앨범 — 수집 카드 공개
```

### 사이드 엔딩 예시 (소꿉친구)

| 관계 궤적 | 에필로그 |
|-----------|---------|
| 7년 절친 유지 | "대학은 달라도 매주 만나는 사이. 결혼식 축사를 부탁받았다." |
| 중간에 멀어짐 → 재회 | "2년 만에 다시 만났을 때, 어색한 침묵 뒤에 웃음이 터졌다." |
| 연애 → 유지 | "같은 대학에 지원했다. 결과는 모르지만 후회는 없다." |
| 연애 → 이별 | "가끔 그 애 SNS를 본다. 잘 지내는 것 같아서 다행이다." |
| 완전히 멀어짐 | "동창회 명단에서 이름을 봤다. 얼굴이 잘 기억나지 않는다." |

---

## 관계가 메인 엔딩의 "결"을 바꿈

같은 메인 엔딩이라도 관계 상태에 따라 연출이 달라짐:

```
[메인 엔딩: 명문대 진학]

관계 풍부 → "합격 문자를 보자마자 지훈이한테 전화했다"
관계 빈약 → "합격 문자를 혼자 봤다. 기쁜데 전화할 사람이 없었다"

→ 같은 '성공'인데 행복 지수가 다름
```

---

## 인생 앨범 (수집 시스템)

엔딩 화면에서 획득한 카드를 수집. 리플레이 동기 부여.

| 카드 종류 | 예시 |
|-----------|------|
| 메인 엔딩 카드 | "SKY 입학" / "인디 뮤지션" / "창업 성공" |
| 관계 카드 | "7년 절친" / "첫사랑" / "라이벌에서 친구로" / "재회" |
| 히든 카드 | 특수 조건 달성 시 (예: 모든 NPC와 절친 유지) |

> "이번 플레이에서 못 본 관계 카드가 있네? 다음엔 수아 루트 가보자"
> → 수집 욕구 + "다른 선택을 했으면?" 호기심 = 리플레이 유도

---

## 기타 리플레이 요소

- 2회차 메타 요소 검토 (예: "평행 세계 모드")
