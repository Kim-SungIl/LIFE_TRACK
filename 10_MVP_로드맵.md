# 10. MVP 범위 / 확장 계획 / TODO

## MVP 범위

```
- 기간: 초등 6학년 → 고등 3학년 (7년)
- 학교: 일반고 + 특목고 1종
- 가정 타입: 강점 6종 중 2개 선택 (15가지 조합)
- NPC: 10명 (동시 활성 4~5명, 시기별 등장/퇴장)
- 엔딩: 메인 8~10개 + NPC별 사이드 에필로그
- 플랫폼: 모바일 (Unity or Godot)
- 연출: 고퀄리티 일러스트 + 텍스트 (애니메이션 최소화)
```
