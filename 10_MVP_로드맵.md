## MVP 범위

```
- 기간: 초등 6학년 → 고등 3학년 (7년, 336주)
- 학교: 일반고만 (특목고/특성화고는 확장판)
- 가정 타입: 강점 6종 중 2개 선택 (15가지 조합)
- NPC: 실제 등장 7명 (지훈·수빈·민재·유나·도윤·하은·준하). 초기 설계의 서아·시우·예린·루나는 미등장(오펀)
- 엔딩: 진로 기반 동적 생성(determineCareer) + 성취/행복 등급 수식 + NPC 근황 (07_엔딩 참조 — "메인 13개" 고정 테이블 아님)
- 온라인 시스템: `sns-activity` 단일 활동으로 단순화 (5채널 세부 구조는 미구현 — 08 참조)
- 플랫폼: 웹 게임 (브라우저 즉시 플레이, 설치 불필요)
- 연출: 고퀄리티 일러스트 + 텍스트 (애니메이션 최소화)
```
