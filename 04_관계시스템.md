# 04. 관계 시스템

## 핵심 원칙

> "관계는 유지하지 않으면 사라진다"
> "관계도 자원이다 — 새 친구를 사귀는 시간에 옛 친구는 멀어진다"

---

## NPC 구조: 총 10명, 동시 활성 4~5명

7년간 같은 5명이 함께하는 것이 아니라, **시기별로 등장/퇴장**하며 자연스럽게 교체됨.
NPC도 자기 인생을 삶 (플레이어가 신경 안 쓰면 NPC 상태가 변함)

### 주인공 성별과 NPC
- 주인공은 남/여 선택 가능
- NPC는 성별 무관하게 동일하게 존재
- **관계 유형(연애 가능 여부 등)만 성별에 따라 달라짐**
- NPC 콘텐츠 1세트로 개발량 관리

## 관계 유지 채널

학교가 달라져도 다른 채널로 관계를 유지할 수 있음. 단, 채널이 없으면 자연스럽게 멀어짐.

| 채널 | 조건 | 효과 |
|------|------|------|
| 같은 학교 | 같은 학교 진학 | 자동 유지 (매주 만남) |
| 같은 학원 | 학원 다니는 주 | 학원 활동 시 만남 |
| 같은 동네 | 동네가 같음 | "놀기" 선택 시 만남 가능 |
| 온라인 | 게임/SNS 활동 | 활동 시 유지 |
| 연락하기 | 시간 소비 (활동 1회분) | 의도적 관계 유지 |

## 관계 상태 (UI에 표시)

```
절친 → 친한 친구 → 보통 → 어색한 사이 → 연락 안 하는 사이
```

- 관계 유지 채널이 없으면 자동으로 한 단계씩 하락 (학기 단위)
- "연락 안 하는 사이"에서도 재회 이벤트로 관계 복구 가능

---

## 진학이 관계를 바꾸는 구조

진학 선택이 스탯뿐 아니라 NPC 관계에도 직접 영향:

```
[고등학교 선택]
├─ 일반고 A → 중학교 친구 2명과 같은 학교 (관계 유지 쉬움)
└─ 특목고   → 아는 사람 0명 (새로 시작, 새 기회)

→ "친구 따라 일반고" vs "꿈 위해 특목고" = 딜레마
```

## 관계 유형별 효과

| 유형 | 효과 |
|------|------|
| 친구 | 공동 활동 → 성장 보너스, 스트레스 감소 |
| 연애 | 안정 → 멘탈 ↑ / 갈등 → 집중력 ↓ |
| 라이벌 | 경쟁 → 성장 가속 / 패배 → 멘탈 ↓ |
| 멘토 | 정보 제공, 진로 조언 이벤트 |
| 온라인 | 비밀 공유, 이중생활 루트 연결 |

## 기타 원칙

- 단순 호감도 숫자가 아니라 **서사적 변화** ("절친이 소원해지는 과정")
- 관계를 통한 **정보 보완** (정보력 낮은 가정도 NPC를 통해 히든 선택지 접근 가능)
- "배신"이 아니라 **"관계 변화"** — 현실적 톤 유지

## 인기 환경 리셋 (진학 시) (v3)

> 진학은 관계의 단절만이 아니라, "새로운 시작"의 기회이기도 하다.
> 인기 스탯은 환경 변화에 따라 부분 리셋되며, 극단적 인기 값이 평균으로 회귀한다.

### 학년 변경 (같은 학교 내 진급)

| 상황 | 인기 변동 |
|------|-----------|
| 같은 학교 내 학년 변경 | 인기 **+3** |

> 새 학년, 새 반 배정으로 인한 소폭의 리셋 효과. 기존 평판이 유지되면서 약간의 상승.

### 학교 변경 (초→중, 중→고)

학교가 바뀌면 인기가 중앙값(45) 방향으로 회귀한다.

```
새 인기 = 기존 인기 + (45 - 기존 인기) × 0.2
```

| 기존 인기 | 새 인기 | 변동 | 설명 |
|-----------|---------|------|------|
| 80 | **73** | -7 | 인기 많던 아이도 새 학교에선 재증명 필요 |
| 60 | **57** | -3 | 약간의 하락 |
| 45 | **45** | ±0 | 중앙값, 변동 없음 |
| 30 | **33** | +3 | 낮은 인기도 새 환경에서 리셋 기회 |
| 10 | **17** | +7 | 왕따/아싸도 새 학교에서 재출발 가능 |
| 5 | **13** | +8 | 극단적으로 낮은 인기의 구제 |

> **설계 의도:** 인기가 극단적으로 높거나 낮은 플레이어 모두에게 전학/진학이 의미 있는 변수가 된다.
> 높은 인기는 "다시 증명해야" 하고, 낮은 인기는 "기회를 얻는다."

### 새 NPC 초기 친밀도 (진학 시)

진학으로 새로 만나는 NPC의 초기 친밀도는 인기에 영향을 받는다.

```
새 NPC 초기 친밀도 = 30 + (인기 × 0.2)
```

| 인기 | 새 NPC 초기 친밀도 | 관계 단계 |
|------|---------------------|-----------|
| 80 | 46 | 보통 |
| 60 | 42 | 보통 |
| 40 | 38 | 어색한 사이 |
| 20 | 34 | 어색한 사이 |

> 인기가 높으면 첫인상이 좋아 관계 시작점이 유리하다.

### 기존 NPC 처리

| 상황 | 친밀도 변동 |
|------|-------------|
| 같은 학교 유지 | 변동 없음 (기존 친밀도 그대로) |
| 다른 학교로 이동 | 기존 친밀도 × **0.8** (-20%) |

> 다른 학교로 간 기존 NPC는 친밀도가 20% 감소한다.
> 이는 "같은 학교 선택"에 전략적 가치를 부여하는 04_관계시스템의 핵심 메커니즘과 일치한다.
> (상세: "진학이 관계를 바꾸는 구조" 섹션 참조)

## 친밀도 수치 체계

> 이 섹션은 02_스탯시스템.md의 내부 세부 스탯 "NPC 호감도 (0~100)"을 구체화한다.
> 친밀도는 인기 스탯의 하위 요소이며, 전체 NPC 친밀도의 가중 평균이 인기 스탯에 반영된다.

### 친밀도 범위와 관계 단계 매핑

| 친밀도 구간 | 관계 단계 | UI 표시 | 설명 |
|-------------|-----------|---------|------|
| 0 ~ 19 | 연락 안 하는 사이 | 회색 (비활성) | 이름만 아는 상태. 대화 선택지 제한 |

| 40 ~ 59 | 보통 | 노랑 | 학교/학원에서 자연스럽게 어울림 |
| 60 ~ 79 | 친한 친구 | 초록 | 시스템 효과 개방 (NPC별 고유 보너스 활성) |
| 80 ~ 100 | 절친 | 금색 | 고유 이벤트·엔딩 루트 해금, 최대 보너스 |

> UI 등급 색상은 02_스탯시스템.md의 스탯 등급 표시(E~A)와 일치시켜 시각적 통일감을 유지한다.

### 친밀도 변동 — 상승

| 행동 | 친밀도 상승 | 슬롯 비용 | 조건 |
|------|-------------|-----------|------|
| 같은 학교 자동 접촉 | +1/주 | 0 | 같은 학교 재학 중 (자동) |
| 같은 학원 자동 접촉 | +1/주 | 0 | 같은 학원 수강 중 (자동) |
| 연락하기 (대상 지정) | +3 | 1슬롯 | 언제든 가능 |
| 같이 놀기 (대상 지정) | +5 | 1슬롯 | 같은 동네 or 같은 학교 |
| 선물 주기 | +4 | 0 (용돈 소비) | 용돈 -2만원 |
| 고민 상담 | +6 | 1슬롯 | 친밀도 40 이상에서만 가능 |
| 같이 공부하기 | +3 | 1슬롯 | 같은 학교/학원, 학업도 +1 |
| SNS 반응 (좋아요/댓글) | +1 | 0 (온라인 활동 시 부가) | 온라인 활동 선택 시 |
| 핵심 이벤트 올바른 선택 | +8 ~ +15 | - | NPC별 고유 이벤트 |

> **상승의 핵심 제약:** 대부분의 의미 있는 친밀도 상승(+3 이상)은 슬롯 1개를 소비한다.
> 주 5슬롯 중 관계에 쓸 수 있는 건 현실적으로 1~2개. 나머지는 학업/운동/휴식에 써야 하기 때문.

### 친밀도 변동 — 자연 감소 (접촉 없을 시)

관계 유지 채널이 **하나도 없는** 상태에서의 주당 자연 감소:

| 감소 조건 | 주당 감소량 | 설명 |
|-----------|-------------|------|
| 채널 0개 (완전 단절) | -3/주 | 학교·학원·동네 모두 다름, 연락도 안 함 |
| 채널 1개 (온라인만) | -1/주 | SNS로 간접 접촉은 하지만 직접 만남 없음 |
| 채널 1개 (같은 동네만) | -1/주 | 가끔 마주치지만 의도적 만남 없음 |
| 채널 2개 이상 | 0/주 | 자연스러운 접촉으로 유지 (성장은 안 함) |

> **감소 속도의 설계 의도:**
> - 완전 단절 시 -3/주 → "보통(40)" 관계가 약 7주(약 2개월)만에 "어색한 사이(20)"로 하락
> - 한 학기(약 24주) 방치하면 어떤 관계든 "연락 안 하는 사이(0~19)"에 도달
> - 이 속도는 현실의 "한 학기 안 보면 어색해지는" 체감과 일치
> - 단, 채널이 1개라도 있으면 -1/주 → 약 20주(5개월)를 버틸 수 있어 "느슨하지만 끊기지 않는" 관계 표현

### 관계 단계 전환 시 특수 효과

| 전환 | 효과 |
|------|------|
| 보통 → 친한 친구 (40→60 돌파) | 해당 NPC의 [게임 시스템 연동] 효과 활성화. 축하 컷신 |
| 친한 친구 → 절친 (60→80 돌파) | 전용 이벤트 체인·엔딩 루트 해금. 특수 컷신 |
| 친한 친구 → 보통 (60→40 이하 하락) | "요즘 좀 서먹하다..." 컷신. 시스템 보너스 비활성화 |
| 보통 → 어색한 사이 (40→20 이하 하락) | "카톡 대화창이 한참 전에 멈춰 있다" 연출. NPC가 다른 친구와 어울리는 장면 |
| 어색한 사이 → 연락 안 하는 사이 (20→0 이하 하락) | NPC 비활성화. UI에서 회색 처리. 재회 이벤트로만 복구 가능 |

## 관계 유지 비용 테이블

> 핵심 원칙: "관계는 무료가 아니다." 모든 관계 유지에는 슬롯, 피로, 용돈 중 최소 하나의 비용이 든다.
> 유일한 예외는 "같은 학교"와 "같은 학원"의 자동 접촉으로, 이는 비용 없이 감소만 방지한다.

| 채널/행동 | 슬롯 비용 | 피로 비용 | 용돈 비용 | 친밀도 효과 | 비고 |
|-----------|-----------|-----------|-----------|-------------|------|
| 같은 학교 (자동) | 0 | 0 | 0 | +1/주 (유지) | 비용 없음. 같은 학교인 한 자동 적용. 성장은 느림 |
| 같은 학원 (자동) | 0 | 0 | 학원비 별도 | +1/주 (유지) | 학원 등록 자체에 용돈 -3/주. 관계 유지는 부산물 |
| 같은 동네 (놀기) | 1슬롯 | +3 | -1만원 | +5 | 02_활동테이블의 "친구와 놀기"와 동일. 대상 NPC 지정 |
| 온라인 (SNS/게임) | 1슬롯 | +2 | 0 | +2 (대상 지정 시) | 02_활동테이블의 "SNS 활동". 루나(#10) 전용 채널이기도 함 |

### 비용 밸런스 분석

```
[관계 1명 유지 — "친한 친구(60)" 수준을 유지하려면?]

시나리오 A: 같은 학교 + 격주 1회 "같이 놀기"
→ 자동 +1/주 + 격주 +5 = 주 평균 +3.5
→ 자연 감소 없음 (채널 2개)
→ 비용: 격주 1슬롯 + 피로 +3 + 용돈 -1만원
→ ✅ 가장 저렴. 같은 학교 NPC는 유지가 쉽다.

시나리오 B: 학교 다름 + 같은 동네 + 격주 1회 "연락하기"
→ 같은 동네 채널 유지 (-1/주 감소)
→ 격주 연락 +3 = 주 평균 +1.5
→ 순 변동: +1.5 - 1 = +0.5/주 (겨우 유지)
→ 비용: 격주 1슬롯 + 피로 +1
→ ⚠️ 유지는 되지만 성장 불가. 놀기를 추가해야 60 이상으로 올림.

시나리오 C: 학교·동네·학원 모두 다름 (온라인만)
→ 감소 -1/주 (온라인 채널만)
→ 매주 "연락하기" +3 = 순 변동 +2/주
→ 비용: 매주 1슬롯 + 피로 +1
→ ⚠️ 유지 가능하지만, 매주 1슬롯을 이 NPC에 고정해야 함.

시나리오 D: 채널 0개 (완전 단절)
→ 감소 -3/주
→ 매주 "연락하기" +3 = 순 변동 0/주 (현상 유지)
→ 성장시키려면 매주 "연락하기" + 격주 "선물" 필요 → 주당 1슬롯 + 격주 2만원
→ ❌ 비용 최대. "멀어진 친구를 붙잡는 건 비싸다."
```

> **핵심 교훈:** 같은 학교/학원이라는 "자동 채널"의 가치가 매우 크다.
> 진학 시 친구와 같은 학교를 선택하면 관계 유지 비용이 극적으로 줄어든다.
> 이것이 "친구 따라 일반고" 선택지에 실질적 가치를 부여하는 시스템적 근거다.

## 관계가 스탯에 미치는 영향

> 각 NPC와의 관계 단계에 따라 스탯에 지속적 보정이 적용된다.
> 이 보정은 [05_NPC설계.md]의 [게임 시스템 연동]을 수치화한 것이다.
> 보정은 **주 단위**로 자동 적용되며, 관계 단계가 바뀌면 즉시 반영된다.

### NPC별 관계 보정 테이블

#### #1 한지훈 (소꿉친구 — 감정적 앵커)

| 관계 단계 | 보정 효과 (주당) | 설명 |
|-----------|------------------|------|
| 보통 (40~59) | 멘탈 +1 | 가끔 마주치는 것만으로도 안심 |
| 친한 친구 (60~79) | 멘탈 +2, 피로 회복 +1 | 함께하는 시간이 쉬는 시간 |
| 절친 (80~100) | 멘탈 +3, 피로 회복 +2 | 위기 시 안전망 이벤트 자동 발생 (멘탈 15 이하 시 멘탈 +10) |
| 어색한 사이 이하 | 멘탈 -1 | "예전엔 친했는데..." 상실감 |

> 지훈과의 활동은 용돈이 들지 않는다 (편의점, 산책, 게임). 대신 학업/재능 성장 보너스가 없다.
> "편하지만 발전 없는 관계"의 트레이드오프.

#### #2 오수빈 (소셜 브릿지)

> 수빈은 직접적인 스탯 성장 보너스가 약하지만, 다른 NPC 관계의 "부스터" 역할.

|-----------|-----------|------|
| 친한 친구 (60~79) | 멘탈 +1/주. NPC 갈등 시 화해 선택지 힌트 제공 | 제3자 시점의 조언 |
| 절친 (80~100) | 멘탈 +2/주. 온라인 활동 시 멘탈 회복 +3 추가. 비밀 공유 이벤트 해금 | 온라인에서만 가능한 깊은 관계 |

> 루나와의 관계 유지에는 "온라인 활동" 슬롯이 필요. 고등학교 갈수록 이 슬롯이 사치가 된다.

### 스탯 보정 요약 — 최대 효과 비교

| NPC | 최대 학업 보정 | 최대 멘탈 보정 | 최대 기타 보정 | 관계 비용 |
|-----|----------------|----------------|----------------|-----------|
| #1 지훈 | 없음 | +3/주 | 피로 회복 +2 | 낮음 (용돈 0) |
| #2 수빈 | 없음 | 없음 | 인기 +2/주, NPC 초기 친밀도 +15 | 중간 |
| #3 도윤 | 없음 | 없음 | 체력 성장 ×1.3, 인기 +1/주 | 중간 (재회 필요) |
| #4 서아 | 없음 | +2/주 (or +3/-2 연애) | 재능(창작) +1/주 | 중간 |
| #5 민재 | 성장 ×1.3 | **-2/주** | 시험 정보 이벤트 | 높음 (멘탈 대가) |
| #6 하은 | 활동 효율 +10% | +2/주 | 진로 조기 해금 | 낮음 (시간 제한) |
| #7 시우 | 없음 | +2/주 | 적응 멘탈 보호, 진로 시너지 | 중간 |
| #8 예린 | 선택지 효율 +15% | 없음 | 히든 정보 3회/학기 | 높음 (외부 제약) |
| #9 준하 | 없음 | +2/주 | 인기 +3 (정보 전달 시) | 높음 (늦은 등장) |
| #10 루나 | 없음 | +2/주 | 갈등 해결 힌트, 온라인 멘탈 +3 | 중간 (온라인 슬롯) |

> **설계 의도:**
> - 학업 직접 보정은 민재(#5)와 예린(#8)만 제공. 둘 다 높은 대가가 있다.
> - 멘탈 보정은 대부분의 NPC가 제공하지만, 동시에 여러 NPC의 보정을 받기는 어렵다 (슬롯 제한).
> - "모든 NPC와 절친 = 모든 보정 획득"은 구조적으로 불가능. 아래 "동시 유지 제한" 섹션에서 증명.

## "동시 유지 제한" 검증

> 핵심 질문: "모든 NPC와 친하게 지낼 수 있는가?"
> 답: **구조적으로 불가능하다.** 아래에서 수학적으로 증명한다.

### 전제 조건 (고1 기준 — 가장 많은 NPC가 활성화되는 시기)

- **주당 총 슬롯**: 5
- **강제/반강제 슬롯**: 2 (02_스탯시스템.md 고1 기준)
- **자유 슬롯**: 3
- **주당 필수 배분**:
  - 학업 활동 최소 1슬롯 (안 하면 학업 -0.5/주 감소 + 부모 압박)
  - 휴식 최소 격주 1슬롯 (안 하면 피로 누적 → 4~5주 내 과로 상태)
- **관계에 쓸 수 있는 실질 슬롯: 주당 1~2개**

### 시나리오 1: 절친 5명 동시 유지 시도

"친한 친구(60~79)" 이상을 5명 동시에 유지한다고 가정.

```
[활성 NPC 5명의 관계 유지 필요량 — 최소 시나리오]

NPC A: 같은 학교 → 자동 +1/주, 감소 없음 → 유지만 하려면 비용 0
NPC B: 같은 학교 → 자동 +1/주, 감소 없음 → 유지만 하려면 비용 0
NPC C: 같은 동네만 → 감소 -1/주 → 최소 격주 1회 "연락하기" 필요 (0.5슬롯/주)
NPC D: 채널 없음 → 감소 -3/주 → 매주 1회 "연락하기"(+3) 필수 (1슬롯/주)
NPC E: 온라인만 → 감소 -1/주 → 최소 격주 1회 온라인 활동 (0.5슬롯/주)

총 관계 유지 비용: 0 + 0 + 0.5 + 1 + 0.5 = 2슬롯/주

남은 자유 슬롯: 3 - 2 = 1슬롯/주
→ 이 1슬롯으로 학업 + 휴식 + 운동 + 재능을 모두 커버해야 함
→ ❌ 불가능. 학업만 해도 주 1슬롯 필수, 휴식도 격주 1슬롯 필수.
```

**피로 시뮬레이션 (5명 유지 + 학업 최소):**

```
주당 활동 배치:
  강제 슬롯 ×2: 학원 ×2 (피로 +8 ×2 = +16)
  자유 슬롯 1: 연락하기-NPC D (피로 +1)
  자유 슬롯 2: 연락하기-NPC C or 온라인-NPC E (피로 +1~+2) [격주 교대]
  자유 슬롯 3: 독학 (피로 +6) [학업 유지 최소치]

  주당 총 피로: +16 + 1 + 1.5 + 6 = +24.5
  자연 회복: -5
  순 피로 증가: +19.5/주

→ 4주차: 피로 78 (위험 구간)
→ 5주차: 피로 97 (한계 → 강제 휴식)
→ 결론: 5주마다 강제 휴식으로 1~2주를 잃음
→ 그 기간 동안 관계 유지 못해 NPC D 친밀도 -6~-9 하락
→ ❌ 사이클이 깨짐. 유지 불가능.
```

### 시나리오 2: 친한 친구 3명 + 보통 2명 (현실적 한계)

```
[활성 NPC 3명 "친한 친구" + 2명 "보통" 유지]

NPC A: 같은 학교 (친한 친구) → 비용 0, 격주 "같이 놀기"로 성장 (0.5슬롯/주)
NPC B: 같은 학교 (친한 친구) → 비용 0, 격주 "같이 놀기"로 성장 (0.5슬롯/주)
NPC C: 같은 동네 (친한 친구) → 격주 "연락하기"로 유지 (0.5슬롯/주)
NPC D: 온라인 (보통) → 격주 온라인 활동으로 현상 유지 (0.5슬롯/주)
NPC E: 채널 없음 (보통) → 자연 감소 -3/주 → 감소 수용 (비용 0, 서서히 멀어짐)

총 관계 비용: 0.5 + 0.5 + 0.5 + 0.5 + 0 = 2슬롯/주 (격주 배분)

실제 주간 스케줄 (격주 패턴):
  [A주] 강제 ×2 + 학업 ×1 + 놀기(NPC A) ×1 + 휴식 ×1
  [B주] 강제 ×2 + 학업 ×1 + 연락(NPC C) ×1 + 온라인(NPC D) ×1

피로 시뮬레이션:
  A주: +16(강제) + 6(학업) + 3(놀기) - 12(휴식) = +13, 자연회복 -5 = +8
  B주: +16(강제) + 6(학업) + 1(연락) + 2(온라인) = +25, 자연회복 -5 = +20
  2주 평균: +14/주

→ 피로가 약 3.5주(7주 주기)마다 과로(50) 진입
→ 격주로 휴식이 있어 완전한 파탄은 아니지만, 여유는 없음
→ NPC E는 매 학기 자연 하락 → 3~4개월 후 "어색한 사이"로
→ ⚠️ 가능하지만 빡빡함. 운동/재능 투자 여유 없음.
```

### 시나리오 3: 핵심 관계 2명 집중 + 나머지 방치 (최적 효율)

```
[NPC 2명 "절친" 집중 + 나머지 자연 추이]

NPC A: 같은 학교 (절친 유지) → 격주 "같이 놀기" (0.5슬롯/주)
NPC B: 같은 학교 (절친 유지) → 격주 "같이 놀기" (0.5슬롯/주)
나머지: 같은 학교 NPC만 자동 +1/주로 "보통" 유지, 나머지 자연 감소

주간 스케줄:
  [A주] 강제 ×2 + 학업 ×1 + 놀기(NPC A) ×1 + 휴식 ×1
  [B주] 강제 ×2 + 학업 ×1 + 놀기(NPC B) ×1 + 운동 or 재능 ×1

피로 시뮬레이션:
  A주: +16 + 6 + 3 - 12 = +13, -5 = +8
  B주: +16 + 6 + 3 + 2~7 = +27~32, -5 = +22~27
  2주 평균: +15~17.5/주 → 격주 휴식으로 통제 가능

→ ✅ 관계 2명 절친 + 학업 + 운동/재능 밸런스 가능
→ 대신 3번째 이상의 NPC는 "보통" 이하로 떨어짐
→ 이것이 "4~5명 동시 활성이지만 절친은 2~3명이 한계"인 구조적 이유
```

### 결론: 동시 유지 한계

| 유지 목표 | 가능 여부 | 조건 | 대가 |
|-----------|-----------|------|------|
| 절친 5명 | ❌ 불가능 | — | — |
| 절친 4명 | ❌ 사실상 불가능 | 모든 NPC가 같은 학교 + 휴식 포기 | 6주 내 번아웃 |
| 친한 친구 5명 | ❌ 극도로 어려움 | 채널 최적화 + 학업/휴식 최소화 | 피로 관리 파탄, 학업 하락 |
| 친한 친구 4명 | ⚠️ 가능하지만 빡빡 | 최소 3명이 같은 학교/학원 | 운동/재능 투자 불가 |
| 절친 2~3명 + 보통 2명 | ✅ 현실적 최적해 | 핵심 관계 선택 + 나머지 자연 추이 | 일부 NPC는 멀어짐을 수용 |
| 절친 1명 + 친한 친구 2명 | ✅ 여유로움 | 학업/재능에 투자 여유 | 관계 보정이 적음 |

> **이것이 04_관계시스템의 핵심 메시지:**
> "모든 친구와 친하게 지낼 수는 없다. 누군가를 선택하면 누군가는 멀어진다."
>
> **수치적 근거:**
> - 주 5슬롯 중 관계에 쓸 수 있는 실질 슬롯은 1~2개 (고등학교 기준)
> - 채널이 없는 NPC 1명 유지에 최소 1슬롯/주 필요
> - 따라서 "적극적 유지"가 가능한 NPC는 최대 2명
> - 나머지는 "같은 학교" 등 자동 채널에 의존하거나 자연 감소를 수용해야 함
> - 이 구조가 진학 선택(같은 학교 진학 = 관계 자동 유지)에 전략적 무게를 부여한다
