# 06. 이벤트 설계

## 설계 철학

> "이벤트는 시스템의 결과를 감정으로 번역하는 장치다."

- 이벤트는 **스탯 변동보다 감정적 체감**이 먼저다
- 모든 딜레마는 "정답이 없는 선택"이어야 한다 — 뭘 골라도 무언가를 잃는다
- 이벤트 밀도는 낮을수록 좋다 — 아무 일 없는 주가 있어야 이벤트가 터졌을 때 강하다
- 정보는 천천히 열린다 — 초반에 시스템을 다 보여주지 않는다

---

## 1. 첫 10분 설계 — "이건 내 얘기다"

> 첫 10분의 목표: ① 감정 이입 ② "선택 → 후회" 구조 체감 ③ 3명의 NPC에 마음이 움직임

### 1-1. 부모 선택 (0~2분): 생활 장면으로

부모 강점 2개를 선택하는 과정을 **숫자 나열이 아닌 짧은 생활 장면**으로 연출한다.
플레이어는 3~4개의 장면을 보고, 끌리는 가정을 고른다.

```
[장면 A — 재력]
초등학교 하교길. 학원 차가 교문 앞에 와 있다.
엄마에게서 카톡이 온다: "오늘 영어학원 끝나면 수학학원 가야 해. 간식비 넣어뒀어."
내 방 책상 위엔 문제집이 쌓여 있다. 대신 용돈은 넉넉하다.
→ "불편한 건 없었다. 이상하게, 그것도 좀 이상했다."

[장면 B — 정서적 지지]
저녁 식탁. 아빠가 국을 떠주며 묻는다: "오늘 학교 어땠어?"
시험을 망쳤다고 하자 엄마가 웃으며 말한다: "다음에 잘하면 되지 뭐."
냉장고엔 간식이 별로 없지만, 집에 오면 항상 누군가 있다.
→ "좋았지만, 괜히 눈물이 날 것 같았다."

[장면 C — 시간/자유]
집에 왔는데 아무도 없다. 냉장고에 메모: "밥 시켜 먹어. 만 원 식탁 위."
TV 리모컨을 잡을지, 숙제를 할지, 밖에 나갈지. 전부 내 맘이다.
아무도 뭐라 안 한다. 아무도 도와주지도 않는다.
→ "편했다. 근데 집이 너무 조용했다."

[장면 D — 엄격함]
현관문이 열리자마자 엄마 목소리: "손 씻고 바로 숙제."
시험 성적표를 냉장고에 붙이고, 목표 점수에 형광펜이 쳐져 있다.
잔소리가 많지만, 아침마다 도시락은 꼭 싸준다.
→ "짜증이 났다. 근데 도시락은... 맛있었다."

[장면 E — 유전자]
체육 시간. 별로 연습한 적 없는데 달리기가 빠르다.
선생님이 말한다: "너 운동 좀 해봤어?"
엄마는 옛날에 학교 대표였다고 한다. 공부는... 알아서 해야 한다.
→ "잘하는 건 좋은데, 왜 그게 내 노력이 아닌 것 같을까."

[장면 F — 정보력]
엄마가 전화 중이다: "그 학원이 제일 낫대. 내가 알아봤는데..."
나는 뭘 정했는지도 모르는데, 이미 학원이 바뀌어 있다.
친구들은 모르는 대회에 내 이름이 올라가 있다.
→ "고마운 건지, 무서운 건지 잘 모르겠다."
```

- 플레이어는 장면 중 **2개를 선택**한다
- 선택 후: "당신의 가정은 이런 곳입니다" → 선택한 2개 장면이 합쳐진 짧은 내레이션
- **어떤 장면을 선택하지 않았는지**도 은근히 보여준다 ("그 집엔 없었던 것들")

### 1-2. 첫 만남 (2~5분): NPC 3명만

개학 첫날. 10명의 NPC를 다 보여주지 않는다. 딱 3명만.

```
[교실 — 한지훈]
짝꿍이 된 애가 말을 건다.
"야, 너 3반이었지? 나 5반이었는데. 아파트 몇 동이야?"
    → 같은 동 같은 라인이란 걸 알게 됨
    → "야 그러면 우리 내려갈 때 같이 내려가자"
    → 특별한 건 없다. 그냥 편하다.

[복도 — 윤서아]
급식 줄에서 뒤에 서 있는 애가 교과서 뒷면에 뭔가를 그리고 있다.
눈이 마주치면 먼저 시선을 돌린다. 살짝 웃은 것 같기도 하고 아닌 것 같기도 하다.
    → 이름도 모른다. 그런데 왠지 신경이 쓰인다.
    → 나중에 보니 교과서에 작은 고양이 낙서가 있었다.
    → (서아의 첫 대사는 2주차에 등장 — "아, 그거 수업 지루해서...")

[체육 시간 — 강민재]
축구/피구에서 눈에 띄는 애가 있다. 뭘 해도 잘한다.
나와 같은 조가 되자 말한다: "집중 좀 해. 지는 거 싫거든."
    → 약간 불편하다. 하지만 인정할 수밖에 없다.
```

**원칙:**
- 지훈 = 편안함 (무조건적 수용)
- 서아 = 호기심 (말도 안 해봤는데 궁금함)
- 민재 = 긴장감 (인정하기 싫은 비교 대상)

나머지 NPC(수빈, 하은, 도윤 등)는 이름만 스쳐 지나가거나, 2~4주차에 점진적으로 등장.

### 1-2b. 서아 카메오 이벤트 (초등, 남주 강화)

서아는 중학교에서 본격 등장하지만, 초등 때 **대화 없이 눈에 밟히는 존재**로 3회 등장한다.
이름도 모르고, 말도 안 해봤지만, 왠지 기억에 남는다.

```
[카메오 1 — 3~4주차]
  하교길 횡단보도. 건너편에서 교과서에 뭔가를 그리고 있는 애가 보인다.
  → (첫 10분에서 급식 줄에서 본 그 애다)
  → 시스템 메시지: "어디선가 본 것 같은 얼굴이다."
  → 신호가 바뀌고, 걔는 반대쪽으로 걸어갔다.

[카메오 2 — 5월 체육대회]
  다른 반 응원석에서 혼자 책을 읽고 있는 애가 보인다.
  → 시끌벅적한 운동장에서 혼자 조용한 게 눈에 띈다.
  → 시스템 메시지: "그 애, 또 있네."

[카메오 3 — 겨울방학 직전]
  문방구 앞. 지훈이랑 같이 있는데, 건너편에 걔가 지나간다.
  → 눈이 마주칠 뻔했다. 아닌 것 같기도 하다.
  → 시스템 메시지: "이름이 뭘까."
```

> **효과**: 중학교 입학식에서 같은 반이 되었을 때, "아, 그때 그 애!"라는 인식이 생기면서 서아의 등장 임팩트가 강화된다. 대화가 없으므로 서아의 "첫 만남" 설정은 훼손되지 않는다.

### 1-2c. 수빈 초등 씨앗 이벤트 (남주 한정)

남주 플레이 시 수빈과의 초등 이벤트에 **미묘한 감정의 씨앗**이 추가된다.
연애가 아님 — 본인도 뭔지 모르는 "괜히 신경 쓰임" 수준.

```
[학원 끝나고 — 4~5주차]
  수빈이랑 떡볶이 먹고 각자 집에 갈 때.
  수빈: "야 너 집 어디야? 같은 방향 아니야?"
  같은 동네라는 걸 알게 됨. 걸어가면서 대화가 이어진다.
  → 집 앞에서 "그럼 나 여기야. 내일 학원에서 봐~"
  → 나레이션 (남주만): "집에 들어왔는데, 왜 웃고 있었는지 모르겠다."

[학원 쉬는 시간 — 6~8주차]
  수빈이 "야 이거 먹어볼래?" 하고 과자를 나눠줌.
  다른 애들에게도 나눠주는 거지만, 나한테는 좀 더 큰 조각을 준 것 같기도 하고.
  → 나레이션 (남주만): "착각이겠지."
```

> **원칙**: 이 씨앗 이벤트는 남주 한정으로, 여주에게는 발생하지 않는다. 수빈의 행동 자체는 동일하지만, 남주의 내면 나레이션이 다르다.

### 1-3. 첫 주말 선택 → 첫 후회 (5~10분)

첫 주말. 처음으로 2슬롯을 직접 고른다.

```
[첫 번째 주말]

토요일: 뭘 하고 싶어?
  ① 지훈이랑 놀기 — 동네 편의점 앞에서 아이스크림
  ② 공부하기 — 첫 시험이 다가온다
  ③ 탐색 — 동네를 돌아다녀본다

일요일: 남은 하루.
  (토요일 선택에 따라 선택지가 미세하게 달라짐)
```

**그리고 선택 직후, 작은 후회를 준다. 후회는 "놓침"을 넘어 관계 온도에 흔적을 남긴다:**

```
[지훈이랑 놀기를 선택한 경우]
  → 월요일 아침, 서아가 복도에서 뭔가를 떨어뜨렸다.
  → 줍다가 잠깐 눈이 마주쳤는데, 서아가 "고마워" 한마디 하고 갔다.
  → (이건 토요일에 '탐색'을 골랐으면 이어지는 대화였다 — 서아 첫 접촉이 1주 늦어짐)
  → 서아 초기 친밀도 보정: 탐색 선택 시 +2 / 미선택 시 +0 (작지만 쌓이는 차이)
  → 나레이션: "그 주말, 나는 편한 쪽을 골랐다. 아직은 그게 뭘 의미하는지 몰랐다."

[공부하기를 선택한 경우]
  → 월요일 아침, 지훈이가 말한다: "야 어제 PC방 갔는데 너도 올걸 ㅋㅋ"
  → 카톡 단톡방에 지훈이가 올린 사진이 있다. 나만 없다.
  → 다음 주, 지훈이 장난이 살짝 줄었다. 대신 민재가 "너 시험 준비해?" 하고 먼저 말을 건다.
  → 지훈 친밀도 변화 없음 (지훈은 익숙하니까). 하지만 민재와의 첫 대화가 일찍 열림.
  → 나레이션: "조용한 일요일이었다. 그게 맞는 선택이었을까."

[탐색을 선택한 경우]
  → 동네를 돌아다니다가 문방구 앞에서 서아를 마주쳤다.
  → "어? 같은 반이지? 너도 여기 사는구나."
  → 짧은 대화. 서아의 표정이 교실에서보다 밝다. 서아 친밀도 +2
  → 하지만 월요일, 지훈: "야 어제 어디 갔어? 불렀는데 없더라." (장난이지만 살짝 서운함)
  → 나레이션: "새로운 길을 택했다. 익숙한 쪽이 조금 멀어졌다."
```

> **핵심: 뭘 골라도 약간의 아쉬움이 남는다. 그리고 그 아쉬움은 관계 온도에 작은 흔적을 남긴다. 이것이 이 게임의 본질이라는 걸 첫 10분에 각인시킨다.**

### 1-4. 정보 공개 순서 (첫 1시간)

초반에 보여주는 것과 나중에 여는 것을 분리한다.

| 시점 | 공개되는 정보 |
|------|-------------|
| 게임 시작 | 학업, 멘탈, 피로 (3개 스탯만), 용돈, 관계 (NPC 이름만) |
| 2주차 | 체력, 인기 공개 ("체육 시간에 힘들었다" → 체력 스탯 등장) |
| 4주차 | 루틴 시스템 해금 ("매주 같은 걸 하니까 좀 익숙해졌다" → 루틴 보너스 설명) |
| 첫 시험 후 | 등급 시스템 공개 (E~A), 교내 순위 처음 확인 |
| 재능 활동 3회 반복 | 하위 스킬 공개 ("너 음악 감각 있는 것 같아" → 음악 하위 스킬 등장) |
| 첫 방학 | 방학 목표 시스템, 묶음 활동 해금 |
| 중학교 진학 | SNS 시스템 해금 ("친구가 SNS 계정 만들었대" → 팔로우 시작) |
| 첫 갈등/사건 이후 | 평판 태그 공개 (사건을 통해 "넌 이런 애구나" 인식이 형성됨) |

> 원칙: "UI는 단순, 내부는 복잡"의 실전 적용. 시스템은 처음부터 돌아가지만, **보여주는 건 체감한 뒤에.**

## 2. 체크포인트 이벤트 타임라인

7년간 반드시 발생하는 **고정 이벤트**. 게임의 뼈대이자 감정의 봉우리.

### 2-1. 연간 반복 이벤트 (매년)

| 시기 | 이벤트 | 연출 |
|------|--------|------|
| 3월 1주 | **개학** | 반배정, 새 NPC 만남 / 기존 NPC 재회 어색함 |
| 4월 말 | **중간고사** | 시험 결과에 따른 NPC 반응 + 부모 반응 |
| 5월 | **체육대회/축제** | 인기 + 재능 이벤트 집중 구간, NPC 특별 이벤트 |
| 5월 | **어버이날** | 부모 관계 이벤트 (가정 상황에 따라 연출 다름) |
| 7월 초 | **기말고사** | 1학기 성적 결산 |
| 7월 중순 | **여름방학 시작** | 해방감 연출, 방학 목표 설정 |
| 8월 말 | **개학** | 방학 동안 변한 관계 확인, "개학 어색함" 이벤트 |
| 10월 | **수학여행** | 관계 집중 이벤트 (룸메이트 선택, 자유시간 조 편성) |
| 10월 중순 | **중간고사 2** | |

| 겨울방학 | **진학 분기점** (해당 학년) | 초→중, 중→고 결정. 게임 최대 분기점 |
| 2월 | **졸업식** (해당 학년) | NPC 이별/재회 확정 |

### 2-2. 학년별 고유 이벤트

#### 초등 6학년 (Y1) — "마지막 초등학생"

| 시기 | 이벤트 | 선택/효과 |
|------|--------|----------|
| 3월 | 첫 10분 (1-1~1-3) | 부모 선택, NPC 3명 만남, 첫 주말 |
| 5월 | **졸업 앨범 사진** | "누구 옆에 서고 싶어?" → NPC 관계 표현 |
| 10월 | **담임 면담** | 중학교 진학 이야기. 부모 기대 첫 등장 |
| 겨울방학 | **중학교 배정 발표** | 같이 가는 친구 / 헤어지는 친구 목록 표시 |
| 2월 | **초등 졸업식** | "6년간의 학교가 끝난다" — NPC별 작별 장면 |

#### 중학교 (Y2~Y4) — "세상이 넓어지는 시기"

| 시기 | 이벤트 | 선택/효과 |
|------|--------|----------|
| Y2 3월 | **중학교 첫날** | 새 교복, 새 환경. NPC #4(서아) 본격 등장, NPC #5(민재) 등장 |

## 3. 딜레마 이벤트 20개

> 원칙: 모든 딜레마는 **"정답 없는 선택"**이다. 어떤 걸 골라도 무언가를 잃고, 무언가를 얻는다.
> 딜레마는 플레이어의 스탯, 관계, 부모 상태에 따라 **조건부로 발생**한다.

### 🔵 초등학교 (Y1)

**딜레마 #1 — "같이 놀자" vs "나 학원 가야 해"**
- **상황**: 하교길. 지훈이가 "야 오늘 PC방 갈래?" 하는데, 학원 시간이다.
- **선택지**:
  - ① 학원 간다 → 학업 루틴 유지, 지훈 친밀도 변화 없음 (지훈은 익숙함)
  - ② 학원 빼고 논다 → 지훈 친밀도 +3, 피로 -5. 단, 부모(엄격함) 있으면 들킬 확률 → 용돈 삭감
- **첫 후회**: ①을 고르면 "PC방 사진이 단톡방에 올라옴. 나만 없다." ②를 고르면 "다음 날 학원에서 진도를 놓쳤다."
- **발생 조건**: Y1 4~6주차 (루틴에 학원이 있을 때)

**딜레마 #2 — "떨어뜨린 지우개"**
- **상황**: 서아가 교실에서 뭔가를 떨어뜨렸다. 줍는 건 간단한데, 주변에 민재가 보고 있다.
- **선택지**:
  - ① 줍어준다 → 서아 친밀도 +2, 첫 대화 발생 ("고마워... 너 몇 반이야?")
  - ② 모른 척한다 → 변화 없음. 하지만 나중에 "그때 줍어줬으면 달랐을까" 내레이션
- **숨은 효과**: ①을 고르면 민재가 "쟤 관심 있나 봐 ㅋ" → 인기(존재감) +1
- **발생 조건**: Y1 2~3주차 (서아 첫 이벤트)

**딜레마 #3 — "체육대회 조 편성"**
- **상황**: 체육대회 때 반 대항전. 조를 짤 수 있는데, 지훈(운동 못함)과 민재(에이스)가 있다.
- **선택지**:
  - ① 지훈이랑 같은 조 → 편하지만 질 확률 높음. 지훈 친밀도 +2
  - ② 민재랑 같은 조 → 이길 확률 높음. 인기 +3 (이기면). 지훈에게 "너 왜 쟤랑 같은 조야?"
  - ③ 아무 데나 → 결과 랜덤. NPC 반응 없음
- **발생 조건**: Y1 5월 체육대회

### 🟢 중학교 (Y2~Y4)

**딜레마 #4 — "새 교복, 새 시작"**
- **상황**: 중학교 첫날. 아는 사람이 거의 없는 교실 (초등 친구가 같은 반이면 1~2명만).
- **선택지**:
  - ① 먼저 말을 건다 → 인기 +2, 새 NPC와 빠른 접촉. 하지만 "오버하는 애" 태그 위험 (인기 40 미만이면)
  - ② 조용히 앉아 있는다 → 변화 없음. 누군가 먼저 말을 걸어올 수도 있고 안 올 수도 있음
  - ③ 초등 친구를 찾는다 → 안정감. 하지만 "걔들끼리만 노는 애" 인상
- **발생 조건**: Y2 3월 1주 (중학교 첫날)

**딜레마 #5 — "시험 성적표"**
- **상황**: 첫 중간고사 성적이 나왔다. 부모에게 보여줘야 한다.
- **선택지**:
  - ① 바로 보여준다 → 부모 반응 분기 (성적에 따라 칭찬/실망/압박)
  - ② 숨긴다 → 일주일 후 부모가 발견. 부모 관계 -5, 신뢰도 하락
  - ③ 먼저 "다음엔 더 잘할게요" 한마디 붙인다 → 부모(정서적 지지): "괜찮아", 부모(엄격함): "말로만 하지 말고 보여줘"
- **숨은 효과**: 성적이 올랐으면 ①이 최선. 떨어졌으면 ③이 관계 손실 최소화. ②는 항상 나쁨.
- **발생 조건**: 매 학기 첫 시험 후 (Y2부터)

**딜레마 #6 — "학원 vs 동아리"**
- **상황**: 동아리 활동이 학원 시간과 겹친다. 동아리는 NPC #6(하은 선배)이 추천해줌.
- **선택지**:
  - ① 학원 유지 → 학업 루틴 보너스 유지. 동아리 기회 상실. 하은에게 "괜찮아, 네가 할 일 하는 게 맞아"
  - ② 동아리 선택 → 루틴 변경 (보너스 리셋!). 재능 성장 루트 개방. 하은과 접촉 채널 추가. 부모(재력/정보력)이면 "학원 끊겠다고?" 갈등 발생
- **핵심**: 루틴 보너스 리셋이라는 시스템 비용 + 부모 갈등이라는 관계 비용 vs 새로운 성장 경로
- **발생 조건**: Y3 봄 (하은 등장 후)

**딜레마 #7 — "친구의 비밀"**
- **상황**: 수빈이 플레이어에게만 고백한다. "나 사실 부모님 이혼했어. 아무한테도 말하지 마."
- **선택지**:
  - ① 비밀을 지킨다 → 수빈 친밀도 +5, 신뢰 관계 형성
  - ② 지훈에게만 말한다 → "지훈은 괜찮잖아." 하지만 지훈이 실수로 다른 애한테 말하면? (확률 분기)
  - ③ SNS에 올린다 → 수빈 친밀도 -30, 관계 파괴. 인기 일시적 +2 (화제성) 후 -5 (평판 악화)
- **발생 조건**: 수빈 친밀도 50 이상 (Y2~Y3)

**딜레마 #8 — "수학여행 룸메이트"**
- **상황**: 수학여행 방 배정. 4명 한 방. 같이 자고 싶은 친구가 4명 이상인데 자리는 3개뿐.
- **선택지**: NPC 3명 선택 (현재 같은 학교인 NPC 중)
  - 선택한 NPC: 친밀도 +3, 특별 이벤트 발생 (밤새 이야기, 장래희망 고백 등)
  - 선택하지 않은 NPC: "나는 안 끼는구나" → 친밀도 변화 없거나, 관계에 따라 -1~-3
- **핵심**: 4명 이상 절친이 불가능한 이유를 자연스럽게 체감시키는 이벤트
- **발생 조건**: Y3 10월 수학여행

**딜레마 #9 — "지훈이네 아버지"**
- **상황**: 지훈이 아버지가 택시 사고로 입원. 지훈이가 학교를 며칠 빠졌다.
- **선택지**:
  - ① 병문안 간다 (주말 1슬롯 소비) → 지훈 친밀도 +8, 특별 대화 "야... 고맙다"
  - ② 카톡으로 위로한다 → 지훈 친밀도 +2. 지훈: "ㅇㅇ 고마워" (짧은 답)
  - ③ 특별히 안 한다 → 지훈 친밀도 -3. 이후 지훈이 먼저 말을 걸지 않는 기간 발생
- **숨은 효과**: ①을 고르면 지훈의 숨겨진 서사가 깊어짐. 고등학교에서 "그때 와줘서 고마웠어" 회상 이벤트 발생
- **발생 조건**: Y3 가을 (지훈 고정 이벤트)

**딜레마 #10 — "질풍노도기: 부모와의 충돌"**
- **상황**: 부모가 성적에 대해 심하게 말했다 / 또는 내가 하고 싶은 걸 반대했다.
- **선택지**:
  - ① 참는다 → 멘탈 -5, 부모 관계 유지. "속이 부글거리지만 삼켰다"
  - ② 대든다 → 멘탈 -2, 부모 관계 -8. "처음으로 소리를 질렀다." 하지만 자립도 +5
  - ③ 집을 나간다 (일시적) → 피로 +10, 멘탈 +3(해방감), 부모 관계 -15. 지훈/수빈에게 연락 가능
- **부모별 후속**:
  - 정서적 지지: "...미안했어. 우리 얘기 좀 하자." → 관계 회복 이벤트
  - 엄격함: "네가 그러고 싶으면 그래." → 냉전 3주. 이후 부모가 먼저 "밥은 먹었어?" 문자
  - 시간/자유: (이 이벤트 자체가 발생하지 않음 — 부모 간섭이 적으니까)
- **발생 조건**: Y3 (멘탈 50 이하 or 부모 기대치 이벤트 2회 이상 발생 시)

**딜레마 #11 — "고등학교 선택"**
- **상황**: 겨울방학. 일반고와 특목고 중 선택해야 한다.
- **선택지**:
  - ① 일반고 (친구 따라) → NPC 관계 유지, 환경 안정. 학업 천장 약간 낮음
  - ② 특목고 (꿈을 위해) → 학업 보너스. 하지만 아는 사람 0명. NPC 관계 대규모 약화. "처음부터 다시 시작"
  - ③ (학업 70+) 자사고 → ②보다 더 극단적. 학업 최고 효율 + 완전한 관계 리셋
- **숨은 효과**: "같이 가는 친구 / 헤어지는 친구" 목록이 화면에 뜬다. 이 장면 자체가 감정 폭탄.
- **발생 조건**: Y4 겨울방학 (필수 이벤트)

### 🔴 고등학교 (Y5~Y7)

**딜레마 #12 — "첫 야자"**
- **상황**: 고등학교 첫 야자. 밤 10시까지 학교에 있어야 한다.
- **선택지**:
  - ① 야자 참여 → 학업 +1 추가, 피로 +5. "교실에 남은 건 나와 형광등뿐이었다"
  - ② 야자 빠지기 → 피로 절약. 하지만 부모(엄격함)이면 갈등. 학교에서 "쟤 야자 빠졌대" 평판 변화
- **발생 조건**: Y5 봄 (고등학교 진학 시, 고정)

**딜레마 #13 — "민재의 슬럼프"**
- **상황**: 항상 잘하던 민재가 성적이 떨어졌다. 복도에서 혼자 앉아 있는 걸 봤다.
- **선택지**:
  - ① 말을 건다 → "... 너한테 이런 말 하기 싫은데, 요즘 좀 힘들어." 민재 친밀도 +5, 라이벌→동료 관계 전환 가능
  - ② 모른 척한다 → 라이벌 관계 유지. 민재는 혼자 회복하지만, 이후 대화 톤이 달라짐
  - ③ 내심 좋다 → (선택지로 직접 나오지 않지만, ②를 고른 뒤 내레이션으로 처리) "솔직히... 약간 후련했다. 그리고 그런 내가 싫었다."
- **발생 조건**: 민재 같은 학교 + Y5~Y6 (민재 학업 65 이상이었을 때)

**딜레마 #14 — "지훈이의 알바 제안"**
- **상황**: 지훈이가 편의점 알바를 시작하면서 "너도 같이 안 할래?" 한다.
- **선택지**:
  - ① 같이 알바 → 용돈 +3만원/주, 학업 루틴 슬롯 1개 소비. 지훈 친밀도 +3/주. "사장님이 우리보고 쌍둥이 같대 ㅋㅋ"
  - ② 거절하되 응원 → 지훈: "ㅇㅇ 그래 너는 공부해" (약간 서운함 + 이해). 친밀도 변화 없음
  - ③ "너 알바 안 하는 게 낫지 않아?" → 지훈 반응 분기: 관계 깊으면 "...네 말이 맞을지도", 관계 얕으면 "넌 모르잖아" (-5)
- **발생 조건**: Y6 봄 (지훈 고정 이벤트)

**딜레마 #15 — "SNS 사건"**
- **상황**: 학교에서 누군가 익명으로 올린 글이 퍼지고 있다. 수빈의 가정 사정이 적혀 있다.
- **선택지**:
  - ① 수빈 편에 선다 (공개적) → 수빈 친밀도 +10, 인기 변동 (인기 높으면 +, 낮으면 위험)
  - ② 수빈에게만 조용히 알려준다 → 수빈 친밀도 +5, 평판 변화 없음
  - ③ 모른 척한다 → 수빈이 혼자 알게 됨. 수빈: "...알고 있었구나." 친밀도 -8
  - ④ (인기 70+) 반 단톡방에서 "그런 거 퍼뜨리지 마" 한마디 → 사건 진화, 수빈 친밀도 +8, 인기 +3
- **발생 조건**: 수빈 같은 학교 + 수빈 비밀 이벤트(#7) 발생 이후

**딜레마 #16 — "예린이의 정보"**
- **상황**: 예린이가 알려준다. "선생님이 추천서 쓸 사람 3명 고른대. 너도 넣을 수 있는데, 생기부 활동 하나를 이번 주에 제출해야 해."
- **선택지**:
  - ① 추천서 도전 (주말 2슬롯 투자) → 진학에 유리, 하지만 이번 주 관계 활동 0
  - ② 포기 → "내 실력으로 가야지." 자립도 +3, 주말 자유
  - ③ 예린에게 더 물어본다 → 정보력 가정이면 추가 정보 획득. 아니면 "나도 그 정도만 알아"
- **발생 조건**: Y6~Y7 (예린 친밀도 40 이상)

**딜레마 #17 — "서아의 갈림길"**
- **상황**: 서아가 진로 고민을 털어놓는다. "나 미대 가고 싶은데... 엄마가 반대해."
- **선택지**:
  - ① "하고 싶은 거 해" → 서아 친밀도 +3. 서아: "...고마워. 근데 그게 쉽지가 않아." (서아의 현실을 무시한 격)
  - ② "부모님 말도 일리 있어" → 서아 친밀도 -2. 하지만 서아가 나중에 "그때 네 말이 맞았어" 할 수도
  - ③ "같이 방법 찾아보자" → 서아 친밀도 +5, 주말 1슬롯 소비 (같이 미대 정보 조사). 정보력 가정이면 실제로 도움이 됨
- **발생 조건**: 서아 친밀도 50 이상 + Y5~Y6

**딜레마 #18 — "루나의 정체"**
- **상황**: 온라인 친구 루나(김도하)와 실제로 만날 기회가 생겼다. 루나가 "우리 한번 만날래?" 한다.
- **선택지**:
  - ① 만난다 → 루나의 실제 모습 공개. 온라인과 다른 느낌에 어색함. 친밀도 +5 또는 -3 (관계 깊이에 따라)
  - ② 안 만난다 → "온라인에서가 더 편해." 루나: "ㅇㅇ 그것도 맞아." 관계 유지되지만 깊어지지 않음
- **핵심**: 온라인 관계의 한계와 가능성. "실제로 만나면 달라질까?"
- **발생 조건**: 루나 친밀도 60 이상 + Y5 이후

**딜레마 #19 — "수능 전날"**
- **상황**: 내일이 수능이다. 밤 10시. 더 공부할 수도 있고, 잘 수도 있다. 그런데 **친밀도 1위 NPC**에게서 전화가 왔다.
- **전화 거는 NPC 결정**: 친밀도가 가장 높은 NPC가 자동 선택됨 (동률 시 관계 기간이 긴 쪽 우선)
- **선택지**:
  - ① 전화 안 받고 공부 → 학업 미세 보정 +0.5. (다음 날 아침 문자) "파이팅!"
  - ② 전화 안 받고 잠 → 컨디션 보정 (피로 -5). 같은 문자
  - ③ 전화 받는다 → 30분 통화. 학업/컨디션 보정 없음. 멘탈 +5. — **이 선택이 가장 점수 비효율적이지만, 엔딩에서 가장 아름다운 회상이 된다**
- **NPC별 통화 내용**:
  - 지훈: "야, 긴장돼? 나도. 끝나고 PC방 가자."
  - 수빈: "야, 나 잠이 안 와서... 너도 그래? 다행이다."
  - 서아: "...나 전화해도 돼? 그냥 목소리 듣고 싶어서."
  - 민재: "야. 내일 잘 봐. ...나도 잘 볼 거야." (웃음)
  - 시우: "긴장 풀어. 우리 끝나면 뭐 하지?"
- **발생 조건**: Y7 11월 수능 전날 (친밀도 1위 NPC 친밀도 30 이상)

**딜레마 #20 — "수능이 끝나고"**
- **상황**: 시험지를 내려놓았다. 교문을 나서는데, 핸드폰을 켠다. 연락처 목록이 뜬다.
- **선택지**: **가장 먼저 연락할 사람 1명을 고른다.**
  - 선택한 NPC: 에필로그에서 "수능 끝나고 내가 가장 먼저 전화한 사람" 카드 생성
  - 선택하지 않은 NPC들: 각자의 반응 ("읽씹", "축하해!", "나도 방금 끝남", 또는 연락이 오지 않음)
- **핵심**: 7년간의 관계가 이 한 번의 선택으로 압축된다. 이 장면 자체가 공유 포인트.
- **발생 조건**: Y7 수능 직후 (필수 이벤트)

## 3-B. 연애 딜레마 8개 — "마음이 흔들리는 순간"

> 연애 이벤트는 NPC 성별과 플레이어 성별 조합에 따라 발생 여부/내용이 달라진다.
> 연애는 "시스템"이 아니라 "감정의 축적"으로 진입한다 — 특정 선택을 밟아야 열리는 구조.

**연애 딜레마 R1 — "우산"**
- **상황**: 비 오는 하교길. 서아(또는 연애 가능 NPC)가 우산이 없다.
- **선택지**:
  - ① 우산을 같이 쓴다 → 연애 플래그 +1. "어깨가 닿았다. 아무 말도 안 했다."
  - ② 우산을 빌려주고 뛴다 → 친밀도 +3. "젖어서 집에 왔는데 기분이 이상하게 좋았다."
  - ③ 모른 척한다 → 변화 없음
- **발생 조건**: 연애 가능 NPC 친밀도 40+ (Y1 후반~Y2)

**연애 딜레마 R2 — "축제에서"**
- **상황**: 학교 축제. 자유시간에 누구와 돌아다닐지 고른다.
- **선택지**: 같이 돌 NPC 1명 선택
  - 연애 가능 NPC 선택 → 둘만의 시간. 연애 플래그 +1. "솜사탕 줄에서 손이 닿을 뻔했다."
  - 다른 NPC 선택 → 우정 이벤트. 연애 가능 NPC: "...재밌었어?" (약간 아쉬운 톤)
- **발생 조건**: Y2~Y3 5월 축제

**연애 딜레마 R3 — "같은 사람을 좋아한다"**
- **상황**: 친한 친구(수빈/지훈)가 내가 신경 쓰는 NPC에 대해 "쟤 좀 괜찮지 않아?" 한다.
- **선택지**:
  - ① "응 나도 좀..." → 친구: "...아 그래?" 미묘한 침묵. 경쟁 관계 시작
  - ② "글쎄? 모르겠는데" → 감정을 숨김. 친구 관계 유지. 하지만 자기 마음에 솔직하지 못한 아쉬움
  - ③ "너 좋아하는 거야?" → 친구의 마음을 먼저 확인. 결과에 따라 양보/경쟁 분기
- **발생 조건**: 연애 플래그 2+ & 해당 NPC 친밀도 50+ (Y3~Y5)

**연애 딜레마 R4 — "고백 타이밍"**
- **상황**: 연애 플래그가 충분히 쌓였을 때. "이 감정, 말해야 할까?"
- **선택지**:
  - ① 고백한다 → 성공/실패 분기 (친밀도 70+면 성공 확률 높음). 성공: 연애 루트 진입. 실패: 친밀도 -10, 1달간 어색함
  - ② 지금은 아니다 → 관계 유지. 하지만 타이밍을 놓치면? "다음에"가 없을 수도 있다
  - ③ 편지를 쓴다 → 고백보다 부드러움. 성공 확률 약간 낮지만 실패 시 손실도 적음 (-5)
- **발생 조건**: 연애 플래그 3+ (Y3 이후)

**연애 딜레마 R5 — "공개 vs 비밀"**
- **상황**: 연애 시작 후. "우리 사귀는 거 친구들한테 말할까?"
- **선택지**:
  - ① 공개 → 인기 변동 (반응 분기), 친구들 반응 다양. "헐 너네 사귀어?!" 화제성
  - ② 비밀 → 둘만의 세계. 하지만 들킬 위험. 들키면 "왜 숨겼어?" 반발
- **핵심**: 공개하면 관계에 외부 압력이 생기고, 숨기면 내부 긴장이 쌓임
- **발생 조건**: 연애 루트 진입 후 2주

**연애 딜레마 R6 — "시험 기간의 거리"**
- **상황**: 시험 기간. 연인에게서 연락이 줄었다. 나도 공부해야 한다.
- **선택지**:
  - ① 먼저 연락한다 → 멘탈 +2 (안심), 학업 시간 -1슬롯
  - ② 기다린다 → 불안감. 시험 끝나고 "왜 연락 안 했어?" 대화 발생
  - ③ "우리 시험 끝나고 보자" 문자 → 성숙한 대처. 친밀도 유지. 하지만 시험 기간이 길면 "우리 식은 거 아니야?"
- **발생 조건**: 연애 중 + 시험 기간

**연애 딜레마 R7 — "이별의 예감"**
- **상황**: 진학으로 학교가 달라지게 되었다. 또는 연인의 관심사가 달라지고 있다.
- **선택지**:
  - ① 매달린다 → 관계 연장 시도. 연락하기 비용 증가. 다른 관계 여유 감소
  - ② 자연스럽게 놓는다 → 이별. 멘탈 -8. 하지만 "좋은 기억으로 남기자"
  - ③ 솔직하게 말한다 → "우리... 이대로 괜찮을까?" 결과는 관계 깊이에 따라 분기
- **발생 조건**: 연애 중 + 진학 분기점 또는 친밀도 50 이하로 하락

**연애 딜레마 R8 — "이별 후 같은 교실"**
- **상황**: 헤어졌는데 같은 반이다. 매일 얼굴을 마주친다.
- **선택지**:
  - ① 무시한다 → 멘탈 -2/주 (지속). "눈을 어디 둬야 할지 모르겠다"
  - ② 친구로 돌아가려 한다 → 시도. 성공하면 친밀도 30에서 재시작. 실패하면 어색함 고정
  - ③ 자리를 바꿔달라고 한다 → 담임에게 요청. 인기 -2 (소문). 하지만 멘탈 안정
- **발생 조건**: 이별 후 같은 학교인 경우

## 3-C. 미세 갈등 이벤트 8개 — "학교 안의 공기"

> 큰 사건이 아닌, **사소하지만 감정적으로 센 순간들**. 한국 학창시절의 "공기"를 만드는 이벤트.
> 선택지가 있지만, 어떤 걸 골라도 큰 수치 변동은 없다. 대신 **감정의 질감**이 달라진다.

**미세 갈등 M1 — "급식 자리"**
- **상황**: 급식 시간. 평소 앉던 자리에 다른 애들이 앉아 있다. 내 자리가 애매하다.
- **선택지**:
  - ① 끼어 앉는다 → "여기 앉아도 돼?" 반응 분기 (인기에 따라)
  - ② 다른 자리를 찾는다 → 조용히 빈자리로. 아무도 모른다.
  - ③ 화장실에서 시간을 보낸다 → 멘탈 -1. "배고픈데 들어갈 타이밍을 놓쳤다."
- **발생 조건**: 인기 40 미만일 때 랜덤 (Y2 이후)

**미세 갈등 M2 — "단톡방 읽씹"**
- **상황**: 반 단톡방에서 주말 약속을 잡고 있다. 내가 "나도!" 했는데 아무도 반응이 없다. 대화가 그냥 넘어갔다.
- **선택지**:
  - ① 다시 한번 말한다 → "나도 갈게!" 반응 올 수도, 안 올 수도 (인기에 따라)
  - ② 지운다 → 삭제. 아무도 모른다. 멘탈 -2
  - ③ 폰을 덮는다 → "...됐다." 주말에 혼자 보냄
- **발생 조건**: 인기 45 미만 + 같은 반 NPC 친밀도 40 미만 (Y2 이후)

**미세 갈등 M3 — "조별과제 남겨짐"**
- **상황**: 조별과제 조 편성. 마지막까지 남았다. "나머지 사람은... 여기로."
- **선택지**:
  - ① 신경 안 쓴다 → "어디든 상관없어." 결과 랜덤
  - ② 열심히 한다 → 조원들이 인정. "쟤 의외로 잘하네?" 인기 +1
  - ③ 아는 애한테 "같은 조 하자" 선제 제안 → 인기 40+면 성공. 미만이면 "아 나 벌써 다른 데..."
- **발생 조건**: 랜덤 (학기당 1회)

**미세 갈등 M4 — "체육시간 짝"**
- **상황**: 체육시간. 둘씩 짝을 지으라는데, 홀수라 한 명이 남는다.
- **선택지**: (짝을 못 구한 경우에만 발생)
  - ① 선생님 옆에 선다 → 체면 손상. 인기 -1. 하지만 선생님 호감 +1
  - ② "나 셋이서 하면 안 돼요?" → 해결. 하지만 "불쌍해서 끼워준 거" 느낌
  - ③ (인기 50+면 이 이벤트 자체가 발생하지 않음)
- **발생 조건**: 인기 50 미만 + 체육 시간 랜덤

**미세 갈등 M5 — "뒷담화"**
- **상황**: 복도에서 내 이름이 들렸다. 누군가 내 얘기를 하고 있다.
- **선택지**:
  - ① 다가간다 → "뭐? 내 얘기야?" 대화 중단. 어색한 침묵. 관계에 따라 "아 아니야 ㅋㅋ" 또는 진짜 뒷담
  - ② 못 들은 척한다 → 멘탈 -2. "뭐라고 했을까." 다음 날까지 신경 쓰임
  - ③ 지훈/수빈에게 물어본다 → "야, 쟤네 나 뭐라 그래?" 정보 획득 (or "몰라 신경 쓰지 마")
- **발생 조건**: 인기 변동이 클 때 (급상승 또는 급하락 후) 랜덤

**미세 갈등 M6 — "쉬는 시간 10분"**
- **상황**: 쉬는 시간. 교실에 혼자 앉아 있다. 다들 복도에 나갔다.
- 선택지 없음. 감정만 전달:
  - (친구 많을 때) "혼자만의 시간이 좋았다. 10분이 금이었다."
  - (친구 적을 때) "다들 어디 간 걸까. 나만 교실에 있는 것 같다."
- **발생 조건**: 평일 랜덤 (분위기 조성용)

**미세 갈등 M7 — "성적표 비교"**
- **상황**: 시험 끝. 친구가 "너 몇 점이야?" 물어본다. 나보다 잘했거나, 내가 더 잘했거나.
- **선택지**:
  - ① 솔직하게 말한다 → 관계에 따라 "오 대박" 또는 미묘한 분위기
  - ② 대충 넘긴다 → "그냥 그래..." 의심 또는 배려로 해석됨
  - ③ 먼저 물어본다 → 주도권 확보. 상대 반응 먼저 확인 후 대응 가능
- **핵심**: 성적이 우정에 미세한 균열을 만드는 순간. 부끄러움/체면의 감정.
- **발생 조건**: 시험 후 매회 (Y2 이후)

**미세 갈등 M8 — "부모님 앞에서"**
- **상황**: 친구가 집에 놀러 왔는데, 부모님이 갑자기 잔소리를 한다. 또는 부모님이 나를 어린애 취급한다.
- **선택지**:
  - ① 참는다 → 멘탈 -1. "제발 그만..." 얼굴이 빨개진다

- **핵심**: 부끄러움. 사적인 공간과 사회적 공간이 충돌하는 순간.
- **발생 조건**: NPC와 "친구 집 방문" 활동 시 랜덤

## 5. 안전망 이벤트 — "바닥을 쳤을 때"

> 02_스탯시스템의 부모 개입 안전망(피로 85+ 또는 멘탈 15 이하)이 발동할 때, 단순 수치 회복이 아닌 **서사가 붙은 장면**으로 연출한다.

### 멘탈 위기 (멘탈 15 이하)

```
[정서적 지지 가정]
  → 저녁 식탁. 엄마가 갑자기 말한다.
  → "오늘 학교 안 가도 돼. 우리 어디 갈까?"
  → 하루 쉬기. 멘탈 +15.
  → "괜찮아. 엄마아빠는 네 편이야."

[엄격함 가정]
  → 아빠가 방문을 열고 들어온다.
  → "...밥은 먹었어?"
  → 평소에 하지 않던 한마디. 그게 더 충격이다.
  → 1주간 압박 해제.
  → 나레이션: "그날 밤, 아빠가 처음으로 등을 토닥여줬다."

[재력 가정]
  → 엄마가 병원 예약을 했다.
  → "상담 한번 받아보자. 나쁜 거 아니야."
  → 피로 -20 (직접적 문제 해결).

[시간/자유 가정]
  → 아무도 눈치채지 못한다.
  → 하지만 **절친 NPC**에게서 연락이 온다 (절친 1명+ 있을 때).
  → 지훈이면: "야, 나와. 그냥 걷자."
  → 수빈이면: "야 너 오늘 학원 안 왔잖아. 괜찮아?"
  → 서아이면: "...오늘 좀 이상해 보여서. 나올 수 있어?"
  → 멘탈 +8 (부모 대신 NPC 안전망 작동).

[정보력 가정]
  → 엄마가 스케줄을 봤다.
  → "이번 주는 학원 쉬어. 내가 선생님한테 말해놓을게."
  → 다음 주 강제 슬롯 전부 해제.
```

### 피로 위기 (피로 85 이상)

```
[공통 연출]
  → 월요일 아침. 알람이 울렸는데 몸이 안 일어난다.
  → "...5분만 더." 그 5분이 30분이 됐다.

[정서적 지지] → 가족 외식. "오늘은 맛있는 거 먹으러 가자." 피로 -15, 멘탈 +5
[재력] → 보약/영양제. "이거 먹어. 비싼 거야." 피로 -20
[엄격함] → "오늘은 쉬어." 한마디. 1주간 루틴 강제 해제.
[시간/자유] → 혼자 하루 종일 잠. 피로 -10. 아무도 모른다.
```

> **원칙**: 안전망은 "게임이 구해주는 것"이 아니라 "내 주변 사람들이 나를 걱정하는 것"으로 느껴져야 한다. 시간/자유 가정은 부모 안전망이 약한 대신, NPC 관계가 안전망 역할을 한다.

## 6. 루틴 이벤트

### 6-1. 루틴 권태 (8주 이상 동일 루틴)

같은 루틴을 8주 이상 유지하면 **권태 이벤트**가 발생한다.

```
[8주차]
  "또 같은 길, 같은 학원, 같은 자리."
  "오늘따라 발이 무겁다."

  선택지:
  ① 그래도 간다 → 루틴 유지. 피로 +2 (권태 추가 피로)
  ② 오늘만 빼먹는다 → 피로 -3, 멘탈 +2. 루틴 카운트 유지 (1회는 허용)
  ③ 루틴을 바꾼다 → 루틴 변경 화면으로. 보너스 리셋 경고.

[12주차 이상]
  "이제 눈 감고도 갈 수 있다. 그게 좋은 건지 나쁜 건지 모르겠다."
  → 권태 피로가 +3으로 증가
  → ②의 "1회 허용"이 이미 사용됐으면, 빼먹기 = 루틴 카운트 리셋
```

> 루틴 보너스의 가치(최대 +2.0)와 권태 피로의 누적 사이에서 트레이드오프 발생.

### 6-2. 루틴 숙련 (엄격함 가정 전용)

엄격함 가정에서 8주 연속 동일 루틴을 유지하면:

```
"몸이 기억한다. 머리보다 먼저 손이 움직인다."
→ 루틴 숙련 달성: 피로 -1/주 (조건부)
→ 루틴 변경 시 즉시 해제
→ 나레이션: "습관이 된다는 건 이런 거구나."
```

## 8. 회고 카드 시스템 — "공유되는 한 문장"

> 매주가 아닌, **구간 전환점**에서만 회고 카드가 생성된다. 희소성이 임팩트를 만든다.

### 8-1. 회고 카드 발생 시점

| 시점 | 카드 유형 | 예시 |
|------|----------|------|
| 학기말 (7월/12월) | **학기 결산 카드** | "이번 학기, 나는 성적을 올렸지만 지훈과 점심을 같이 먹은 횟수가 줄었다." |
| 방학 시작 | **해방 카드** | "방학이 시작됐다. 가장 먼저 떠오른 건 공부가 아니라 바다였다." |
| 방학 끝 | **개학 카드** | "이번 여름, 나는 누구와 가까워졌고 누구와 멀어졌다." |
| 진학 시 | **전환 카드** | "초등학교 6년이 끝났다. 같이 가는 사람, 헤어지는 사람." |
| 수능 직후 | **결산 카드** | "7년이 끝났다. 내가 가장 자주 선택한 건 ____였다." |

### 8-2. 회고 카드 생성 로직

카드 문장은 해당 기간의 **실제 플레이 데이터**를 기반으로 자동 생성된다.

```
[학기 결산 카드 생성 변수]
- 가장 많이 한 활동
- 가장 많이 만난 NPC
- 가장 많이 놓친 NPC (친밀도 가장 많이 하락한)
- 스탯 변화량 (가장 많이 오른 것, 가장 많이 떨어진 것)
- 주요 선택 (딜레마에서 뭘 골랐는지)

[조합 예시]
- 학업이 가장 많이 올랐고 + 지훈 친밀도가 가장 많이 떨어졌으면:
  → "이번 학기, 나는 책상 앞에서 가장 많은 시간을 보냈다. 지훈이의 카톡은 읽지 못한 게 3개다."

- 멘탈이 가장 많이 떨어졌고 + 서아를 가장 많이 만났으면:
  → "힘든 학기였다. 그래도 서아와 나눈 대화가 나를 버티게 했다."

- 아무 NPC도 잘 안 만났으면:
  → "조용한 학기였다. 혼자 보낸 시간이 길었다. 그게 나쁜 건 아니었다."
```

### 8-3. 카드 디자인

- 카드 형태: 가로형 이미지 + 한 줄 문장
- 공유 버튼: 캡처용 이미지 자동 생성 (SNS 공유 최적화)
- 수집: 인생 앨범에 자동 저장 → 엔딩 후 전체 타임라인으로 열람 가능

```
[2단계: 주말 반강제 — 2주 연속 하락]
  "이번 주말은 학원 특강 넣어놨어."
  → 주말 1슬롯이 "학원 특강"으로 고정됨 (2주간)
  → 선택지:
    ① 수용 → 학업 활동 진행, 멘탈 -2
    ② 거부 → 부모 관계 -5, 용돈 삭감. "네가 알아서 한다며?"

[3단계: 확대 — 3주 이상 지속]
  "학원도 늘려야 할 것 같아."
  → 주말 반강제 지속 + 추가 페널티 (용돈 삭감 or 자유 시간 축소)
  → 멘탈 -2 추가
  → 이때 딜레마 #10(질풍노도기)이 발생할 확률 상승
```

### 9-2. 비학업 과투자 트리거

재능이나 인기에 집중하면서 학업이 떨어질 때:

```
[부모(재력/정보력)]
  "밴드부? 그건 취미로 해. 학원부터 다녀."
  → 재능 활동 선택 시 부모 관계 -1/주 (지속적 마찰)

[부모(정서적 지지)]
  "네가 좋아하는 거 하면 돼. 근데... 시험은 좀 챙기자?"
  → 부드러운 경고. 패널티 없음. 하지만 학업 40 이하가 되면 진지한 대화 이벤트 발생.

[부모(시간/자유)]
  → (이벤트 없음. 알아서 해야 함.)
```

## 10. 특수 이벤트

### 10-1. 계절 감정 이벤트

한국 학창시절의 계절감을 살리는 소소한 이벤트. 선택지 없이, 감정만 전달.

```
[3월 — 봄]
  "벚꽃이 교문 앞에 폈다. 새 교과서 냄새가 났다."

[7월 — 여름방학 전]
  "시험이 끝났다. 교실 창문 너머로 매미 소리가 들렸다."

[9월 — 개학]
  "여름이 끝났다. 거울 속의 나는 살짝 달라져 있었다."

[12월 — 겨울]
  "하교길에 눈이 왔다. 누군가와 같이 걸으면 좋겠다고 생각했다."

[2월 — 졸업]
  "교실이 비어 있었다. 칠판에 누군가 적어놓은 글씨가 남아 있었다."
```

### 10-2. "그때 그 선택" 회상 이벤트

이전에 내린 딜레마 선택이 나중에 **회상**으로 돌아온다.

```
[예시: 딜레마 #9(지훈 아버지 병문안)에서 ① 병문안을 갔었다면]

고3 겨울. 지훈이와 편의점 앞에 앉아 있다.
지훈: "야, 기억나? 중2 때 우리 아빠 입원했을 때."
지훈: "그때 네가 와줬잖아. 그때 진짜 고마웠어."
지훈: "...나 그때 혼자였으면 진짜 힘들었을 거야."

→ 멘탈 +3 (과거의 선택이 현재의 보상으로)
→ 이 장면은 "수능 끝나고 가장 먼저 전화한 사람" 선택에 영향
```

> 모든 딜레마가 회상으로 돌아오지는 않는다. **5~6개의 핵심 딜레마**만 후반부에 회수된다. 나머지는 엔딩 에필로그에서 간접 언급.

## 이벤트 밀도 가이드

| 구간 | 주당 이벤트 확률 | 설명 |
|------|----------------|------|
| 평일 (루틴 자동진행) | 10~15% | 대부분 아무 일 없이 지나감 |
| 주말 (선택 후) | 20~25% | 선택에 따른 소규모 이벤트 |
| 시험 기간 | 80% | 시험 관련 이벤트 집중 |
| 방학 | 15~20% | 자유 시간이 많지만 이벤트는 적당히 |
| 체크포인트 (축제/수학여행 등) | 100% | 고정 이벤트 |

> **아무 일도 일어나지 않는 주가 게임에 리듬을 준다.** 매주 이벤트가 터지면 피로감이 온다. 조용한 주가 있어야, 이벤트가 터졌을 때 "오늘은 뭔가 다르다"는 긴장이 생긴다.
