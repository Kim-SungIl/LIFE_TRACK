# 08. 학교 시스템 / 시험·등수 / SNS 시스템

> 설계 원칙
> - 학교는 "공개적으로 보이는 나" — 가시성, 평판, 등수가 자존감과 관계를 좌우
> - SNS는 "사적으로 흔들리는 관계" — 기존 관계를 증폭하거나 식게 만드는 온도 조절기
> - 앱 이름을 재현하지 않는다 — 피드/스토리/단톡/개인채팅/익명소문이라는 감정 단위로 추상화
> - 학교에서 보이는 모습과 온라인에서 보이는 모습은 다르다 — 이 간극이 드라마를 만든다

---

## 1. 학교 종류

### MVP 범위
- **MVP: 일반고만 구현**
- 확장(DLC): 특목고(과학고/외고), 예술고, 체육고, 특성화고(공업/상업)

> **MVP에서 특목고를 제외하는 이유:**
> 특목고는 슬롯 구조, 등수 분포, 야자 규칙, NPC 풀, 멘탈 압박이 전부 달라져
> 사실상 "다른 게임 모드"에 가깝다. 일반고 하나를 완벽하게 만드는 것이 우선.
> 단, 리플레이 가치를 위해 확장에서 반드시 추가해야 한다.

### 일반고 기본 특성

| 항목 | 일반고 |
|------|--------|
| **진학 조건** | 없음 (배정) |
| **학교 수업 효과** | 학업 +1 |
| **야자 강도** | 반강제 (거부 가능) |
| **자유 슬롯 (고1~2)** | 방과후 1 + 주말 2 = 3 |
| **NPC 풀** | 다양한 유형 (관계/재능/일탈) |
| **동아리** | 자유 선택 |
| **대회 기회** | 교내 → 지역 |
| **분위기** | 다양성 높음, 압박 보통 |

### 확장용 학교 설계 (DLC — 설계만 기록, MVP 미구현)

<details>
<summary>특목고 설계안 (접기)</summary>

| 항목 | 특목고 |
|------|--------|
| 진학 조건 | 학업 85+ 또는 재능 85+ (분야별) |
| 학교 수업 효과 | 학업 +1.5 (심화 수업) |
| 야자 강도 | 강제 (거부 시 퇴학 경고) |
| 자유 슬롯 (고1~2) | 방과후 0 + 주말 2 = 2 |
| NPC 풀 | 학업 특화 NPC 중심 |
| 동아리 | 학교 지정 + 1개 자유 |
| 대회 기회 | 교내 → 전국 (직행) |
| 분위기 | 경쟁 극심, 멘탈 소모 큼 |
| 학업 자연감소 (Y5~6) | -1.5/주 (수업 난이도 높음) |

설계 의도: "빠른 학업 성장"과 "자유의 박탈" 사이의 교환.
학업 특화 빌드에게는 최적이지만, 올라운더에게는 지옥.
"좋은 학교 = 좋은 인생"이 아님을 보여준다.
같은 학업 85여도 일반고에서는 4등, 특목고에서는 27등 — "물고기가 큰 연못에 갈 것인가" 딜레마.
</details>

### 학교 배정 타이밍

```
[초등학교 (Y1)]
→ 선택 없음. 모든 플레이어가 같은 초등학교.
→ "모두가 같은 출발선"을 체감하는 시기.

[중학교 (Y2~Y4)]
→ 선택 없음. 자동 배정.
→ 단, 같은 중학교에 가는 초등 NPC는 부모 조합/지역에 따라 달라짐.
→ "운에 의해 친구가 결정되는" 현실 반영.

[고등학교 (Y5~Y7)] — MVP: 일반고 자동 배정
→ MVP에서는 일반고 배정. 확장판에서 학교 선택 이벤트 추가.
→ 확장 시: Y4 겨울방학에 일반고 vs 특목고/특성화고 선택
```

## 2. 학교별 슬롯 구조 (01_게임루프.md 연동)

> 학년이 올라갈수록 자유 슬롯이 줄어든다.
> 이것은 "어른이 된다 = 자유가 줄어든다"는 게임의 핵심 메타포.

### 평일 슬롯

| 학년 | 슬롯1 | 슬롯2 | 슬롯3 | 비고 |
|------|-------|-------|-------|------|
| 초등 (Y1) | 학교(고정) | 방과후(자유) | 방과후(자유) | 가장 여유로운 시기 |
| 중학 (Y2~Y4) | 학교(고정) | 방과후(자유) | 방과후(자유) | 구조 동일, 난이도 상승 |
| 고등 1~2 (Y5~6) | 학교(고정) | 방과후(자유) | 야자(반강제) | 야자 거부 가능하나 대가 있음 |
| 고3 (Y7) | 학교(고정) | 야자(강제) | 야자2(강제) | 자유 슬롯 극감 |

### 주말 슬롯

| 학년 | 슬롯A | 슬롯B | 비고 |
|------|-------|-------|------|
| 초~고2 | 자유 | 자유 | 2슬롯 완전 자유 |
| 고3 | 자유 | 자습(반강제) | 실질 자유 1슬롯 |

### 야자 규칙 (일반고)

```
[일반고 야자]
- 기본: 반강제 (거부 가능)
- 거부 시: 학교 평판 -2, 부모 반응 (엄격함 가정: 거부 불가)
- 선택 시: 학업 +2, 피로 +5
- 시간/자유 가정: 거부 패널티 감소
```

## 3. 시험·등수 시스템

> "공부만 잘한다고 1등이 아니다. 다른 학생도 공부한다."

### 3-1. 시험 일정

| 시험 | 시기 | 범위 |
|------|------|------|
| 중간고사 | 4월 4주 / 10월 3주 | 해당 학기 전반부 |
| 기말고사 | 7월 1주 / 12월 2주 | 해당 학기 전체 |
| 수능 모의고사 | 3월, 6월, 9월 (고등만) | 전 범위 |
| 수능 | 11월 2주 (Y7만) | 전 범위 — 게임 최대 이벤트 |

### 3-2. 등수 산정

플레이어의 학업 스탯을 **학교 내 가상 학생 분포**와 비교하여 등수를 산정한다.

```
[가상 학생 분포]

학교별로 "가상 학생 풀"이 존재한다 (200명 가정).
각 학생의 학업 수치는 정규분포를 따른다.

일반고: 평균 45, 표준편차 15
특목고: 평균 70, 표준편차 10

플레이어의 학업 스탯을 이 분포에 대입하여 등수 계산.
```

| 플레이어 학업 | 일반고 등수 (200명) | 특목고 등수 (200명) |
|-------------|-------------------|-------------------|
| 95 | **1등** (0.04%) | 약 6등 (상위 3%) |
| 85 | 약 4등 (상위 2%) | 약 27등 (상위 13%) |
| 70 | 약 19등 (상위 10%) | 약 100등 (중위) |
| 55 | 약 74등 (상위 37%) | 약 172등 (하위 14%) |
| 40 | 약 130등 (중하위) | 약 197등 (꼴찌권) |

> **설계 의도:**
> - 같은 학업 85여도 일반고에서는 4등, 특목고에서는 27등
> - "특목고 가면 성적이 떨어진다"는 한국 학생의 보편적 경험 반영
> - 등수가 멘탈, 부모 반응, NPC 인식에 영향 → 학교 선택이 단순한 버프가 아님

### 3-3. 등수의 영향

| 등수 구간 | 효과 |
|-----------|------|
| **전교 1~3등** | "전교 수석권" 태그. 부모 관계 +5, 선생님 호감 최대, "전교 1등" 칭호, NPC들이 인식 |
| **상위 10%** | "상위권" 태그. 부모 만족, 교사 추천 대상, 특목고 지원 자격 |
| **상위 30%** | "중상위" 태그. 안정권. 부모 반응 중립 |
| **상위 50%** | "중위" 태그. 부모에 따라 압박 시작 (재력/엄격함 가정) |
| **하위 30%** | "하위" 태그. 부모 갈등 이벤트, 보충 수업 반강제, 멘탈 -2 |
| **하위 10%** | "최하위" 태그. 담임 면담 강제, 부모 긴급 개입, NPC 반응 변화 |

### 3-4. 시험 이벤트 플로우

```
[시험 2주 전]
→ "시험 기간" 분위기 전환 (BGM 변경, UI 색상 변화)
→ 방과후 슬롯 1개가 "시험 공부" 반강제 (거부 가능하지만 성적에 영향)
→ 시험 벼락치기 활동 해금 (+4 학업, +14 피로)
→ NPC 반응: "야 시험공부 같이 할래?" (스터디 그룹 이벤트)

[시험 주]
→ 시험 연출: 시험지 펼치는 장면 + 긴장감 BGM
→ 결과: 현재 학업 스탯 기준으로 등수 산정
→ ±운 요소: -5% ~ +5% 변동 (컨디션 반영)
  → 멘탈 70+: 변동 범위 축소 (-3%~+3%)
  → 멘탈 30-: 변동 범위 확대 (-10%~+5%)
  → "시험 불안" — 실력이 있어도 멘탈이 낮으면 못 치는 현실

[시험 결과 발표]
→ 성적표 UI: 과목별 등급 + 전교 등수 + 석차 백분율
→ 등수 변화 연출: 전 시험 대비 ↑↓ 표시
→ NPC 반응: 등수에 따라 축하/위로/무시/질투
→ 부모 반응: 부모 유형별 차등 반응 (§3 연동)
→ SNS 반응: "시험 끝났다!" 게시물 → 등수에 따른 반응
```

### 3-5. 시험과 부모 반응 매트릭스

| 등수 변화 | 재력 가정 | 엄격함 가정 | 정서적지지 가정 | 시간/자유 가정 |
|-----------|-----------|------------|----------------|---------------|
| 대폭 상승 (20등+) | "우리 돈이 아깝지 않네" 용돈+2 | "그래, 이 정도는 해야지" (칭찬 없음) | "정말 대단하다!" 멘탈+5 | "잘했네~" (담백) |
| 소폭 상승 | 중립 | "더 올라갈 수 있어" 압박 유지 | "꾸준히 하고 있구나" 멘탈+2 | 중립 |
| 유지 | "현상 유지네" | "제자리걸음이야" 멘탈-2 | "안정적이네" 중립~긍정 | 중립 |
| 소폭 하락 | "학원을 바꿔야 하나" 면담 이벤트 | "왜 떨어졌어?" 멘탈-5, 주말 1슬롯 강제학습 | "괜찮아, 다음에" 멘탈+1 | "뭐 그럴 수도 있지" |
| 대폭 하락 (20등+) | 과외 강제 투입 + 멘탈-3 | 폭풍 면담 + 멘탈-10 + 다음 시험까지 외출 금지 | "같이 방법 찾아보자" 멘탈-2 | "...너 요즘 뭐 하니?" 면담 |

## 4. 멀티 랭킹 시스템

> "세상에는 성적 말고도 순위가 있다."

### 4-1. 랭킹 종류

| 랭킹 | 기준 | 업데이트 주기 | 전용 칭호 (1등) |
|------|------|-------------|-----------------|
| **전교 등수** | 학업 스탯 (시험 결과) | 시험 후 | 전교 수석 |
| **인기 랭킹** | 인기(존재감) 스탯 | 매달 | 학교 연예인 |
| **재능 랭킹** | 재능 스탯 (분야별) | 대회 후 | 분야별 칭호 (음악의 신, 코트의 왕 등) |
| **생활기록부 점수** | 봉사+동아리+출석+수상 합산 | 학기 말 | 모범생 |
| **문제아 지수** | 일탈 수치 (숨겨짐) | 실시간 | (숨겨진 칭호: 학교의 그림자) |

### 4-2. 랭킹의 게임 내 영향

```
[전교 등수]
→ 부모 반응 직결 (§3-5 참조)
→ 진학 자격 (특목고, 대학 수시/정시)
→ NPC 인식: "걔 공부 잘하잖아" / "걔 성적 떨어졌다며?"
→ 장학금 이벤트 (상위 5%)

[인기 랭킹]
→ NPC 첫 만남 호감도 보정
→ 학교 행사 주도권 (반장 선거, 축제 MC 등)
→ SNS 팔로워 수 연동
→ 연애 이벤트 확률 증가

[재능 랭킹]
→ 대회 출전 자격 (교내 → 지역 → 전국)
→ 특기자 전형 지원 자격
→ 생활기록부 특기 기재
→ NPC 인식: "걔 ○○ 진짜 잘하더라"

[생활기록부 점수]
→ 대학 수시 전형에 직접 반영
→ 교사 추천서 품질 결정
→ "성적은 별로인데 생기부가 빵빵한 학생" 루트 가능

[문제아 지수]
→ 숨겨진 수치. 일탈 활동 누적 시 상승
→ 교사 태도 변화 (감시 강화, 반성문 이벤트)
→ 일정 수준 이상 → "퇴학 경고" 이벤트
→ 특수 엔딩 분기 (자립 루트)
```

### 4-3. 랭킹은 "상대적"

```
중요한 점: 랭킹은 절대 수치가 아닌 상대 순위.

학업 70으로 일반고에서 19등 → "상위권" 기분 좋음
학업 70으로 특목고에서 100등 → "중위" 자존감 하락

같은 실력이라도 어디에 있느냐에 따라 체감이 완전히 다르다.
이것이 "물고기가 큰 연못에 갈 것인가, 작은 연못에 갈 것인가" 딜레마.
→ 고등학교 선택의 핵심 갈등 요소.
```

## 5. 학교 가시성 시스템

> "친밀도는 1:1 관계. 가시성은 학교 안에서 '내가 어떻게 보이느냐'."

NPC 친밀도(§4 관계시스템)와 별개로, 학교라는 공간에서 플레이어가 **어떻게 인식되는가**를 다루는 시스템.

### 5-1. 가시성 3축

| 축 | 의미 | 올라가는 방법 | 내려가는 방법 |
|----|------|-------------|-------------|
| **존재감** | "걔 이름 알아?" | 동아리, 대회 입상, 학교 행사, SNS | 조용히 지냄, 전학/반 변경 |
| **호감도 평판** | "걔 괜찮은 애야?" | 친절한 선택, 도움 이벤트, 봉사 | 갈등, 일탈, 루머 |
| **화제성** | "요즘 걔 이야기 많던데?" | 성적 급변, 연애 소문, 사건 연루 | 시간 경과 (자연 소멸) |

**존재감**은 인기(존재감) 스탯과 직접 연동.
**호감도 평판**과 **화제성**은 별도 내부 수치 (0~100).

```
[가시성 조합에 따른 학교생활 체감]

존재감 高 + 호감도 高 + 화제성 低 = "안정적 인싸" (편안함)
존재감 高 + 호감도 低 + 화제성 高 = "논란의 중심" (불안정)
존재감 高 + 호감도 高 + 화제성 高 = "학교의 스타" (정점, 유지 어려움)
존재감 低 + 호감도 高 + 화제성 低 = "소수정예 좋은 애" (안전)
존재감 低 + 호감도 低 + 화제성 低 = "투명인간" (관계 이벤트 감소)
존재감 低 + 호감도 低 + 화제성 高 = "안 좋은 의미로 유명" (위기)
```

> **설계 의도:**
> 숫자 경쟁(랭킹 1등)보다 **"반에서 이름이 오르내리는 정도"**로 연출.
> "걔 요즘 공부 잘한대", "걔 ○○이랑 사귄대?" 같은 **소문의 온도**가 가시성.

### UI 표시 원칙 (디렉터 합의)

```
[내부 계산: 3축 유지]
존재감, 호감도 평판, 화제성을 내부적으로 추적한다.

[플레이어에게 보여주는 방식: 축약]
- 평소: "학교 내 위치" 태그 1개만 표시 (모범생, 인싸, 조용한 아이 등)
- 사건 발생 시: "요즘 말이 많다", "눈에 띈다" 같은 상태 메시지로 표시
- 화제성은 이벤트 기반 임시 상태로만 표시 (2~3주 후 자연 소멸)
- 수치 3개를 직접 보여주지 않는다

원칙: "내부는 풍부하게, 플레이 체감은 단순하게"
```

### 5-2. 같은 반 / 다른 반 구분

```
[같은 반 효과]
- 학교 수업 슬롯에서 자동 접촉 → 친밀도 자연감소 속도 x0.5
- 가시성 영향이 직접적 (내 모든 행동이 반 아이들에게 보임)
- 반 내 NPC 3~4명 배정 (학년마다 재배정)
- "짝꿍" 시스템: 한 학기에 1명, 강제 근접 → 대화 이벤트 확률 증가
- **주의: "같은 반 = 자동 친해짐"이 아니라 "유지가 더 쉬움"**
- 같은 반이어도 갈등/무관심이면 멀어질 수 있음

[다른 반 효과]
- 쉬는 시간/점심시간에만 접촉 가능 → 자동 접촉 없음
- 의도적으로 "찾아가야" 관계 유지 (친구놀기 활동 필요)
- 가시성 영향이 간접적 (소문으로만 전달)
- "같은 학교인데 다른 반이라 멀어졌다" — 현실 한국 학교의 핵심 경험

[반 편성 리셋 (매 학년)]
- 새 학년 시작 시 반 재배정
- 같은 반 NPC 중 1~2명만 유지, 나머지는 다른 반으로
- 가시성(존재감) 소폭 리셋 (전년 대비 -10%)
- "작년에 친했던 애가 올해는 다른 반이라 멀다" — 자연스러운 관계 리셋
```

> **설계 의도:** 한국 학교에서 "같은 반"과 "다른 반"의 차이는 거의 "같은 동네"와 "다른 동네" 수준.
> 반 편성 하나로 1년 동안의 관계 지도가 바뀌는 경험을 반영.

### 5-3. 시험 기간의 관계 온도 변화

```
[시험 2주 전 ~ 시험 주]

전체 분위기:
- NPC들의 SNS 활동 감소, 단톡 텐션 하락
- "놀자" 이벤트 발생 확률 -50%
- NPC들이 먼저 연락하는 빈도 감소

관계 변화:
- 학업 높은 NPC: "같이 공부하자" 이벤트 증가 (스터디 그룹 기회)
- 학업 낮은 NPC: "시험 끝나면 놀자" 연기 메시지
- 경쟁 관계 NPC: 긴장감 상승, 대화 감소
- 연인 NPC: "시험 끝나면 보자" vs "시험 기간인데 만나자" 딜레마

[시험 후]
- 분위기 반전: SNS 활동 폭증, "시험 끝났다!" 단톡 폭발
- 등수 비교 이벤트: NPC 간 "몇 등이야?" 질문 → 자존감 변동
- 놀기 이벤트 집중 발생 (관계 회복 기회)
```

## 6. 온라인 시스템 — 관계의 증폭기

> "SNS가 관계를 만드는 게 아니다. 원래 있던 관계를 더 달아오르게 하거나, 더 식게 만든다."

### 6-1. 해금 시점

| 학년 | 온라인 상태 |
|------|----------|
| 초등 (Y1) | **없음** — "아직 스마트폰이 없다" (부모에 따라 차이) |
| 중학 (Y2+) | **해금** — "친구가 계정 만들었대" 이벤트로 자연 도입 |

> 플레이어가 "만들까 말까" 선택 가능. 안 만들어도 되지만, 관계 유지가 어려워짐.

### 6-2. 5개 채널 구조

특정 앱을 재현하지 않는다. **감정 단위**로 추상화한다.
10년 전 유저는 "카톡+싸이월드" 감성으로, 요즘 유저는 "인스타+DM+단톡" 감성으로 읽을 수 있게.

### 체감 구조 (디렉터 합의: 내부 5채널, 플레이어 조작 2채널)

```
[핵심 조작 채널] — 플레이어가 직접 선택하고 관리
  ├─ 개인채팅 (DM): 핵심 관계 유지, 먼저 연락 선택
  └─ 단톡: 참여/잠수/편들기 선택

[반자동 연출 채널] — 시스템이 자동으로 보여주고, 플레이어는 반응만
  ├─ 피드: NPC 게시물이 올라옴 → 좋아요/무시 선택
  └─ 스토리: NPC 근황 표시 → 보기/넘기기

[이벤트 채널] — 특정 사건 시에만 등장
  └─ 익명게시판/소문: 소문 발생 → 대응 선택 (무시/해명/역이용)
```

```
[채널 1: 피드]
= 공개 게시물. "나를 세상에 보여주는 창"

- NPC들이 게시물을 올림 (일상, 자랑, 감성글)
- 플레이어도 업로드 가능 (주 1회, 슬롯 미소모)
- 반응하기: 좋아요/댓글 → 친밀도 +0.5 (미세 효과)
- 가시성(존재감) 연동: 피드 활동이 많을수록 존재감 상승
- 코스튬(BM) 연동: 프로필/게시물에 코스튬 반영

감정 포인트: "누가 내 게시물에 반응했는가" → 후회 데이터
```

```
[채널 2: 스토리/짧은 근황]
= 24시간 후 사라지는 일상 공유. "가벼운 온도 확인"

- NPC들의 일상 스토리가 수시로 올라옴
- 보기만 해도 "관심 있음" 표시 → 관계 온도 미세 유지
- 안 보면? → NPC가 느끼진 못하지만, 플레이어가 NPC 근황을 모름 → 대화 선택지 축소
- "스토리만 보고 지나가는 사이" = 알고는 있지만 안 가까운 상태

감정 포인트: "같이 찍힌 사진이 남았는가" → 앨범/회고 시스템
```

```
[채널 3: 단톡 (그룹 채팅)]
= 소속감의 온도계. "나는 이 그룹의 일원인가?"

- 반 단톡, 동아리 단톡, 친구 그룹 단톡 등 자동 생성
- 단톡에서의 역할:
  → 말을 자주 하면: 그룹 내 존재감 상승
  → 읽고 말 안 하면: "잠수" 태그 (미세 호감도 하락)
  → 특정 NPC 편들기: 관계 분기 이벤트
- 단톡 정보: NPC들이 나누는 정보를 캐치 → 이벤트 선택지 확장
  → 미가입/잠수 시 정보 놓침

감정 포인트: "누가 단톡에서 내 편을 들어줬는가" → 후회/감사 데이터
```

```
[채널 4: 개인 채팅 (DM)]
= 핵심 관계 유지 도구. "진짜 가까운 사이의 증거"

- "연락하기" 활동의 시각적 표현 (슬롯 소모 or 무료 주 3회)
- NPC 반응 시간 = 친밀도 반영:
  → 친밀도 80+: 즉시 답장 + 길게 대화
  → 친밀도 50~79: 몇 시간 후 답장
  → 친밀도 30~49: 하루 뒤 짧은 답장
  → 친밀도 30-: 읽씹 또는 이틀 뒤 한 줄
- 방학/다른 학교 NPC와의 유일한 관계 유지 수단
- "먼저 연락했는가" 기록 → 후회 지표 축적

감정 포인트: "누구에게 먼저 연락했는가", "누구 메시지를 읽고 넘겼는가"
```

```
[채널 5: 익명 게시판 / 소문]
= 학교 내 긴장감의 진원지. "통제할 수 없는 정보"

- 학교 익명 게시판에 소문이 올라옴 (시스템 자동 생성)
- 플레이어 관련 소문: 가시성(화제성) 급등
  → "○○이랑 ○○ 사귀는 거 맞지?"
  → "시험 등수 떨어진 거 봤어?"
  → "걔 요즘 누구랑 다녀?"
- 대응 선택지:
  → 무시하기: 화제성 서서히 감소 (2~3주)
  → 해명하기: 화제성 즉시 감소, but 존재감 상승 (관심 유발)
  → 역이용: 화제성 유지 + 특정 방향으로 평판 조작 (위험)
- NPC 관련 소문: NPC에게 알려줄지 말지 선택 → 관계 분기

감정 포인트: "소문이 온라인으로 번짐" → 학교/온라인 간극 체감
```

### 6-3. 온라인의 핵심 원칙: 증폭기

```
[SNS는 독립 성장 시스템이 아니다]

틀린 설계: SNS 활동 → 인기 +2 (독립 성장)
맞는 설계: 기존 관계의 온도를 증폭/감소

예시:
- 친밀도 70인 NPC와 DM을 자주 주고받으면 → 75로 서서히 상승 (증폭)
- 친밀도 70인 NPC의 게시물을 계속 무시하면 → 65로 서서히 하락 (냉각)
- 친밀도 30인 NPC에게 DM을 보내면 → 읽씹당하고 멘탈-2 (역효과)
- 피드에 게시물을 올리면 → 기존 관계 전체에 미세 유지 효과 (+0.3)
```

### 6-3-1. "SNS 활동" 슬롯 재해석 (디렉터 합의)

기존 활동 테이블의 "SNS 활동 (1슬롯, 인기+2, 멘탈-1)"은 더 이상 범용 스탯 버프가 아니다.
**현재 관계망을 기준으로 결과가 달라지는 "온라인 관계 관리 시간"**으로 재해석한다.

```
[SNS 활동 (1슬롯) — 재해석]

플레이어가 "SNS 활동"을 선택하면, 아래 중 하나를 고른다:

1. "피드 올리기" → 가시성(존재감) +1, 친밀도 50+ NPC 전원에게 유지 +0.3
2. "단톡 참여하기" → 소속 그룹 호감도 +1, 단톡 정보 습득 (이벤트 선택지 확장)
3. "답장 정리하기" → 읽씹 중인 NPC 1명에게 답장 → 친밀도 하락 방지 + 후회 데이터 감소
4. "그냥 둘러보기" → 피로 -3, 멘탈 -1 (시간만 흘러감, 비교 우울 위험)

결과가 "인기+2" 고정이 아니라, 선택과 현재 관계에 따라 달라진다.
```

### 6-4. 온라인을 안 하는 선택

```
[미가입 시]

장점:
- 온라인 이벤트(악성댓글, 비교우울, 소문 확산) 미발생
- 멘탈 안정성 상승
- "조용한 학교생활" 느낌

단점:
- 개인 채팅/단톡 불가 → 방학 중 관계 유지가 오프라인(전화)만 가능
- 단톡 정보 놓침 → 이벤트 선택지 축소
- 가시성 성장 경로 차단
- "걔 SNS 안 해?" NPC 대사 → 약간의 소외감

→ "안 하는 것도 선택. 하지만 2020년대 학교에서 SNS 없이 산다는 건 쉽지 않다."
```

### 6-5. 온라인 → 후회 시스템 데이터 연결 (§7 엔딩 연동)

온라인 시스템은 엔딩의 **후회 카드**에 필요한 데이터를 생성하는 핵심 레이어.

| 온라인 행동 | 축적 데이터 | 후회 카드 연결 |
|------------|-----------|--------------|
| NPC 메시지 읽고 안 답함 | 읽씹 횟수 | "답장하지 않은 메시지 ○건" |
| NPC 놀자는 제안 거절 | 거절 횟수 | "거절한 약속 ○건" |
| 먼저 연락 안 함 | 먼저 연락 비율 | "먼저 연락한 횟수 / 받은 횟수" |
| 단톡 잠수 | 잠수 기간 | "○주 동안 단톡에 아무 말도 안 했다" |
| 게시물 반응 안 함 | 무시 횟수 | "친구의 게시물 ○개를 그냥 넘겼다" |
| 같이 찍은 사진 | 사진 수 | 앨범 "빈 슬롯" vs 채워진 슬롯 |

> **설계 의도:** 이 데이터는 플레이 중에는 보이지 않는다.
> 엔딩에서 처음으로 합산되어 보여지며, "나도 모르게 쌓인 후회"를 체감하게 한다.

### 6-6. 학교와 온라인의 간극 — 드라마의 원천

```
[같은 NPC의 학교 모습 vs 온라인 모습이 다를 수 있다]

예시:
- 학교에서는 조용한 NPC가 단톡에서는 유쾌함
  → "걔 원래 저런 애였어?" 이벤트
- 학교에서는 인싸인 NPC가 DM 답장은 느림
  → "학교에서는 친한 척하는데..." 실망 이벤트
- 학교에서는 어색한 NPC와 온라인에서는 잘 맞음
  → "실제로 만나면 어색한데 채팅은 편함" → 관계 발전 분기
- 학교에서는 멀어졌는데 단톡은 아직 남아있음
  → "나가기" 선택지 → 관계 완전 종료 vs 유지

이 간극이 이벤트를 만들고, 선택을 강제하고, 후회를 남긴다.
```

### 6-7. 코스튬 시스템 연동 (BM)

```
플레이어 캐릭터의 프로필/게시물에 코스튬이 반영된다.
- 코스튬 구매 (유료) → 온라인 프로필에 표시
- NPC들이 "오 새 옷?" 반응 → 친밀도 +1 (최초 1회)
- 코스튬은 순수 비주얼. 스탯 영향 없음.

→ "과금의 이유"를 온라인이라는 게임 내 사회 시스템에 자연스럽게 연결.
```

## 7. 학교 전환 이벤트 (01·06 연동)

### 7-1. 초등 → 중학교

```
[Y1 겨울방학: 중학교 배정]

이벤트 흐름:
1. "중학교 배정 결과가 나왔다" — 긴장감 연출
2. NPC별 같은 학교/다른 학교 발표 (부모 조합/지역에 따라 결정)
3. 같이 가는 NPC: 관계 유지, 친밀도 변동 없음
4. 다른 학교 NPC: "연락 빈도 급감" 경고, 친밀도 자연감소 속도 x2
5. 초등 졸업식 컷신: NPC별 작별 장면

시스템 변화:
- 인기(존재감): 중앙값(45) 방향으로 20% 수렴 (새 환경 리셋)
- NPC 친밀도: 다른 학교 NPC는 유지 채널이 "연락"만 남음
- 새 NPC 등장: 2~3명 (중학 전용 캐릭터)
- SNS 해금 (Y2 3월 이벤트)
```

### 7-2. 중학교 → 고등학교

```
[Y4 겨울방학: 고등학교 선택]

이벤트 흐름:
1. 담임 면담 이벤트: "너 어떤 고등학교 갈 생각이니?"
2. 부모 의견 이벤트: 부모 유형별 반응
   - 재력: "특목고 가야지" (압박)
   - 정서적지지: "네가 가고 싶은 곳으로" (자유)
   - 엄격함: "일반고는 안 돼" (강제)
3. 선택지 제시:
   ├─ 일반고 (조건 없음)
   └─ 특목고 (학업 85+ 또는 재능 85+ 필요)
4. 특목고 지원 시: 면접 이벤트 (멘탈 50+ 필요, 미만 시 긴장으로 실패 확률 증가)
5. 합격/불합격 발표: 감정 연출

시스템 변화:
- 인기(존재감): 중앙값(45) 방향으로 20% 수렴
- 슬롯 구조 변경: 야자 등장
- NPC: 다른 학교 NPC는 연락 채널만, 새 NPC 2~3명 등장
- 학업 난이도 상승: §15 학년별 학업 유지 비용 적용 시작
```

### 7-3. 전학 이벤트 (특수)

```
[전학 조건]
- 특목고 부적응: 3회 연속 야자 거부 + 등수 하위 20% + 멘탈 30-
- 가정 환경 변화: 특정 이벤트 분기 (부모 이혼, 이사 등)
- 플레이어 선택: "전학 가고 싶다" 선택지 (조건부 등장)

[전학 효과]
- 인기 리셋 (20으로 초기화)
- NPC 풀 전면 교체 (기존 NPC는 연락 채널만)
- 멘탈 -15 (환경 변화 스트레스)
- "모든 걸 처음부터" — 가장 극단적인 리셋 이벤트
```

## 8. 수능 시스템 (Y7 특별 이벤트)

> "7년의 선택이 하루에 결정된다."

### 8-1. 수능 전 (Y7 9~11월) — 감정 밀도 극대화

```
[수능 D-60]
→ 분위기 전환: UI 색상 채도 저하, BGM 긴장감
→ 모의고사 성적 기반 "예상 등급" 표시
→ "수시 vs 정시" 전략 선택 이벤트

[수능 D-30]
→ 학교 분위기 변화 체감:
  - 단톡 텐션 하락 (NPC 메시지 빈도 감소)
  - 스토리 업로드 감소 (NPC들도 긴장)
  - 말수 줄어드는 NPC (평소 수다쟁이도 조용해짐)
→ 학교 응원 이벤트: 후배 NPC가 응원 편지/메시지
→ "마지막 한 달, 어디에 집중할 것인가?" 선택

[수능 D-7]
→ 부모 반응 이벤트 (부모 유형별):
  - 재력: "과외 선생님이 마지막 특강 잡아줬어"
  - 엄격함: "마지막이니까 정신 차려" (압박 극대화)
  - 정서적지지: "결과가 어떻든 우리 딸/아들 잘했어" (멘탈+3)
  - 시간/자유: "네가 알아서 해. 믿는다" (담백)
→ 교사 면담: "마지막 모의고사 기준 ○등급, 현실적으로..."
→ NPC 반응: 경쟁 관계 NPC와 긴장, 친한 NPC는 위로

[수능 D-1]
→ "내일이면 끝이다" 독백 컷신
→ 친밀도 1위 NPC와 전화 통화 이벤트 (#19 딜레마)
→ 선택: 일찍 자기 / 벼락치기 / 친구와 통화
→ 밤 장면: 책상 위 7년간의 흔적들이 보이는 연출
```

### 8-2. 수능 당일

```
[수능 연출]
→ 아침: 부모님이 엿/떡 주는 장면 (부모 유형별)
→ 시험장 입장: 7년간의 장면이 스쳐 지나가는 몽타주
→ 교문 들어서는 순간: 소리가 멈추고, 발소리만 들리는 연출
→ 쉬는 시간마다 NPC 대사:
  - "야 1교시 어땠어?" / "나 망한 것 같아..." / "마지막이다, 파이팅"
→ 시험: 직접 풀지 않음 — 결과만 표시
  → 학업 스탯 + 멘탈 보정 + 운 요소 = 최종 등급

[결과 산정]
최종_수능_등급 = 학업_스탯 기반_등급 ± 멘탈_보정 ± 운

멘탈 보정:
- 멘탈 80+: +0.5 등급 보정 (안정적 실력 발휘)
- 멘탈 50~79: 보정 없음
- 멘탈 30~49: -0.5 등급 (긴장으로 실수)
- 멘탈 30-: -1.0 등급 (패닉)

운 요소: -0.3 ~ +0.3 등급 (랜덤)
```

### 8-3. 수능 이후

```
[결과 발표]
→ 성적표 UI: 과목별 등급 + 백분위 + 예상 진학 대학
→ 시험 끝난 직후: 해방감 or 공허감 분기 (멘탈/성적 예상에 따라)
  - 멘탈 70+: "드디어 끝났다..." (해방감 + 약간의 허무)
  - 멘탈 50-: "끝났는데... 왜 이렇게 불안하지" (공허감)
→ NPC 반응: 성적 공유/비교 이벤트
→ 부모 반응: 최종 부모 평가 이벤트 (7년간의 기대 vs 결과)
→ 결과 확인 직전: 정적. 클릭하기 전 3초 대기 연출 (긴장의 정점)

[진학 분기]
→ 성적 + 생활기록부 + 재능 점수에 따라 대학 결정
→ 대학 선택은 엔딩 분기에 직접 연결 (§7 엔딩시스템 연동)
→ "원하는 대학에 떨어졌다" → 재수/취업/유학 선택지
```

## 9. 학교 내 숨겨진 시스템

### 9-1. 교사 호감도

```
[교사는 NPC가 아니지만, 시스템으로 작동]

교사 호감도 (0~100):
- 학업 상위 → 자동 상승
- 봉사활동, 동아리 활동 → 상승
- 일탈, 지각, 야자 거부 → 하락
- 문제아 태그 → 급락

교사 호감도의 영향:
- 생활기록부 서술 품질 (수시 전형에 중요)
- 추천서 작성 여부
- "괜찮은 대회 있는데 나가볼래?" 추천 이벤트
- 문제 발생 시 관대한 처리 vs 엄격한 처리
```

### 9-2. 소그룹 계층

```
[학교 안의 작은 그룹들]

현실 학교에는 "전교/학년/반" 외에도 작은 그룹이 항상 존재한다.
이 그룹들이 단톡 구성, 편들기, 소문 확산 범위에 영향을 준다.

소그룹 유형:
- 같은 반 친한 무리 (3~4명, 점심 같이 먹는 사이)
- 학원/과외 같이 다니는 그룹 (학원 채널 공유)
- 동아리 그룹 (동아리 단톡)
- 취미 그룹 (게임, 음악, 운동 등 공통 관심사)

소그룹의 시스템적 역할:
- 단톡 자동 생성 단위 (반 단톡, 동아리 단톡, 친한 무리 단톡)
- 소문 확산 범위 (소그룹 내 → 반 전체 → 학년 전체)
- 편들기 이벤트 시 "같은 무리" 압박
- NPC 간 갈등 시 중립이 어려운 구조

UI 표시: 직접 노출하지 않음.
단톡 구성과 이벤트 반응에서 자연스럽게 체감되도록 설계.
```

### 9-3. 학교 평판

```
[학교 내 플레이어 평판 태그]

태그는 복수 보유 가능하며, 주요 태그가 NPC 반응에 영향.

- 모범생: 학업 상위 30% + 일탈 낮음
- 인싸: 인기 70+
- 실력파: 재능 70+ (분야명 표시)
- 조용한 아이: 인기 30- + 일탈 낮음
- 문제아: 일탈 누적
- 요즘 뜨는 애: 인기 급상승 시 (일시적)
- 성적 떨어진 애: 등수 급락 시 (일시적)

태그에 따라 NPC 첫인사, 이벤트 선택지, 교사 반응이 달라진다.
```
