# LIFE TRACK : 선택의 결과

> "이 게임은 성적을 올리는 게임이 아니라, 인생을 설계하는 게임이다."

한국형 성장 시뮬레이션 — 초등 6학년부터 고등학교 3학년까지 7년간의 선택과 결과.

---

## 게임 실행

```bash
cd game
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속

---

## 기술 스택

| 항목 | 기술 |
|------|------|
| 프레임워크 | Vite + React + TypeScript |
| 상태관리 | Zustand |
| 플랫폼 | 웹 (브라우저) → Steam (Electron 래핑) |
| 아트 | AI 생성 + 수동 보정 |

---

## 핵심 시스템

- **5대 스탯**: 학업, 인기, 재능, 멘탈, 체력 (0~100)
- **부모 선택**: 6개 강점 중 2개 선택 → 15가지 조합 = 15가지 인생
- **주간 루프**: 평일(루틴 자동) + 주말(직접 선택) + 방학(자유 배분)
- **구간 감쇠**: 높을수록 성장이 느려짐 → "모든 걸 가질 수 없다"
- **관계**: NPC 10명, 유지하지 않으면 사라짐
- **엔딩**: 성취지수 x 행복지수 이중 평가, 후회카드 시스템
