# 03. 부모(가정) 시스템 — 핵심 차별화 요소

## 설계 원칙

> "부모는 버프가 아니라 조건이다"
> "부모 선택 = RPG의 클래스 선택"

모든 가정에는 강점과 약점이 공존한다. 사기적인 부모는 존재하지 않는다.

---

## 가정 설계: 강점 2개 선택 방식

플레이어가 게임 시작 시 **가정의 강점 2가지**를 선택한다.
(아빠/엄마로 고정하지 않음 → 한부모, 조부모 등 다양한 가정 형태 포용)

### 강점 풀 (택 2)

| 강점 | 효과 | 대가 |
|------|------|------|
| **재력** | 용돈 多, 고가 학원/과외 선택지 개방, 브랜드 아이템 구매 가능 | 부모 기대치 높음 → 압박 ↑, 자유시간 ↓ |
| **정보력** | 히든 선택지 개방 (명문 학원, 특별전형 컨설팅, 인맥 소개 등) | 간섭 ↑, 플레이어 자율성 ↓ ("엄마가 다 알아봤으니 이렇게 해") |
| **유전자 (외모/지능)** | 시작 스탯 보너스 (학업 or 외모 기본치 높음) | 부모가 "너는 원래 잘하잖아" → 지원이 소극적, 실패 시 실망 큼 |
| **정서적 지지** | 멘탈 회복 빠름, 번아웃 내성 ↑, 실패해도 안전망 | 자원(돈) 부족, 경쟁적 환경에서 뒤처질 수 있음 |
| **시간/자유** | 자유시간 최대, 이중생활/일탈 루트 가능, 자율적 선택지 多 | 자원 부족, 정보 부족 → "알아서 해야 함" |
| **엄격함/규율** | 학업 성장 속도 보너스, 루틴 효과 ↑ | 멘탈 소모 빠름, 반항 이벤트 트리거, 관계 제약 |

### 선택 규칙
- 6개 중 2개 선택 → 조합: 15가지 (관리 가능한 범위)
- 선택하지 않은 강점은 "보통~약함"으로 자동 배정
- 같은 카테고리 중복 불가

---

## 정보력 상세 설계

> "아이의 입시 성공은 할아버지의 재력, 아빠의 무관심, 엄마의 정보력"

정보력은 **플레이어에게 보이는 선택지의 수**를 바꾸는 핵심 변수.

```
[정보력 높은 가정]
이번 주 학업 선택지:
├─ 독학 (무료)
├─ 인터넷 강의 (용돈 -1)
├─ 일반 학원 (용돈 -2)
├─ [히든] 소문난 학원 (용돈 -2, 학업 성장 1.3배) ← 같은 돈, 더 좋은 곳
└─ [히든] 선배 과외 소개 (용돈 -3, 약점 과목 집중 보강)

[정보력 낮은 가정]
이번 주 학업 선택지:
├─ 독학 (무료)
├─ 인터넷 강의 (용돈 -1)
└─ 일반 학원 (용돈 -2)

→ "소문난 학원"이 존재하는지 자체를 모름
```

> **정보력의 핵심: 더 비싼 게 아니라, 같은 돈으로 더 효율적인 선택지가 보이는 것**
>
> - 용돈 -2짜리 일반 학원 vs 용돈 -2짜리 소문난 학원 → 가격은 같지만 효과가 다름
> - 재력이 부족해도 정보력이 있으면 한정된 예산을 최적화할 수 있음
> - 현실의 "엄마 네트워크"가 작동하는 방식 그대로

정보력이 낮아도 **NPC 관계**를 통해 정보를 얻을 수 있음:
- 정보력 높은 친구 엄마와 친해지면 히든 선택지 일부 개방
- 선생님과 관계 좋으면 진로 정보 이벤트 발생
→ "정보력 = 고정 수치"가 아니라 "관계로 보완 가능"

### 정보력 vs 재력의 관계

정보력과 재력은 **보완 관계**이지, 곱하기 관계가 아님:

```
[재력 高 + 정보력 高] → 최적의 선택지 + 여유 자원 (가장 편한 플레이)
[재력 低 + 정보력 高] → 적은 돈을 최대 효율로 사용 (전략적 플레이)
[재력 高 + 정보력 低] → 돈은 있지만 비효율적으로 씀 (비싼 학원만 알음)
[재력 低 + 정보력 低] → 독학/자력 루트 (가장 어려운 난이도, 관계로 돌파)
```

→ 재력이 없어도 정보력으로 커버 가능한 구조
→ "무조건 비싼 게 좋다"가 아니라 "잘 아는 게 힘이다"

## 가정 환경이 게임에 미치는 영향 (종합)

| 영향 범주 | 작동 방식 |
|-----------|-----------|
| 시작 스탯 | 유전자 강점 → 초기 보너스 |
| 보이는 선택지 | 정보력/재력에 따라 카드가 다르게 보임 |
| 자원 (용돈) | 재력에 비례, 활동 카드에 비용 표시 |
| 제약 | 압박/규율 → 특정 활동 강제 or 금지 |
| 이벤트 | 가정 타입에 따라 고유 이벤트 트리거 |

---

## 가정 변동 이벤트

게임 중반 가정환경이 변할 수 있음. 단, **반드시 플레이어 선택과 인과관계**가 있어야 함.

```
❌ 나쁜 예: "아빠가 갑자기 명퇴당했습니다" (랜덤)
✅ 좋은 예: 부모 압박을 계속 무시 → 부모 스트레스 누적 → 가정 위기 이벤트
           → 플레이어: "내가 너무 반항만 해서...?"
```

## 플레이 예시

```
[재력 + 정보력 가정] — "강남 엄마" 스타일
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
열리는 것: 최상위 학원, 특별전형 루트, 브랜드 아이템
닫히는 것: 자유시간 극소, 부모 기대치 풀가동
리스크:   번아웃, 반항 이벤트, "내 인생인데" 갈등
전략:     학업 올인 or 부모 기대 vs 내 꿈 사이 줄타기

[정서적 지지 + 시간/자유 가정] — "믿어주는 부모님"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
열리는 것: 자유로운 활동 선택, 이중생활 가능, 멘탈 안정
닫히는 것: 고가 학원 불가, 정보 부족 (히든 선택지 안 보임)
리스크:   경쟁에서 뒤처짐, 기회를 놓칠 수 있음
전략:     관계/재능 특화 or NPC 통해 정보 보완

[유전자 + 엄격함 가정] — "타고난 엘리트 + 스파르타"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
열리는 것: 높은 시작 스탯, 빠른 학업 성장
닫히는 것: 멘탈 관리 어려움, 관계 구축 시간 부족
리스크:   번아웃 직행, 부모 반항 시 지원 급감
전략:     빠르게 성적 확보 후 관계 투자 or 멘탈 관리 밸런스
```
