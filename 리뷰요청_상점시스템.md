# LIFE TRACK 리뷰 요청 — 상점 시스템 설계

## 배경

현재 돈을 버는 경로(용돈, 알바, 이벤트 보상)는 있지만 **쓸 곳이 학원/헬스 같은 루틴 비용밖에 없습니다.** 향후 경진대회 입상 상금 등이 추가되면 돈이 더 쌓이게 되는데, 소비처가 부족하면 돈 스탯 자체가 무의미해집니다.

상점 시스템을 도입해서 돈에 전략적 의미를 부여하고 싶습니다.

---

## 현재 돈의 흐름

### 수입
- 매주 용돈: 3만원 (재력 부모: 8만원)
- 알바 (고등부터): +3만원/주
- 이벤트 보상: 가변

### 지출
- 학원 수업: 3만원/주
- 헬스/PT: 2만원/주
- 예체능 레슨: 2만원/주
- 인강: 1만원/주
- 친구와 놀기: 1만원
- **그 외 소비처 없음**

## 디렉터에게 자유 의견 요청

1. **상점에서 어떤 종류의 아이템을 팔아야 할까요?**
   - 일회성 소비 (간식, 음료 등)
   - 장기 효과 아이템 (참고서, 장비 등)
   - 관계 아이템 (선물, 이벤트 티켓 등)
   - 외형/자기표현 (옷, 악세서리 등)
   - 기타 아이디어?

2. **상점의 형태는?**
   - 항상 접근 가능한 메뉴?
   - 주말에만 갈 수 있는 장소?
   - 특정 이벤트에서만 열리는 기간한정?

3. **아이템 효과의 강도는?**
   - 스탯에 직접 영향? (참고서 → 학업 +3)
   - 활동 효율 버프? (장비 → 다음 3주간 특기 성장 +20%)
   - 이벤트 해금? (콘서트 티켓 → 특별 이벤트)
   - NPC 관계? (선물 → 친밀도 +10)

4. **돈의 전략적 딜레마를 어떻게 만들까요?**
   - 비싼 루틴(학원) vs 아이템 구매 vs 저축
   - 학교급이 올라갈수록 물가 상승?
   - 고등학생 때 대학 원서비/교재비 같은 큰 지출?

5. **재력 부모의 밸런스 영향은?**
   - 현재 재력 = 용돈 8만원 (일반 3만원)
   - 상점이 강해지면 재력 부모가 OP가 될 수 있음
   - 어떻게 조절?

6. **이 외에 돈과 관련해서 추가하고 싶은 아이디어가 있다면 자유롭게**

---

각 디렉터의 자유로운 의견을 기다립니다.
